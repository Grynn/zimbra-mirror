Scheduling the Migration
Select how to migrate the users to ZCS. You can schedule to migrate now or schedule for later.
1.
On the Migrate screen, select:
•
List of users being migrated.
•
Provisioned users display with a red icon.
•
Unprovisioned users display with a greyed-out icon and have not been provisioned on the ZCS server. You must provision the users before migrating and assign an initial password and class of service for the users.
•
Schedule: Select date and time to migrate users to ZCS.
•
Provisioning: Set the password and Class of Service for unprovisioned users.
2.
Click Preview to review the accounts to be migrated. Make changes as appropriate by accessing the screens using the screen menu.
3.
If the migration is ready, on the Migrate screen click Migrate.
