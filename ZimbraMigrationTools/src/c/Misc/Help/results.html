Viewing Migration Results
In the Results dialog, view accounts that are migrated and their status, including progress, errors, and warnings.
1.
To view a results log, double click the user account and a new tab opens displaying the selected account log information.
•
Accounts: This contains the list of all users being migrated. The progress bar shows the state of the migration, and is accompanied by status messages.
•
For each user migrated, you can double click on that user in Accounts, and an individual tab will come up. Click the x on the individual tab to close view.
•
Open Log File: Opens the appropriate log file for a given user.
•
Stop: Click the Stop button to stop the migration. To restart, go back to the previous view, and click the Migrate button.
2.
Click Exit.
