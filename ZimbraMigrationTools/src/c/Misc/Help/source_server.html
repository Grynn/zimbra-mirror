Server Migration Source Information
You can migrate data from an Exchange server using either an Exchange administrator profile, or by entering the Exchange server’s credentials.
Read the Migration Wizard Guide for complete migration details. A copy can be found on the Help Desk page on the administration console.
1.
If you have an Exchange administrator profile, select Outlook Profile. If do not have an Exchange administrator profile, select Exchange Server.
•
If Outlook Profile is selected, select the profile name from the menu and Exchange prompts for credentials.
•
If Exchange Server is selected, enter the Exchange server credentials:
•
Hostname/IP: The host name of the Exchange server name or IP address.
•
Admin ID. The name of the administrator account on the Exchange server.
•
Password. The password of the administrator account on the Exchange server.
An Exchange profile is created and used to authenticate the server. This profile is deleted after the migration process.

Click Next.
Use Load or Save
You can use Load or Save to save or upload profile information:
Click Load to upload previously saved profile information to populate the profile fields.
Click Save to save the profile information you are entering. This allows you to use this information at a later time.
