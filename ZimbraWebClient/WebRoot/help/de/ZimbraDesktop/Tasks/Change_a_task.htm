So aktualisieren Sie eine Aufgabe

  1. }}}Öffnen Sie die Aufgabenliste, die Ihre Aufgabe enthält.{{{

  2. }}}Klicken Sie im Fenster Inhalt auf die Aufgabe, die Sie ändern möchten, oder wählen Sie die Aufgabe aus und klicken Sie in der Symbolleiste auf Bearbeiten.{{{ Daraufhin wird die Seite "Aufgabe" angezeigt.

  4. }}}Klicken Sie auf Speichern, wenn Sie alle Änderungen eingegeben haben.{{{
