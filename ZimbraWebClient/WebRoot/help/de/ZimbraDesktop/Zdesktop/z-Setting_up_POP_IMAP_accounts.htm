POP-/IMAP-Accounts (einschließlich Microsoft-Exchange-IMAP) einrichten

Um den Account einzurichten, benötigen Sie die Informationen zu Ihren Account-Einstellungen einschließlich der Angaben für Posteingangs- und Postausgangsserver. Sie erhalten diese Informationen bei Ihrem Systemadministrator oder Internetanbieter.

Sie können alle Mails Ihrer Mailbox synchronisieren oder aber festlegen, dass nur die Mails Ihres Posteingangs synchronisiert werden.

So richten Sie Ihren Account ein

Prüfen Sie zunächst, ob eine Internetverbindung besteht.

  1. Öffnen Sie Zimbra Desktop und klicken Sie in der rechten oberen Ecke auf Einrichten.  

  2. Klicken Sie auf NEUEN ACCOUNT HINZUFÜGEN.  Wählen Sie in der Liste "Account-Typ" den entsprechenden Account-Typ aus. Die Seite zum Einrichten eines Zimbra-Accounts wird angezeigt. Geben Sie die folgenden Daten ein.

Accountname: Dies ist der Name, der in Zimbra Desktop als Accountname angezeigt wird. Wenn Sie mehrere Accounts einrichten, geben Sie ihnen aussagekräftige Namen, um die einzelnen Accounts unterscheiden zu können. Für Ihren Account können Sie z. B. "Yahoo" eingeben.

Ihr Vor- und Nachname: Geben Sie Ihren Namen ein. (optional).

Mail-Adresse: Geben Sie Ihre Mail-Adresse für diesen Account ein.

  1. Geben Sie im Abschnitt Mail empfangen die folgenden Informationen ein:

Nutzername: Geben Sie den Nutzernamen ein, mit dem Sie Ihre Mails abfragen. Dies ist typischerweise ein Name in der Form nutzername@beispiel.com.

Passwort: Geben Sie das Passwort für die Anmeldung an diesem Account ein.

Posteingangsserver: Geben Sie die Adresse Ihres Servers für eingehende Mails an. Dies ist typischerweise ein Name in der Form mail.beispiel.com.

Sicherheit: Aktivieren Sie ggf. folgende Option: SSL-Verschlüsselung für den Zugriff auf diesen Server verwenden.  Wenn Ihr Posteingangsserver einen anderen Port nutzt, klicken Sie auf Bearbeiten, um die Portnummer zu ändern. Wenn Sie nicht sicher sind, welchen Port Ihr Posteingangsserver nutzt, wenden Sie sich an Ihren Systemadministrator.

  1. Geben Sie im Abschnitt Mail senden die folgenden Informationen ein:

SMTP-Mailserver: Geben Sie die Adresse Ihres Servers für ausgehende Mails an. Dies ist typischerweise ein Name in der Form smtp.beispiel.com.

Sicherheit: Aktivieren Sie ggf. folgende Option: SSL-Verschlüsselung für den Zugriff auf diesen Server verwenden.  Wenn Ihr Posteingangsserver einen anderen Port nutzt, klicken Sie auf Bearbeiten, um die Portnummer zu ändern. Wenn Sie nicht sicher sind, welchen Port Ihr Posteingangsserver nutzt, wenden Sie sich an Ihren Systemadministrator.

Wählen Sie Authentifizierung, wenn Nutzername und Passwort zum Senden von Mails eingegeben werden müssen. Wenn Sie diese Option aktivieren, geben Sie Ihren Nutzernamen und das Passwort ein.

Anwortadresse: Wenn die Antwortadresse eine andere sein soll als Name und Mail-Adresse Ihrer gesendeten Mails, geben Sie die gewünschten Angaben für Name und Mail-Adresse ein.

  1. Legen Sie im Abschnitt Synchronisierungseinstellungen den Zeitplan für die Synchronisierung mit dem Server fest. Standardmäßig wird alle 15 Minuten auf neue Mails geprüft. Wenn Sie die Option Manuell auswählen, müssen Sie in der Symbolleiste von Zimbra Desktop auf "Senden/Empfangen" klicken, um Ihren Account mit dem Zimbra-Server zu synchronisieren. Die Auswahl eines zu kurzen Zeitraums – z. B. eine Minute – beeinträchtigt unter Umständen die Leistungsfähigkeit Ihres Computers.

Sie können diese Einstellung jederzeit ändern, indem Sie auf Einrichten klicken, den Account auswählen und die Einstellung ändern.

  1. Klicken Sie abschließend auf Prüfen und Speichern.

Zimbra Desktop prüft Ihre Account-Daten. Wurde Ihr Account erfolgreich geprüft, wird das Dialogfeld Dienst erstellt angezeigt. Wurde Ihr Account nicht erfolgreich geprüft, wird erneut das Dialogfeld "Einrichten" angezeigt, so dass Sie Ihre Eingaben korrigieren können.

  1. Klicken Sie auf Desktop starten, um Zimbra Desktop zu öffnen. Während der Synchronisierung Ihres Accounts können Sie in Zimbra Desktop arbeiten.
