Mails drucken

Sie können eine oder mehrere Mails zum Drucken auswählen, und Sie können eine komplette Unterhaltung zum Drucken auswählen.

So drucken Sie eine Mail

  1. Wählen Sie die Mail aus und klicken Sie auf Drucken.

Die ausgewählte Mail wird in einem neuen Fenster angezeigt, außerdem wird das Standarddialogfeld zum Drucken aufgerufen.

Wenn Sie mehrere Mails auswählen, werden sie als eine Nachricht ausgedruckt.

  1. Wählen Sie im Dialogfeld einen Drucker aus und klicken Sie auf Drucken, um die Mails an den ausgewählten Drucker zu senden.

Die Standard-Schriftgröße für die Druckausgabe ist 12 pt. Sie können die verwendete Schriftgröße im Bereich Einstellungen>Allgemein>Sonstiges ändern

Hinweis: Verwenden Sie zum Ausdrucken Ihrer Mail nicht die Funktion Datei>Drucken des Browsers. Die Seite wird dabei nicht gut formatiert und die Mail ist möglicherweise schwer zu lesen.
