Im Ordner "Dateien" arbeiten

Der Ordner "Dateien" kann zum Austausch und Verwalten von Dokumenten, die Sie im Ordner "Dateien" erstellen verwendet werden, bzw. alternativ von Dokumenten und Bildern, die Sie in Ihren Ordner "Dateien" hochladen. Im Ordner "Dateien" können Sie Dateien in Ihrem Account speichern, sodass Sie diese Dateien jederzeit nach dem Anmelden bei Ihrem Account von einem beliebigen Computer aus aufrufen können.

Sie können Dokumente, Kalkulationstabellen, Präsentationen, Bilder, PDF-Dateien von Ihrem Computer oder Ihrem Computer-Netzwerk hochladen, und Sie können mit Ihren Mails versendete Anhänge in einem Ordner "Dateien" speichern.  Sie haben die Möglichkeit, verschiedene Ordner im Ordner "Dateien" anzulegen, um Ihre Dateien zu organisieren, und Sie können Ihre Dateien für andere Nutzer freigeben.

Die Dateien im Ordner "Dateien" sind Kopien der ursprünglichen Datei auf Ihrem Computer.  Änderungen, die Sie an einer Datei im Ordner "Dateien" vornehmen, haben keine Auswirkungen auf die ursprüngliche Datei.

Ordner "Dateien" anlegen

  1. Klicken Sie auf die Registerkarte Dateien.

  2. Wählen Sie in der  Symbolleiste unter Dateien die Option Neu aus.

  1. Daraufhin wird das Dialogfeld Neuen Ordner in Dateien erstellen angezeigt. Geben Sie einen Namen für den neuen Ordner "Dateien"ein.

Dieser Name muss innerhalb der Hierarchie Ihrer Mailbox-Ordner eindeutig sein. Ordner "Dateien" dürfen nicht denselben Namen haben wie ein übergeordneter Ordner in der Mail, im Kalender oder im Adressbuch von @CompanyName@. Wenn Sie beispielsweise einen Kalender mit dem Namen „Urlaub“ haben, dürfen Sie keinem übergeordneten Ordner „Dateien“ den Namen „Urlaub“ geben. Ein Ordner "Dateien", der sich in einem untergeordneten Ordner befindet, darf "Urlaub" heißen.

  1. Wählen Sie die  Ordnerhierarchie aus. Sie können einen neuen übergeordneten Ordner anlegen oder den Ordner in einen vorhandenen Ordner einfügen.

  2. Klicken Sie auf OK. Der neue Ordner wird in der Übersicht angezeigt.

Dateien hochladen

Beim Hochladen wird eine Datei von Ihrem Computer auf Ihren Mail-Account hochgeladen. Dadurch steht die Datei jederzeit zur Verfügung, wenn Sie sich an Ihrem Account anmelden. Ihr Accountkontingent bestimmt, wie viele Dateien unter „Dateien“ hochgeladen werden können.

So laden Sie Dateien hoch

  1. Öffnen Sie den Ordner „Dateien“ und klicken Sie in der Symbolleiste auf Datei hochladen.

  2. Klicken Sie im Dialogfeld Neue Datei in Dateien hochladen auf Durchsuchen, um die Dateien, die hochgeladen werden sollen, zu finden. Sie können mehrere Dateien gleichzeitig zum Hochladen auswählen, wenn sich diese Dateien in demselben Verzeichnis befinden. Zum Hinzufügen anderer Dateien klicken Sie auf Hinzufügen und navigieren zur gewünschten Datei.

Sie können keine Datei hochladen, die denselben Namen hat wie eine Datei, die bereits unter "Dateien" vorhanden ist. Sie können im Dialogfeld „Hochladen“ angeben, wie beim Hochladen von Dateien mit demselben Namen vorgegangen werden soll. Wenn Sie diese Option ändern, wird die Änderung im Dialogfeld gespeichert.

    • Wenn Sie benachrichtigt werden möchten, wenn Sie eine Datei mit demselben Namen hochladen, wählen Sie Nachfragen aus. Wenn eine Datei denselben Namen wie eine bereits hochgeladene Datei hat, wird das Dialogfeld Konflikt beim Hochladen angezeigt. Sie können die vorhandene Datei behalten oder durch die neue Datei ersetzen. Sie können sich die neue Datei auch ansehen. Dies ist die Standardeinstellung.

    • Wenn Sie möchten, dass die hochgeladene Datei immer die bereits vorhandene Datei gleichen Namens ersetzt, wählen Sie Durch meine Datei ersetzen. Die neue Datei wird hochgeladen und ersetzt die vorhandene Datei. Sie werden über diesen Vorgang dann nicht mehr informiert.

    • Wenn Sie nicht möchten, dass Ihre Dateien unter "Dateien" ersetzt werden, wählen Sie Vorhandene Datei behalten. Die Datei wird nicht hochgeladen und Sie werden dann nicht mehr informiert.

  1. Wenn alle Dateien hochgeladen sind, klicken Sie auf OK. Die Dateien werden unter "Dateien" angezeigt.  

Hinweis: Für Ihren Account ist möglicherweise eine maximale Größe für hochgeladene Dateien definiert. Wenn eine Datei zu groß ist, wird eine Warnung angezeigt.
Dateien, die Sie hochladen, wirken sich auf die Kapazität Ihres Accounts aus.  

Einem Ordner "Dateien" Mail-Anhänge hinzufügen

Sie können Mail-Anhänge rasch unter "Dateien" speichern.

  1. Um einen Anhang hinzuzufügen, klicken Sie in der Mail-Kopfzeile neben dem Link des Anhangs auf den Link Dateien.

  2. Im Dialogfeld Zu Dateien hinzufügen wählen Sie den Zielordner aus oder legen einen neuen Ordner "Dateien" an.

  3. Klicken Sie auf OK.

Inhalt in den Vorschaufenstern anzeigen

Sie können auswählen, wie Sie die Dateien in einem Vorschaufenster in Ihren Ordnern "Dateien" anzeigen möchten.

  • Vorschaufenster unten/Vorschaufenster rechts Dateien mit einem Dateityp, der erkannt wird, werden im Vorschaufenster angezeigt. Sie können das Vorschaufenster ausblenden.

Dateien freigeben

Ordner "Dateien" können für andere Personen in Ihrer Firma und auch für externe Nutzer freigegeben werden. Sie können anderen  Nutzern die Berechtigungen erteilen, im Ordner "Dateien" Inhalte zu erstellen und zu bearbeiten - oder Sie vergeben nur eine Leseberechtigung.

Weitere Informationen zum Freigeben von Dateien finden Sie unter Dateien freigeben.
