Mit Dateien im Ordner "Dateien" arbeiten

Dateien im Ordner "Dateien" öffnen

Dateien, die sich in Ihrem Ordner "Dateien" befinden, können Sie auf jedem Computer öffnen. Sie benötigen die entsprechende Softwareanwendung, um die Datei zu öffnen, zu lesen und zu ändern. Häufig genutzte Dateiformate wie Textdateien, HTML-Dateien und Bilder in den Formaten .gif oder .jpg können mit einer Vielzahl von Programmen geöffnet werden.

Um eine Datei zu öffnen, klicken Sie mit der rechten Maustaste auf die Datei und wählen Öffnen aus.

Hinweis: Durch Doppelklicken wird die Datei ebenfalls geöffnet.

  • Wenn der Dateityp unbekannt ist, wird ein Dialogfeld angezeigt, in dem Sie gefragt werden, was Sie mit der Datei vorhaben, ob Sie sie diese Datei auf Festplatte speichern oder in einer bestimmten Anwendung öffnen möchten.

    1. Wenn Sie die Dateien für andere freigegeben haben, sollten Sie die Datei lieber auschecken, statt sie zu öffnen, damit keine anderen Benutzer die Datei im Ordner "Dateien" bearbeiten können.

    2. Klicken Sie mit der rechten Maustaste auf die Datei und wählen Sie dann Datei auschecken. Neben der Datei wird ein Vorhängeschloss-Symbol angezeigt.

    3. Wählen Sie im Dialogfeld "Herunterladen" die Option Datei zum Bearbeiten auschecken.. Klicken Sie auf Herunterladen.

    4. Nachdem Sie Ihre Änderungen vorgenommen haben, laden Sie die Datei hoch. Im Dialogfeld Konflikt beim Hochladen werden Sie gefragt, ob Sie die neue Version hochladen oder die Version im Ordner "Dateien" beibehalten wollen. Wählen Sie Neu und klicken Sie auf OK.

  • Wenn der Dateityp ein Zimbra-Dokument ist, wird die Datei zur Ansicht auf einer neuen Seite geöffnet. Sie können in dieser Ansicht kein Zimbra-Dokument bearbeiten.

Zimbra-Dokumentendatei bearbeiten

  • Klicken Sie mit der rechten Maustaste auf die Zimbra-Dokumententdatei und wählen Sie Dokument bearbeiten. Die Datei wird auf einer neuen Browserseite geöffnet.

  • Bearbeiten Sie die Seite und und klicken Sie zum Schließen der Seite auf Speichern und Schließen. Wenn Sie Ihre Arbeit nicht speichern wollen, schließen Sie die Browser-Registerkarte.

Hinweis: Wenn Sie auf Speichern oder Speichern und Schließen klicken, wird eine neue Version der Datei erstellt

Datei umbenennen

  • Klicken Sie mit der rechten Maustaste auf die Datei und wählen Sie Umbenennen.

Dateien verschieben

Sie können Dateien von einem Ordner "Dateien" in einen anderen verschieben.

  1. Wählen Sie die Datei(en), die verschoben werden soll(en), aus.

  2. Klicken Sie in der Symbolleiste auf .

  3. Wählen Sie den Zielordner oder erstellen Sie einen neuen Ordner "Dateien".

  4. Klicken Sie auf OK.

Dateien mit Tags markieren

Sie können Dateien mit Tags markieren.

  1. Klicken Sie auf die gewünschte Datei und dann in der Symbolleiste auf .

  2. Wählen Sie den Tag aus. Tags werden nur in der Detailansicht angezeigt.

  3. Klicken Sie auf OK.
