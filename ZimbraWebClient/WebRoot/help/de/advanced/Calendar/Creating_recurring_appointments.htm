Wiederkehrende Termine anlegen

Wiederkehrende Termine sind Termine, die mit gewisser Regelmäßigkeit stattfinden. Sie können wiederkehrende Besprechungen für die tägliche, wöchentliche, monatliche oder jährliche Wiederholung planen. Außerdem können Sie wiederkehrende Termine benutzerdefiniert planen.

  1. Klicken Sie in einer beliebigen Ansicht, mit Ausnahme der Ansicht "Einstellungen", in der Symbolleiste auf den Pfeil von und wählen Sie Termin aus.

Wenn Sie einen Termin auf andere Art einrichten möchten, klicken Sie hier.

  1. Wird das Feld Von angezeigt, so können Sie daraus ersehen, dass Sie mehr als eine Persönlichkeit für Ihren Account erstellt haben. Wählen Sie die Identität, die bei dem Anlegen dieses Termins verwendet werden soll.

  2. Das Feld Betreff ist ein Pflichtfeld. Geben Sie eine Beschreibung ein. Der Termin wird mit dieser Beschreibung im Kalender angezeigt.

  3. Geben Sie die Namen der Teilnehmer ein. Wenn Sie Namen eingeben, werden auf dem Minikalender im Übersichtsfenster alle vorgeschlagenen Uhrzeiten in Grün angezeigt, zu denen alle Teilnehmer verfügbar sind und zu denen Räume frei sind.  Die Namen der Teilnehmer können Sie auf eine der folgenden Arten eingeben:

    • Geben Sie im Textfeld Teilnehmer die Mail-Adressen getrennt durch Semikolons (;) ein. Während Sie tippen, werden übereinstimmende Namen in Ihren Kontaktlisten angezeigt. Der Planer zeigt alle Frei/Belegt-Informationen, die verfügbar sind.

    • Klicken Sie auf Teilnehmer, um nach Adressen zum Hinzufügen zu suchen.

    • Öffnen Sie den Planer, um Frei/Belegt-Informationen bei der Eingabe der Teilnehmernamen zu sehen. Wenn die Pläne der Teilnehmer bekannt sind, erscheint die Verfügbarkeit in den horizontalen Leisten neben den Namen.

Optionale Teilnehmer. Ist die Anwesenheit einiger Teilnehmer bei der Besprechung nicht unbedingt erforderlich, möchten Sie diese aber dennoch gerne einladen, dann wählen Sie Optionale anzeigen und geben Sie die Mail-Adressen in das Feld ein, oder klicken Sie im Planer auf das Symbol neben dem Namen und wählen Sie Optionaler Teilnehmer.  Nach dem Senden der Einladung werden die Namen dieser Teilnehmer als optionale Teilnehmer in der Liste angezeigt.

  1. Klicken Sie auf Ort, um einen Raum zu suchen, der als Ressourcen-Account angelegt wurde und der in Ihrem Firmenverzeichnis aufgeführt ist, oder geben Sie im Feld "Ort" eine Beschreibung über die Lage des Raums an, beispielsweise die Adresse, Gebäudenummer oder Konferenzraumnummer.  Klicken Sie auf Ressourcen anzeigen, um die zu reservierenden Geräte auszuwählen.

  2. Klicken Sie auf Anzeige, um festzulegen, wie Ihre Verfügbarkeit während dieser Zeit im Planer angezeigt werden soll. Wählen Sie entweder Frei, Vorläufig, Belegt oder Außer Haus.

  3. Wenn Sie mehrere Kalender haben, wählen Sie im Menü Kalender aus, in welchem Kalender das Ereignis eingerichtet werden soll.

  4. Geben Sie Anfangs- und Endzeit ein.

Hinweis: Wenn eine Zeitzone angezeigt wird, ist dies die Zeitzone, in der Sie sich befinden. Normalerweise müssen Sie die Zeitzone nicht ändern. Wenn Sie Besprechungen planen, bei denen sich die Teilnehmer in verschiedenen Zeitzonen befinden, wird in der Einladung die Uhrzeit für die Besprechung in der Zeitzone des jeweiligen Teilnehmers angezeigt. Wenn Sie z. B. eine Besprechung mit Teilnehmern in Hamburg und London planen, zeigt die Einladung an die Teilnehmer in Hamburg die Uhrzeit als GMT+1 und die Einladung an die Teilnehmer in London die Uhrzeit als GMT an (eine Stunde Unterschied).

  1. Klicken Sie auf Wiederholung, um das Wiederholungsmuster anzuzeigen. täglich, wöchentlich, monatlich, jährlich oder benutzerdefiniert. Die Besprechung ist ab dem Anfangsdatum und der Uhrzeit mit dem Wiederholungsmuster geplant. Für benutzerspezifische Wiederholungen können Sie spezielle Muster planen.

    • Jeden Tag: Für tägliche Wiederholungen können Sie angeben, ob die Besprechung jeden Tag oder in einer anderen Folge, z. B. jeden zweiten oder vierten Tag, stattfinden soll.

    • Jede Woche: Für wöchentliche Wiederholungen können Sie angeben, ob die Besprechung jede Woche oder in einem anderen Muster, z. B. jede zweite Woche, stattfinden soll.  Sie können auswählen, an welchem Wochentag die Besprechung stattfinden soll.

    • Jeden Monat: Für monatliche Wiederholungen können Sie angeben, ob der Termin jeden Monat oder in einem anderen Muster, z. B. jeden zweiten Monat, stattfinden soll.  Es wird ein Kalender angezeigt, damit Sie auswählen, an welchem Tag bzw. an welchen Tagen die Besprechung stattfinden soll.

    • Jedes Jahr: Für jährliche Wiederholungen wählen Sie den Monat und das Datum aus.

Klicken Sie auf Anpassen, um ein Wiederholungsmuster auszuwählen.

  1. Sie können den Zeitpunkt ändern, zu dem die Erinnerungsmail gesendet werden soll, indem Sie auf das Feld Erinnerung klicken. Wenn die Erinnerung an eine Mailadresse oder als SMS gesendet werden soll, aktivieren Sie das entsprechende Kontrollkästchen.

  2. Im Feld Text können Sie weitere Informationen angeben, die in der Mail enthalten sein sollen. Um Anhänge hinzuzufügen, klicken Sie in der Symbolleiste auf Anhänge hinzufügen.

  3. In der Symbolleiste unter Optionen können Sie das Versenden von Antworten auf Fragen und das Versenden einer Benachrichtigungsmail deaktivieren. Darüber hinaus können Sie den Termin als "Privat" kennzeichnen ...

  4. Klicken Sie auf Senden. Alle Teilnehmer werden per Mail eingeladen.

Wenn Sie die Einladung zu einem späteren Zeitpunkt versenden wollen, klicken Sie auf Speichern. Dadurch wird eine lokale Kopie des Termins gespeichert, es werden jedoch noch keine Einladungen an die Teilnehmer versendet.
