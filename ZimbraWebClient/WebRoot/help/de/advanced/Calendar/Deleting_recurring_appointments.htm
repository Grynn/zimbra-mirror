Wiederkehrende Termine ändern

Wenn Sie wiederkehrende Besprechungen angelegt haben, können Sie einen Einzeltermin oder die Besprechungsserie ändern oder löschen. Die Teilnehmer werden per Mail benachrichtigt.

So ändern Sie eine Besprechung

  1. Doppelklicken Sie auf die zu ändernde Besprechung oder klicken Sie mit der rechten Maustaste darauf und wählen Sie Löschen.

  2. Wählen Sie aus, ob Sie nur das ausgewählte Datum oder die gesamte Serie öffnen möchten. Daraufhin wird die Seite Termin angezeigt.

  3. Geben Sie Ihre Änderungen ein. Sie können den Plan oder die Teilnehmer ändern, im Feld Text eine Erläuterung hinzufügen oder einen Anhang anhängen.

  4. Klicken Sie auf OK. Die Teilnehmer werden per Mail benachrichtigt.

So löschen Sie eine wiederkehrende Besprechung

  1. Klicken Sie auf den Termin.

  2. Klicken Sie mit der rechten Maustaste und wählen Sie Löschen oder ziehen Sie das Element auf das Symbol .

  3. Geben Sie an, ob Sie den Einzeltermin oder die Terminserie löschen möchten.

  4. Die Teilnehmer werden per Mail benachrichtigt, und der Termin wird aus den Kalendern der Teilnehmer gelöscht.
