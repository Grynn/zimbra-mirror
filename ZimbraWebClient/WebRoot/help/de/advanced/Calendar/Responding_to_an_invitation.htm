Einladung zu einer Besprechung beantworten

Wenn Sie eine Einladung zu einer Besprechung erhalten, erhalten Sie zugleich eine Mail-Benachrichtigung in Ihr Postfach. Wenn Sie die Kalendereinstellungen Empfangene Termine automatisch in Kalender einfügen aktiviert haben, wird diese Besprechung Ihrem Standardkalender ebenfalls hinzugefügt und als Neu markiert. Sie können entweder im Posteingang oder in der Kalenderliste auf die Einladung antworten.

Hinweis: Wenn Sie eine Mail-Einladung als .ics-Anhang erhalten, klicken Sie in der Kopfzeile auf Zum Kalender hinzufügen, um die Einladung auf Ihrem Kalender zu speichern.  Wählen Sie den Kalender aus, in dem die Einladung gespeichert werden soll.

So reagieren Sie auf eine Einladung

  1. Öffnen Sie die Mail, die die Besprechungsanfrage enthält, im Posteingang auf oder klicken Sie mit der rechten Maustaste auf Kalender.

  2. Um auf die Einladung zur Besprechung zu reagieren, klicken Sie auf  Vorläufig, Annehmen oder Ablehnen. Wenn der Ersteller eine Antwort eingestellt hat, wird ihm automatisch eine zugesendet. Bevor Sie die Antwort abschicken, können Sie Kommentare ergänzen. Sie können den Kalender auswählen, in dem die Konferenz geplant sein soll. Wenn Sie keinen bestimmten Kalender auswählen, wird Ihre Besprechung im Standard-Kalender gespeichert.

Um bei der Annahme einer Einladung aus der Kalenderansicht einer Antwort Kommentare hinzuzufügen, klicken Sie mit der rechten Maustaste auf die Einladung, und wählen Sie dann Antwort bearbeiten.

Wenn die Zeit für die Besprechung nicht gut war, Sie aber gern an der Besprechung teilnehmen würden, können Sie eine neue Besprechungszeit vorschlagen. Klicken Sie auf Neue Uhrzeit vorschlagen, um eine andere Zeit auszuwählen. Der Besprechungsorganisator kann den Vorschlag annehmen oder ablehnen.

Nachdem Sie Ihre Wahl getroffen haben, wird die Mail-Nachricht in den Papierkorb verschoben.

Standardmäßig wird die Mail als Öffentlich markiert. Wenn der Ersteller die Mail als "Privat" gekennzeichnet hat, wird die Einladung als privat empfangen, wenn Ihre Einstellungen so eingestellt ist, dass alle Mails als "Privat" gekennzeichnet werden, wird die Mail im Kalender als "Privat" gekennzeichnet.

Klicken Sie auf Ihrem Kalender auf den Besprechungshinweis, um die Einladungen zu lesen und möglicherweise mitgesendete Anhänge zu sehen.

Sie können die Einladung zu einem anderen Kalender verschieben, wenn Sie sie nicht auf Ihrem Standardkalender wünschen. Klicken Sie oben in der Mail auf Kalender und wählen Sie den Kalender aus.

Hinweis: Wenn der Ersteller eine Besprechung ändert oder storniert, die Sie angenommen bzw. vorläufig angenommen haben, werden die Änderungen automatisch auf Ihrem Kalender vermerkt.  Der Ersteller kann sich entscheiden, Sie über diese Änderungen zu informieren bzw. Sie entsprechend nicht zu informieren.

Ihre Kalendereinstellungen für Besprechungseinladungen ändern

Wenn Sie nicht wollen, dass Besprechungen, die Sie nicht akzeptiert haben, automatisch Ihrem Kalender hinzugefügt werden, können Sie die Option Empfangene Termine automatisch in Kalender einfügen auf der Kalender-Einstellungsseite deaktivieren. Besprechungen werden Ihrem Kalender nur hinzugefügt, wenn Sie bei der Mail-Einladung auf "Annehmen" oder "Vorläufig" geklickt haben. Wenn Sie auf "Ablehnen" klicken, wird der Termin Ihrem Kalender nicht hinzugefügt.

Wenn Empfangene Termine automatisch in Kalender einfügen aktiviert ist, werden abgelehnte Termine auf Ihrem Kalender in einer Ausblendansicht dargestellt, als Erinnerung an die Besprechung, die Sie abgelehnt haben.  Abgelehnte Besprechungen können Sie jederzeit löschen.

Per Standard werden Einladungsmails für Besprechungen in Ihrem Posteingang in den Papierkorb verschoben, wenn Sie nicht auf die Einladung antworten.  Wenn Sie wollen, dass die Mail in Ihrem Posteingang bleibt, deaktivieren Sie in Ihren Kalendereinstellungen die Option Nach der Beantwortung einer Einladung ... Einladung bei Antwort löschen.
