Ganztägige Ereignisse planen

Sie können ein ganz- oder mehrtägiges Ereignis, z. B. eine Konferenz, erstellen. Das Ereignis wird am Veranstaltungstag am oberen Rand des Kalenders angezeigt.

Ganztägigen Termin planen

  1. Klicken Sie in einer beliebigen Ansicht, mit Ausnahme der Ansicht „Einstellungen“, auf den Pfeil von und wählen Sie Termin aus.

  2. Geben Sie den Betreff ein. Der Betreff wird als Beschreibung im Kalender angezeigt und muss eingegeben werden.

  3. Geben Sie einen Ort ein. Sie können jeden beliebigen Ort eingeben. Orte, die von Ihrem Systemadministrator als Ressourcen angelegt wurden, können reserviert werden. Geben Sie den Namen des Ortes, sofern Sie ihn kennen, in das Feld „Ort“ ein. Wenn Sie den Namen nicht kennen, können Sie auf der Registerkarte Orte suchen nach dem Ort suchen. Wenn Sie den Ort auswählen, wird er in diesem Feld angezeigt.

  4. Geben Sie das Anfangs- und Enddatum ein. Aktivieren Sie rechts von der Uhrzeit das Kontrollkästchen Ganztägiges Ereignis.  

  5. Wenn Sie mehrere Kalender haben, wählen Sie in der Dropdown-Liste Kalender aus, in welchem Kalender das Ereignis eingerichtet werden soll.

  6. Fügen Sie Ressourcen hinzu. Geben Sie den Namen der Ressource, sofern Sie sie kennen, in das Feld „Ressource“ ein. Wenn Sie den Namen nicht kennen, können Sie auf der Registerkarte Ressourcen suchen nach der Ressource suchen.

  7. Geben Sie die Namen der Teilnehmer ein. Die Namen der Teilnehmer können Sie auf eine der folgenden Arten eingeben:

    • Öffnen Sie die Registerkarte Teilnehmer suchen. Geben Sie einen Namen ein und wählen Sie die zu verwendende Liste aus, entweder „Kontakte“ oder „Globale Adressliste“. Wählen Sie die Namen aus und klicken Sie auf Hinzufügen. Klicken Sie abschließend auf OK.

    • Geben Sie im Textfeld Teilnehmer die Mail-Adressen getrennt durch Semikolons (;) ein. Während Sie tippen, werden übereinstimmende Namen aus Ihrer Kontaktliste angezeigt.

    • Um die Frei/Belegt-Zeiten der Teilnehmer anzuzeigen, öffnen Sie die Registerkarte Teilnehmer planen. Während Sie Namen und Mail-Adressen der Teilnehmer eingeben, wird deren Verfügbarkeit in horizontalen Leisten neben den Namen angezeigt, sofern die Planung der Teilnehmer bekannt ist. Wenn die Liste der Teilnehmer vollständig ist, kehren Sie zur Registerkarte Termin zurück.  

  8. Geben Sie zusätzliche Informationen zu der Besprechung in das Textfeld ein. Um Anhänge hinzuzufügen, klicken Sie in der Symbolleiste auf Anhänge hinzufügen. Der Anhang wird dann mit der Mail gesendet.

  9. Klicken Sie auf Speichern. Alle Teilnehmer werden per Mail eingeladen. Das Ereignis wird als Banner am oberen Rand des Kalenders angezeigt.

  10. Wenn Sie einen Termin für sich selbst, ohne Teilnehmer, in Ihrem Kalender anlegen, überspringen Sie Schritt 7.
