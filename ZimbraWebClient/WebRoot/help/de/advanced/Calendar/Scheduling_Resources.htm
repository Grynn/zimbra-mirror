Ressourcen planen

Als Ressourcen werden ein Ort (Tagungsraum, Gebäude oder anderer Ort) oder ein Gerät (audio/visuell) angesehen, die für eine Besprechung geplant werden können. Beim Einrichten einer Besprechung wählen Sie zum Planen einer Ressource die entsprechende Ressource aus und "laden" Sie zur Besprechung ein.  Die Ressource erhält die Einladung und nimmt sie an, sofern sie frei ist.  

Administratoren richten Ressourcenaccounts ein und konfigurieren die Planungsregeln der Ressource. Eine Besprechung kann abgelehnt werden, wenn die Ressource bereits über eine geplante Besprechung für diesen Zeitraum verfügt.  In einigen Fällen wird die Ressourcenplanungsregel so eingerichtet, dass eine Ressource Datumsangaben in wiederkehrenden Besprechungen annehmen kann, selbst wenn es zu Konflikten kommt.  Wenn Sie eine Ressource zu einer wiederkehrenden Besprechung einladen und einige der Daten bereits geplant sind, wird ein Dialogfeld "Ressourcenkonflikt" mit dem Datum und der Uhrzeit, die den Konflikt verursachen, eingeblendet.

Wenn für die wiederkehrende Besprechung eine Ressource geplant wurde, sie jedoch über Datumsangaben verfügt, die bereits von einer anderen Besprechung geplant sind, erhalten Sie ebenfalls eine Mail, die angibt, welche Datumsangaben einen Konflikt verursachen.
