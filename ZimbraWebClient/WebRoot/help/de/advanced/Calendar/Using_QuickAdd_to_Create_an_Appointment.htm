Termin über Schnelleingabe anlegen

Im Dialogfeld "Schnelleingabe" können Sie rasch einen Termin anlegen. Wählen Sie in Ihrem Kalender eine Anfangsuhrzeit und doppelklicken Sie auf den Zeitraum bzw. ziehen Sie die Maus über den Zeitraum. Daraufhin wird das Dialogfeld "Schnelleingabe Termin" geöffnet.

Die Schnelleingabe ist eine Option. Wenn Sie möchten, dass beim Doppelklicken oder Ziehen im Kalender immer die Terminseite geöffnet wird, deaktivieren Sie diese Funktion unter "Kalendereinstellungen".

So legen Sie schnell einen Termin an

  1. Wählen Sie den Kalender für den Termin aus. Klicken Sie auf die Zeit des Termins oder klicken Sie und ziehen Sie  von der Startzeit bis zur Endzeit für den Termin.  Das Dialogfeld Schnelleingabe Termin wird eingeblendet.

Hinweis: Um in der Tages- oder Wochenansicht einen ganztägigen Termin anzulegen, klicken und ziehen Sie die Maus vom oberen Rand des Kalenders über einen oder mehrere Tage.

  1. Geben Sie Betreff und Ort ein. Wenn es sich um einen wiederkehrenden Termin handelt, geben Sie die Wiederholungsart ein.

  2. Stellen Sie den Erinnerungshinweis auf die Zeit ein, zu der Sie über den Besprechungsstart erinnert werden wollen.

  3. Klicken Sie auf OK. Der Termin wird im Kalender angezeigt.

Wenn Sie Teilnehmer einladen müssen oder die Optionen für die Wiederholung ändern möchten, klicken Sie auf Mehr Details, um die Terminseite zu öffnen.
