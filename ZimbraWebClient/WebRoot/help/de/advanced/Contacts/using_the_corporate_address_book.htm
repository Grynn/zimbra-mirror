Mit dem Firmenadressbuch arbeiten

Auch innerhalb Ihrer Firma kennen Sie vielleicht nicht jede Mail-Adresse. Johann Schmidt bei der Beispiel GmbH kann z. B. die Adresse johann@beispiel.de, oder jschmidt@beispiel.de oder sogar js123@verkauf.beispiel.de haben.

Beim Verfassen von Mails stehen üblicherweise zwei Arten von Adressbüchern zur Verfügung: ein Firmenadressbuch (wird auch globale Adressliste, GAL, genannt) und Ihre persönlichen Adressbücher. Das Firmenadressbuch steht jedem Nutzer zur Verfügung. Ihre persönlichen Adressbücher hingegen stehen nur Ihnen und den Nutzern, für die Sie diese freigegeben haben, zur Verfügung.

So fügen Sie Namen aus einem Adressbuch in eine neue Mail ein

  1. Öffnen Sie eine neue Mail, indem Sie im Menü auf klicken.

  2. Klicken Sie auf An. Es wird ein Dialogfeld zum Auswählen von Adressen angezeigt.

  3. Wählen Sie unter Namen anzeigen von entweder Kontakte oder Globale Adressliste aus, um darin zu suchen.

  4. Geben Sie einen Namen oder den Teil eines Namens in das Suchfeld links ein und klicken Sie auf Suchen. Auf der linken Seite wird eine Liste mit Namen und Mail-Adressen angezeigt.

  5. Doppelklicken Sie auf einen Namen, um ihn zur Liste der Empfänger auf der rechten Seite hinzuzufügen. Mithilfe der Schaltflächen An, CC und BCC legen Sie fest, in welchem Adressfeld der Name in der Mail erscheint. Klicken Sie auf die gewünschte Schaltfläche, bevor Sie den Namen übernehmen.

  • Mithilfe der Schaltfläche Entfernen löschen Sie einen Namen in der Liste der Empfänger auf der rechten Seite.

  1. Wiederholen Sie die Suche nach Bedarf, bis Sie alle Namen hinzugefügt haben.

  2. Klicken Sie abschließend auf OK.
