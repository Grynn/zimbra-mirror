Im HTML-Editor arbeiten

Sie können Ihre Mails durch die Verwendung des HTML-Mail-Editors mit Formatierungen gestalten. Mails, die im HTML-Format verfasst werden, können formatierten Text, Nummerierungen, Aufzählungszeichen, farbigen Hintergrund, Tabellen und Links enthalten, wodurch der Text leichter lesbar wird.

  • Einige Mail-Clients können keine HTML-Mails anzeigen.

Über die Registerkarten Einstellungen, Allgemein im Web-Client stellen Sie den HTML-Editor als Ihren Standardeditor ein. Hier können Sie auch die Schriftart einschließlich Stil, Größe und Farbe vorgeben.
