Adressen in die Felder „CC“ und „BCC“ einfügen

Die Abkürzungen CC und BCC stehen für „Carbon Copy“ (Kopie) und „Blind Carbon Copy“ (Blindkopie).

  • Über das Feld CC können Sie einem interessierten Nutzer eine Kopie einer Mail senden, die an einen anderen Nutzer als Hauptempfänger gerichtet ist. Alle Empfänger im Feld CC sehen die vollständige Liste der Empfänger, wenn sie die Mail lesen.

  • Über das Feld BCC können Sie einem Nutzer eine Kopie einer Mail senden, ohne dass die anderen Empfänger der Mail die Adresse dieses Nutzers sehen.

So geben Sie Adressen ins Feld „BCC“ einer neuen Mail ein

  1. Klicken Sie im Fenster zum Verfassen einer Mail in der Symbolleiste zum Verfassen auf Optionen. Wählen Sie BCC-Feld anzeigen.

  2. Klicken Sie bei Bedarf neben den Adressfeldern auf eine der Schaltflächen An, CC oder BCC, um das Dialogfeld zum Auswählen von Adressen aufzurufen. Geben Sie ansonsten die Adressen direkt in die gewünschten Felder ein.

Sie können eine Mail ohne Adressen im Feld „An“ senden, vorausgesetzt, es befindet sich mindestens eine Adresse in einem der Felder „CC“ oder „BCC“.
