Neues Dokument erstellen

Sie können eine Seite im Ordner "Dateien" anlegen und deren Inhaltslayout mithilfe der Funktionen für Rich-Text-Format gestalten. Sie können die Rechtschreibung auf der Seite überprüfen und über Suchbegriffe problemlos Links auf andere Seiten erstellen.

  1. Klicken Sie auf den Ordner "Dateien", in dem Sie arbeiten wollen. Der Fensterbereich Dokumentenliste zeigt die Dateien an, die sich bereits im Ordner "Dateien" befinden.

  2. Wählen Sie in der Symbolleiste aus der Liste Neu die Option Dokument aus. Daraufhin wird eine leere Seite "Zimbra-Dokumente" geöffnet.

  3. Geben Sie in die Symbolleiste den Namen für das Dokument ein.

  4. Verfassen Sie den Inhalt der Seite. Standardmäßig arbeiten Sie im so genannten Rich-Text-Editor, mit dem Sie zu Ihrer Seite Stile, Farben und Tabellen hinzufügen können.

  1. Klicken Sie auf Speichern, um die Datei zu speichern und mit der Arbeit fortzufahren. Klicken Sie auf der Browser-Registerkarte auf das X, um das Dokument ohne zu speichern zu schließen. Klicken Sie auf Speichern und Schließen, um die Datei zu speichern und zurückzukehren zur Seite "Dokumentenliste".

Wenn Sie ein Dokument speichern, ändert sich die Dokumentversion.

Hinweis: Wenn Sie das Dokument im falschen Ordner „Dateien“ erstellt haben, können Sie die Datei in den richtigen Ordner verschieben. Dazu wählen Sie das Dokument aus und ziehen es in den richtigen Ordner.
