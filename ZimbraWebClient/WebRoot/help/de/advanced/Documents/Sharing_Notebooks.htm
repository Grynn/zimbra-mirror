Notizbücher freigeben

Sie haben für die Freigabe Ihrer Notizbücher, die sich unter „Notizen“ befinden, folgende Möglichkeiten:

  • Interne Nutzer oder Gruppen: Sie legen fest, welche Rechte die internen Nutzer erhalten:

    • Leseberechtigung: Der Nutzer kann den Inhalt des Notizbuchs lesen, er kann ihn jedoch nicht ändern.

    • Manager: Der Nutzer hat vollständige Zugriffsrechte und kann den Inhalt des Notizbuchs anzeigen und ändern, neue Seiten hinzufügen oder Seiten löschen.

  • Externe Nutzer: Sie richten ein Passwort für das Notizbuch ein, das die Nutzer eingeben müssen, um die Seiten anzuzeigen. Änderungen an den Daten können nicht vorgenommen werden.

  • Öffentlich: Jeder, der die URL des Notizbuchs kennt, kann sich die Seiten ansehen. Änderungen an den Daten können nicht vorgenommen werden.

So geben Sie Ihr Notizbuch frei

Sie können ein Notizbuch auf einer beliebigen Ebene für die Freigabe auswählen. Wenn Sie das oberste Notizbuch freigeben, werden alle darin enthaltenen Notizbücher ebenfalls freigegeben.

  1. Klicken Sie mit der rechten Maustaste auf das Notizbuch, das Sie freigeben möchten.

  1. Klicken Sie auf Notizbuch freigeben. Daraufhin wird das Dialogfeld Eigenschaften der Freigabe angezeigt. Geben Sie an, wem Sie die Freigabe erteilen möchten: internen Nutzern oder Gruppen, externen Nutzern oder öffentliche Freigabe.

  2. Je nach Freigabe gehen Sie folgendermaßen vor:

  • Bei der Option Interne Nutzer oder Gruppen geben Sie die Mail-Adressen ein und wählen unter Rolle die Zugriffsrechte aus.

  • Bei der Option Externe Nutzer geben Sie die Adressen der externen Nutzer und das Passwort für den Zugriff auf Ihr Notizbuch ein.

  • Bei der Option Öffentlich teilen Sie den öffentlichen Nutzern die URL mit, die unter URL angezeigt wird. Sie können z. B. diese URL kopieren und den Nutzern per Mail senden, die sich die Seite ansehen sollen.

  1. Wählen Sie im Bereich Mail aus, welche Art von Mail gesendet werden soll. Diese Mail kann an interne und externe Nutzer gesendet werden. Es gibt folgende Möglichkeiten:

    • Keine Mail über diese Freigabe senden: Es wird keine Mail gesendet.

    • Standardmail senden: Die Standardmail mit der Freigabebenachrichtigung wird an die Mail-Adresse gesendet. Diese Mail enthält den Namen des freigegebenen Notizbuchs, Ihre Adresse, die Adresse des Nutzers, dem die Freigabe erteilt wird, und die zugewiesene Rolle mit einer Liste der zulässigen Aktionen.

    • Notiz zur Standardmail hinzufügen: Sie können die Standardmail um zusätzliche Informationen erweitern.

    • Mail in neuem Fenster verfassen: Verfassen und senden Sie eine neue Mail.

  1. Klicken Sie auf OK.

  2. Beachten Sie, dass Sie den jeweiligen Nutzern die URL mitteilen müssen, wenn Sie die Option „Öffentlich“ ausgewählt haben.
