Funktionen und Tastenkürzel der Benutzeroberfläche

Die Benutzeroberfläche umfasst eine Vielzahl von komfortablen Funktionen.

  • Dropdown-Menüs. Einige Befehlsschaltflächen enthalten einen kleinen, nach unten gerichteten Pfeil. Wenn Sie auf die Schaltfläche klicken, wird eine Aktion ausgeführt. Wenn Sie auf den Pfeil klicken, werden weitere Menüoptionen angezeigt.

        

  • Mit der Maus ziehen. Klicken Sie auf Mails bzw. Unterhaltungen und ziehen Sie sie mit der Maus von einem Ordner in einen anderen.

  • Kontextmenüs: Wenn Sie mit der rechten Maustaste auf Objekte wie Ordner, Unterhaltungen, Mails, die Spalte "Von" in Mails oder Unterhaltungen oder auf Elemente von Mailkopfzeilen klicken, wird ein Kontextmenü mit Befehlen angezeigt, die Sie für das Objekt durchführen können.

  • Verschiebbare Fensterteiler: Bei unterteilten Fenstern wie der Unterhaltungsansicht können Sie mithilfe des Fensterteilers die relative Größe des oberen und unteren Teilfensters vergrößern bzw. verkleinern. Klicken Sie hierzu auf den Fensterteiler und ziehen Sie ihn mit der Maus.

 

  • Klickbare Objekte. @CompanyName@ verhält sich wie ein herkömmlicher Browser und zeigt den Mauszeiger in Form einer "Hand" an, wenn Sie ihn über einen aktiven Link bewegen.

  • Tooltips: Wenn Sie Ihren Mauszeiger über Elemente wie Befehlsschaltflächen, Unterhaltungen oder Mail-Adressen in Mailkopfzeilen bewegen, wird ein kleines Textfeld mit einer Beschreibung zu dem Element angezeigt.
