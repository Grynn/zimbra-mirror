Im Ordner „Dateien“ arbeiten

Im Ordner „Dateien“ können Sie Dateien speichern, die Sie nutzen möchten. Sie können auf diese Dateien zugreifen, wenn Sie sich auf einem beliebigen Computer an Ihrem Account anmelden.  

Sie können Dokumente, Kalkulationstabellen, Präsentationen, Bilder, PDF-Dateien von Ihrem Computer oder Ihrem Computer-Netzwerk hochladen, und Sie können mit Ihren Mails versendete Anhänge unter „Dateien“ speichern. Sie haben die Möglichkeit, verschiedene Ordner unter „Dateien“ anzulegen, um Ihre Dateien zu organisieren, und Sie können Ihre Dateien für andere Nutzer freigeben.

Die Dateien in diesem Ordner sind Kopien der ursprünglichen Datei. Änderungen, die Sie an den Dateien in diesem Ordner vornehmen, haben keine Auswirkungen auf die ursprüngliche Datei.

Ordner unter „Dateien“ anlegen

  1. Klicken Sie auf die Registerkarte „Dateien“.

  2. Klicken Sie in der Symbolleiste auf Neue Dateien.

  3. Daraufhin wird das Dialogfeld Neuen Ordner in Dateien erstellen angezeigt. Geben Sie einen Namen für den neuen Ordner ein.

Dieser Name muss innerhalb der Hierarchie Ihrer Mailbox-Ordner eindeutig sein. Ordner unter „Dateien“ dürfen nicht denselben Namen haben wie ein übergeordneter Ordner in der Mail, im Kalender oder im Adressbuch von Zimbra. Wenn Sie beispielsweise einen Kalender mit dem Namen „Urlaub“ haben, dürfen Sie keinem übergeordneten Ordner unter „Dateien“ den Namen „Urlaub“ geben. Ein Ordner unter „Dateien“, der sich in einem untergeordneten Ordner befindet, darf „Urlaub“ heißen.

  1. Wählen Sie die Ordnerhierarchie aus. Sie können einen neuen übergeordneten Ordner anlegen oder den Ordner in einen vorhandenen Ordner einfügen.

  2. Klicken Sie auf OK. Der neue Ordner wird in der Übersicht angezeigt.

Dateien hochladen

Beim Hochladen wird eine Datei von Ihrem Computer in Ihren ZWC-Account übertragen. Dadurch steht die Datei jederzeit zur Verfügung, wenn Sie sich an Ihrem Account anmelden.

So laden Sie Dateien hoch

  1. Wechseln Sie zur Registerkarte „Dateien“ und klicken Sie in der Symbolleiste auf „Datei hochladen“.

  2. Klicken Sie im Dialogfeld „Neue Datei in Dateien hochladen“ auf „Durchsuchen“, um die gewünschte Datei zu finden. Um eine weitere Datei hinzuzufügen, klicken Sie auf Hinzufügen.

Sie können keine Datei hochladen, die denselben Namen hat wie eine Datei, die bereits unter „Dateien“ vorhanden ist. Sie können im Dialogfeld „Hochladen“ angeben, wie beim Hochladen von Dateien mit demselben Namen vorgegangen werden soll. Wenn Sie diese Option ändern, wird die Änderung im Dialogfeld gespeichert.

  • Wenn Sie benachrichtigt werden möchten, wenn Sie eine Datei mit demselben Namen hochladen, wählen Sie Nachfragen aus. Wenn eine Datei denselben Namen wie eine bereits hochgeladene Datei hat, wird das Dialogfeld Konflikt beim Hochladen angezeigt. Sie können die vorhandene Datei behalten oder durch die neue Datei ersetzen. Sie können sich die neue Datei auch ansehen. Dies ist die Standardeinstellung.

  • Wenn Sie möchten, dass die hochgeladene Datei immer die evtl. bereits vorhandene Datei gleichen Namens ersetzt, wählen Sie Mit meiner Datei ersetzen. Die neue Datei wird hochgeladen und ersetzt die vorhandene Datei. Sie werden über diesen Vorgang dann nicht mehr informiert.

  • Wenn Sie nicht möchten, dass Ihre Dateien unter „Dateien“ ersetzt werden, wählen Sie Vorhandene Datei behalten. Die Datei wird nicht hochgeladen und Sie werden dann nicht mehr informiert.

  1. Wenn alle Dateien hochgeladen sind, klicken Sie auf OK. Die Dateien werden unter „Dateien“ angezeigt.  

  2. Für Ihren Account ist möglicherweise eine maximale Größe für hochgeladene Dateien definiert. Wenn eine Datei zu groß ist, wird eine Warnung angezeigt.
    Dateien, die Sie hochladen, wirken sich auf die Kapazität Ihres Accounts aus.  

Mail-Anhänge unter „Dateien“ speichern

Sie können Mail-Anhänge rasch unter „Dateien“ speichern.

  1. Um einen Anhang aufzunehmen, klicken Sie in der Mail-Kopfzeile neben dem Link des Anhangs auf den Link Zu Dateien hinzufügen.

  2. Im Dialogfeld Zu Dateien hinzufügen wählen Sie den Zielordner aus oder legen einen neuen Ordner unter „Dateien“ an.

  3. Klicken Sie auf OK.

Dateien im Ordner „Dateien“ anzeigen

Sie können auswählen, wie Sie die Dateien in Ihrem Ordner „Dateien“ anzeigen möchten. Wählen Sie in der Symbolleiste auf der Registerkarte „Dateien“ eine der folgenden Optionen im Menü Anzeigen.

  • Explorer-Ansicht: Die Dateien werden als Miniaturansichten angezeigt.

  • Detailansicht: Die Dateien werden in einer Zeile mit Dateigröße, Hochladedatum, Besitzer (bei einer freigegebenen Datei sind möglicherweise nicht Sie der Besitzer) und dem Ordner, in dem sich die Datei befindet, angezeigt.

  • Browser-Ansicht Spalten: Die Dateien werden in Spalten angezeigt. In dieser Ansicht können Sie auf eine Datei klicken, um Angaben zu der Datei wie Größe, Erstellungsdatum, Ersteller, Änderungsdatum und ändernder Nutzer, anzuzeigen.

Dateien im Ordner „Dateien“ öffnen

Dateien, die sich in Ihrem Ordner „Dateien“ befinden, können Sie auf jedem Computer öffnen. Sie benötigen die entsprechende Softwareanwendung, um die Datei zu öffnen, zu lesen und zu ändern. Häufig genutzte Dateiformate wie Textdateien, HTML-Dateien und Bilder in den Formaten *.gif oder *.jpg können mit einer Vielzahl von Programmen geöffnet werden.

  • Um eine Datei zu öffnen, klicken Sie mit der rechten Maustaste auf die Datei und wählen Öffnen aus. Wenn der Dateityp unbekannt ist, wird ein Dialogfeld angezeigt, in dem Sie gefragt werden, was Sie mit der Datei vorhaben, ob Sie sie auf Festplatte speichern oder in einer bestimmten Anwendung öffnen möchten.

Dateien verschieben

Sie können Dateien von einem Ordner „Dateien“ in einen anderen verschieben.

  1. Klicken Sie in der Symbolleiste auf .

  2. Wählen Sie den Zielordner oder erstellen Sie einen neuen Ordner unter „Dateien“.

  3. Klicken Sie auf OK.

Dateien mit Tags markieren

Sie können Dateien mit Tags markieren.

  1. Klicken Sie auf die gewünschte Datei und dann in der Symbolleiste auf .

  2. Wählen Sie den Tag aus. Tags werden nur in der Detailansicht angezeigt.

  3. Klicken Sie auf OK.

Mails mit einem Link auf eine Datei senden

Sie können Mails senden, die einen Link auf eine Datei im Ordner „Dateien“ enthalten. Die Empfänger benötigen die entsprechende Software, um die Datei zu öffnen.

  1. Klicken Sie mit der rechten Maustaste auf die gewünschte Datei.

  2. Wählen Sie Senden. Eine Bestätigungsmeldung wird angezeigt.

  3. Klicken Sie auf Ja.
