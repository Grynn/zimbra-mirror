Durch @ClientNameLong@ navigieren

Wenn Sie sich beim @ClientNameLong@-Client anmelden, wird eine Ansicht ähnlich der folgenden dargestellt.

 

Eine Seite des @ClientNameLong@-Clients umfasst die folgenden Bereiche:

  1. Kopfzeile - dieser Bereich umfasst die Links Leute suchen, um das Firmenverzeichnis zu durchsuchen, Ihren Namen, die Hilfe und den Link Abmelden links.

  2. Registerkarten der Anwendungen: Die Anwendungen, die Sie aufrufen können, sind auf den Registerkarten dieser Zeile aufgeführt.

  3. Suchleiste. Die Suchoptionen Suchen und Erweitert werden in diesem Bereich angezeigt. Sie können die Suchen auch von hier aus speichern.

  4. Symbolleiste: Sie zeigt Befehle für die aktuelle Anwendung an. In diesem Beispiel wird die Symbolleiste der Anwendung Mail angezeigt.

  5. Übersichtsfenster: Hier finden Sie Ihre Ordner, auch die Systemordner wie Posteingang, Gesendet, Entwürfe, Spam und Papierkorb sowie alle von Ihnen angelegten Ordner, Suchvorgänge, die Sie gespeichert haben, von Ihnen erstellte Tags und Zimlet-Links, die für die Integration von Fremdanwendungen in Ihre Mailbox erstellt wurden. In der Anwendung "Kalender" zeigt das Übersichtsfenster Ihre Kalenderliste und Zimlets an.

  6. Inhaltsfenster: Der Inhalt in diesem Bereich richtet sich nach der jeweils aufgerufenen Anwendung. Im Posteingang zeigt das Inhaltsfenster alle Mails und Unterhaltungen Ihres Posteingangs an.

  7. Lesefenster. Das Lesefenster wird zur Vorschau von Mails verwendet. Sie können Ihre Ansicht auch zu einer Drei-Fenster-Ansicht ändern, mit dem Lesefenster auf der rechten Seite, oder Sie können das Lesefenster vollständig aus der Ansicht in der Mail-Symbolleiste ausblenden.

  8. Unterhalb des Übersichtsfensters kann ein Mini-Kalender angezeigt werden. Die Anzeige des Mini-Kalenders ist optional und kann unter Kalender-Einstellungen aktiviert werden.
