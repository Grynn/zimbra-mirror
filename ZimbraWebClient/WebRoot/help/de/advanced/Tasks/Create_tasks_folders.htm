Aufgabenlisten anlegen

So legen Sie eine Aufgabenliste an

  1. Um Aufgaben zu öffnen, wählen Sie die Registerkarte Aufgaben aus.

  2. Klicken Sie mit der rechten Maustaste auf die Kopfzeile „Aufgaben“ und wählen Sie Neue Aufgabenliste aus.

  3. Geben Sie unter Neue Aufgabenliste erstellen einen Namen für die Aufgabenliste ein und wählen Sie eine Farbe zum Anzeigen der Aufgaben aus.

  4. Klicken Sie auf OK. Die Liste wird im Übersichtsfenster unter „Aufgaben“ hinzugefügt.
