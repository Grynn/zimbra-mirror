Persönlichkeiten anlegen

Wenn Sie mehrere Mail-Identitäten, so genannte Persönlichkeiten, anlegen, können Sie in Ihrer Mailbox mehrere Mail-Adressen nutzen. Dies ist nützlich, um verschiedene Rollen in Ihrem Beruf und Ihrem Privatleben über einen einzigen Mail-Account zu verwalten.

Um eine Persönlichkeit anzulegen, öffnen Sie die Registerkarte Einstellungen und richten Folgendes ein:

  1. Klicken Sie auf der Registerkarte Accounts auf Persönlichkeit hinzufügen.

  2. Geben Sie unter Einstellungen der Persönlichkeit einen Namen für diese Persönlichkeit ein. Anhand dieses Namens erkennen Sie die Persönlichkeit beim Verfassen von Mails in der Liste „Von“. Der Name wird in der Mail nicht angezeigt.

  3. Geben Sie im Textfeld unter Von den Namen ein, der in Ihren ausgehenden Mails im Feld „Von“ angezeigt werden soll (z. B. Peter Meier). Dies ist der ausgeschriebene Name, der vor Ihrer Mail-Adresse angezeigt wird. Wählen Sie im zweiten Feld die Mail-Adresse aus, von der Sie die Mails senden möchten. Je nach Account-Konfiguration können Sie möglicherweise eine Adresse anlegen.

  4. Wenn Sie möchten, dass Antworten an einen anderen Namen und an eine andere Adresse und nicht an die im Feld „Von“ angezeigte Adresse gesendet werden sollen, aktivieren Sie Antwortadresse. Geben Sie den Namen und die Adresse ein, an die die Antworten für diese Identität gesendet werden sollen. Beispiel: Antworten auf Schulungsmails, die Peter Meier sendet, sollen an Mitarbeiterschulung unter schulung@beispiel.de gesendet werden.

  5. Für diese Persönlichkeit können Sie eine Signatur auswählen. Wenn Sie die entsprechende Signatur noch nicht angelegt haben, können Sie diese Persönlichkeit zu einem späteren Zeitpunkt weiter bearbeiten. Wenn Sie nicht möchten, dass die Signatur automatisch eingefügt wird, wählen Sie Signatur nicht anhängen.

  6. Wenn Sie möchten, dass Mails in bestimmten Ordnern automatisch diese Persönlichkeit nutzen, aktivieren Sie die Option Antworten oder Weiterleiten von Mails in folgenden Ordnern. Klicken Sie auf das Ordnersymbol, um einen oder mehrere Ordner in der Liste der vorhandenen Ordner auszuwählen oder um einen neuen Ordner anzulegen. Geben Sie ein Komma oder ein Semikolon ein, um die in das Feld eingegebenen Ordner voneinander zu trennen.

  7. Klicken Sie auf Speichern, um diese Persönlichkeit zu speichern.  

Wenn Sie die Signatur anlegen möchten, gehen Sie nach dem Speichern zur Registerkarte „Signatur“. Siehe Signaturen verwenden.
