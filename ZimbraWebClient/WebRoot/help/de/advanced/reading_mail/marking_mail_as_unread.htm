Mails als ungelesen markieren

Mails, die Sie geöffnet haben, werden als gelesen gekennzeichnet. Unterhaltungen, die ungelesene Mails enthalten, sind fettgedruckt.

Sie können in den Einstellungen festlegen, ob eine Mail beim Lesen im Lesefenster als gelesen gekennzeichnet werden soll. Sie können in Ihren Einstellungen festlegen, dass eine Mail, die Sie im Lesefenster gelesen haben, als gelesen gekennzeichnet werden soll, oder dass eine Mail nur dann als gelesen gekennzeichnet wird, wenn Sie sie im Inhaltsfenster (oder einem separaten Fenster) öffnen. Diese Optionen können unter Mail-Einstellungen festgelegt werden.

Sie können den Status einer Mail von "gelesen" in "ungelesen" ändern, indem Sie mit der rechten Maustaste auf die Mail klicken und im Kontextmenü Als ungelesen markieren auswählen. Sie können auch eine Unterhaltung als ungelesen markieren.

  • Wenn Sie eine Unterhaltung als ungelesen markieren, werden alle Mails in der Unterhaltung als ungelesen markiert.

  • Wenn Sie eine einzelne Mail als ungelesen markieren, wird die Unterhaltung, in der sich die Mail befindet, als ungelesen markiert.

Auf die gleiche Weise können Sie ungelesene Mails als gelesen markieren.
