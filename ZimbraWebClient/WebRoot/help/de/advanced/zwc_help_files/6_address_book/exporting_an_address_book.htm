Adressbuch exportieren

Sie können Ihr Adressbuch in eine durch Trennzeichen getrennte Datei (CSV) exportieren.

  1. Öffnen Sie die Seite Einstellungen > Importieren/Exportieren.

  2. Wählen Sie im Abschnitt Exportieren > Typ aus, welche Elemente Sie exportieren möchten.

    Option

    Beschreibung

    Accounts

    Sämtliche Account-Daten können in einer "mit Gzip komprimierten TAR-Datei" (.tgz) exportiert werden, die zurück in das System importiert werden kann.

    Bei Bedarf können Sie auch die erweiterten Einstellungen verwenden.

    Kalender

    Sie können Ihre Termine im iCalendar-Standardformat (ICS) exportieren und anschließend in ein anderes Kalenderprogramm importieren.

    Kontakte

    Sie können Ihre Kontakte in einem durch Trennzeichen getrennten Format (CSV) exportieren und anschließend in ein anderes Kontaktverwaltungsprogramm importieren.

  3. Wählen Sie den Quellordner der Kontakte aus, die Sie exportieren möchten.

  4. Klicken Sie auf OK.

  5. Klicken Sie auf Exportieren.
