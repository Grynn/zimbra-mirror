Neue Kontakte hinzufügen

Sie können neue Kontakte hinzufügen, Kontaktgruppen anlegen und Ihre Kontaktinformationen ändern.  

So fügen Sie einen neuen Kontakt hinzu

Wenn Sie einen neuen Kontakt hinzufügen, geben Sie die Informationen zu Name und Firma in getrennten Feldern ein. Sie können auswählen, wie Sie den Kontakt in Ihre Liste aufnehmen möchten. Sie können bis zu drei Mail-Adressen und drei Postadressen eingeben.

  1. Klicken Sie in der Symbolleiste des Adressbuchs auf Kontakt. Daraufhin wird das Formular "Neuer Kontakt" angezeigt.

  2. Geben Sie die Kontaktinformationen ein. Dazu gehören Vor- und Nachname, Mail-Adresse, Funktion, Geschäftsdaten und Notizen.

  3. Wählen Sie unter Speichern als aus, wie der Name in Ihrem Adressbuch angezeigt werden soll. Standardmäßig wird der Kontakt in der Form "Nachname, Vorname" angezeigt.

  4. Wählen Sie unter Adressbuch das Adressbuch aus, in dem der Kontakt gespeichert werden soll.

  5. Klicken Sie auf Speichern.

So legen Sie eine neue Kontaktgruppe an

Kontaktgruppen sind Verteilerlisten, die mehrere Mail-Adressen enthalten. Wenn Sie beim Verfassen einer Mail eine Kontaktgruppe auswählen, werden alle Kontakte, deren Adressen sich in der Liste der Gruppe befinden, automatisch ins Adressfeld der Mail eingefügt.

Hinweis: Von Hand eingegebene Mail-Adressen werden nicht überprüft.  

  1. Klicken Sie in der Symbolleiste des Adressbuchs auf Gruppe. Daraufhin wird das Formular "Neue Kontaktgruppe" angezeigt.

  1. Geben Sie im Feld Gruppenname einen Namen für diese Gruppenliste ein.  

  2. Wenn Sie die Gruppenliste in einem anderen Adressbuch als unter "Kontakte" anlegen möchten, wählen Sie im Feld "Adressbuch" das Adressbuch aus.

    Hinweis: Nachdem Sie eine Kontaktgruppe angelegt haben, können Sie sie in eine andere Liste verschieben.

  3. Geben Sie im Feld Gruppenmitglieder die Mail-Adressen ein. Geben Sie jeweils eine Adresse pro Zeile ein. Über das Feld Suchen auf der rechten Seite können Sie nach Mail-Adressen suchen. Geben Sie einen Namen ein und wählen Sie aus, ob Ihr Adressbuch oder das globale Adressbuch der Firma durchsucht werden soll. Wählen Sie die Kontakte aus, die Sie in die Gruppenliste aufnehmen möchten.

  1. Klicken Sie in der Symbolleiste auf Speichern. Die neue Kontaktgruppe wird zu Ihrem Adressbuch hinzugefügt.

Das Symbol zeigt, dass es sich bei dem Kontaktnamen um eine Kontaktgruppe handelt.
