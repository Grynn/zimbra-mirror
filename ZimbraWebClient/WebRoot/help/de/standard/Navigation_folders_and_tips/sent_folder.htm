Gesendete Mails speichern

Im Ordner Gesendet finden Sie Kopien der Mails, die Sie gesendet haben. Sie können Ihre dort abgelegten Kopien jederzeit löschen.

Standardmäßig ist die Option Im Ordner „Gesendet“ speichern aktiviert. Diese Einstellung können Sie auf der Registerkarte Einstellungen, Verfassen ändern.
