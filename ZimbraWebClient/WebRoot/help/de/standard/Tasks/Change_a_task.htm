Aufgaben ändern

So aktualisieren Sie eine Aufgabe

  1. Öffnen Sie die Aufgabenliste, die Ihre Aufgabe enthält.

  2. Klicken Sie im Fenster Inhalt auf die Aufgabe, die Sie ändern möchten. Daraufhin wird die Seite "Aufgabe" angezeigt.

  3. Ändern Sie die Daten zu dieser Aufgabe.

  4. Klicken Sie auf Speichern, wenn Sie alle Änderungen eingegeben haben.

So markieren Sie eine Aufgabe als erledigt

Es gibt zwei Möglichkeiten, eine Aufgabe als erledigt zu markieren:

  • Markieren Sie eine oder mehrere Aufgaben als erledigt. Klicken Sie mit der rechten Maustaste in die Aufgabenzeile und wählen Sie Als abgeschlossen markieren. Der Prozentsatz der Fertigstellung wird nun als 100% angegeben.

  • Markieren Sie eine oder mehrere Aufgaben als erledigt. Klicken Sie in der Symbolleiste auf Als abgeschlossen markieren.

Die Aufgabeninformationen bleiben in Ihrem Aufgabenordner.
