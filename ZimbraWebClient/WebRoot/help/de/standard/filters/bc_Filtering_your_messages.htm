Mails filtern

Beim Filtern werden eingehende Mails anhand von Vergleichsregeln geprüft und entsprechende Maßnahmen eingeleitet.

Sie können eingehende Mails filtern, um sie in bestimmte Ordner zu verteilen, um sie automatisch zu kennzeichnen, weiterzuleiten oder zu löschen. Sie können z. B. folgende Regel erstellen:

  • Alle Mails von Ihrem Vorgesetzten werden im Ordner "Vorgaben vom Chef" abgelegt.

  • Alle Mails der Mailing-Liste "Firmenveranstaltungen" werden mit dem Tag "Veranstaltungen"gekennzeichnet.

So erstellen oder ändern Sie eine Filterregel

  1. Klicken Sie auf die Registerkarte Einstellungen. 

  2. Öffnen Sie die Registerkarte Mailfilter und klicken Sie auf Neuer Filter.

  3. Geben Sie unter Filtername einen Namen für die Regel ein.

  4. Wählen Sie unter Falls eine/alle der folgenden Bedingungen erfüllt sind eine Bedingung aus. Erstellen Sie nun die Bedingung.

Sie können die folgenden Schritte wiederholen, um mehrere Bedingungen und Aktionen in einem Filter einzurichten.

  • Wählen Sie in der ersten Dropdown-Liste aus, welchen Teil der Mail die Filterbedingung betreffen soll.  

  • Wählen Sie aus, welcher Vergleich vorgenommen werden soll. Die gezeigten Optionen richten sich nach Ihrer Auswahl in der vorherigen Dropdown-Liste. Es wird beispielsweise "enthält" oder "enthält nicht" angezeigt.

  • Geben Sie im Textfeld ein Wort oder einen Satzteil für den Vergleich ein.

Klicken Sie auf hinzufügen, wenn Sie weitere Bedingungen definieren möchten. Sie können weitere Bedingungen hinzufügen oder mit dem nächsten Bereich fortfahren, in dem Sie eine oder mehrere Aktionen hinzufügen.

  1. Wählen Sie unter Folgende Aktionen ausführen in der Dropdown-Liste eine Aktion aus.

  • Geben Sie ggf. einen Ordner bzw. ein Tag an. Was Sie angeben, richtet sich nach der ausgewählten Aktion.

Klicken Sie auf hinzufügen, wenn Sie weitere Aktionen hinzufügen möchten. Sie können weitere Aktionen hinzufügen oder Ihre Eingaben mit OK beenden.

  1. Aktivieren Sie neben Filtername das Kontrollkästchen Aktiv, um die entsprechende Filterregel einzuschalten.

Der Filter wird automatisch auf alle neu empfangenen Mails angewendet, sobald diese eingehen.

Weitere Informationen zu den Bedingungen und Aktionen finden Sie unter Von Filtern unterstützte Bedingungen und Aktionen.
