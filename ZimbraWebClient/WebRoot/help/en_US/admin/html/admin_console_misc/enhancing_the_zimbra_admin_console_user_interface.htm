Enhancing the Zimbra Admin Console User Interface

You can create custom modules to add to the Zimbra administration console interface and then you can use the administration console to easily upload and install the modules you created.  

    • Click Extending Admin UI for documentation about how to create an extended administration console UI module.

    The Zimbra  extensions to the administration console UI are listed in the content pane as view only. Any extension that you add can be removed.

    How to deploy new administration console UI modules

    1. Save the module zip file to the computer you use to access the administration console.

    2. Log on to the administration console and in the Navigation pane's Configuration section, click Admin Extensions. The Admin Extensions content pane is displayed.

    3. Click Deploy. The Deploying a Zimlet or an extension dialog opens. Browse to the custom module zip file to be uploaded.

    4. Click Deploy.  The file is uploaded and the extension is immediately deployed on the server.
