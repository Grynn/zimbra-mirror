Assigning Accounts to Different Mail Servers

In an environment with multiple mail servers, the Class of Service can be used to assign the next account to a mailbox server or you can specify a specific server when you create the account.

In the COS, the Server Pool lists the available mailbox servers, and you select which servers should be available or the pool. When a COS is assigned to a new account, a mail server is randomly chosen for the new account from the server pool associated with that COS.

To add a server to the server pool

  1. Open a COS or create a new COS, and select the Server Pool tab.

  2. Click the server name from the left column.

  3. Click Add, to move the name to the right.

  4. Click Save, to commit your changes.

To add an account to a specific mail server

  1. In Accounts, select New Account.

  2. On the General Information dialog, in the Mail Server field uncheck auto.

  3. From the drop-down list, select the mail server for the new account.

  4. Continue to create the account and click Finish.
