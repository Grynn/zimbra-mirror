Customizing Zimbra Web Client UI Themes

The appearance of the Zimbra Web Client user interface can be customized globally and by domain. You have several options to change themes::

  • Change the look of the user interface from the administration console, selecting colors on the Global Settings or Domain Themes tab

  • Use the CLI to select exact company colors for the user interface.

  • Create a completely customized theme. For this option, go to the Zimbra Wiki, and view the Zimbra Rebranding and Themes page for complete documentation about how to customize a theme and deploy it to your Zimbra servers.
