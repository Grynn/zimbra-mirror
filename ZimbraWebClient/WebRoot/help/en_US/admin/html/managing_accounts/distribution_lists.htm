Creating Distribution Lists

A distribution list, also known as a mailing list, is a group of mail addresses contained in a list with a common mail address. When an email message is sent to a distribution list, the email is sent to everyone whose address is included in the distribution list. The address line displays the distribution list name and address. Individual recipient addresses cannot be viewed.

Only administrators can create distribution lists.

The maximum number of members in a distribution lists is 1000 recipients. The 1000 recipients includes addresses in distribution lists that are nested within a distribution list. The distribution list does not give an error if you have more than 1000 members but messages are not sent to more than 1000 recipients.

Users can share their contact lists, calendars, and notebooks with a distribution list. All members of the list have the same sharing privileges. As the administrator, you can manage these group lists so that new members to the list receive notification about the items that are shared with them. The Shares tab is used to published shared folders and enable automatic notification about new shared items.  

When the Can receive mail field is not checked, the distribution list cannot receive email. If you do not want to delete the distribution list, use this option to block email delivery to the list.

To create distribution lists

A distribution list can include email addresses listed in your company directory and you can add other addresses to the list, but email addresses you type are not verified.

  1. In the List Name field, type the distribution list name to be used as the mail address. The list name cannot have any spaces.

  2. In the Display name field, type the common name for the list.

  3. In the Description field, describe the type of list or usage.

  4. On the right, in the Add Members to this list area, either search for names from the company address book, or in the Or enter addresses below section, type a complete mail address.  The maximum number of members in a distribution list is 1000 recipients.

a. To search for names in the company address book, enter a partial name or a complete name in the Find field and click Search, or leave the Find field blank and click Search. Names display, ten at a time, in the box below.

If the Find field is blank when you click Search, the search results return the all mail addresses in the address book. Use Back and Forward at the bottom of this box to scroll through the address list.

b. To use the Or enter addresses below section, type a complete mail address. Either separate addresses with a comma or press  Enter, to add the next address on a new line.

  1. If you do not want the distribution list to be viewed in the GAL, check Hide in GAL.

  2. Click the Aliases tab to add a distribution list alias name (optional).

  3. Click Save when all names have been added to the list. The distribution list is enabled and ready to use. If you did not check Hide in GAL, the distribution list is listed in the GAL.
