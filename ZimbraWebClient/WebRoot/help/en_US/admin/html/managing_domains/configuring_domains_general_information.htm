Editing Domains General Information

  1. Go to Configure > Domains and select the domain to edit.

  2. In the General Information page, General Information section, edit any of the following fields as required:

Option

Description

Domain name

Full name of the domain.

Public service host name

Host name of the REST URL. This is the URL that is used when sharing email folders, Briefcase folders, sharing Task lists, Address Books, and Calendars. See Setting up a Public Service Host Name in the Managing ZCS Configurations chapter in the ZCS Administrator's Guide.

Public Service protocol

 Protocol to be used in the public API (REST or SOAP proxy)

Public service port

Port to be used in public API.

Inbound SMTP host name

If your MX records point to a spam-relay or any other external non-Zimbra server, enter the name of the server here

Description

Text field that displays in the Domain list in the Content pane.

Maximum accounts for this domain

Maximum number of accounts that can be created on this domain.  The number entered here is displayed in the Account Limits tab.  The default is no limit (blank field).

Default Class of Service

Default COS for the domain. If you do not enter a COS, the default COS is used. The COS entered here is  automatically assigned to accounts created on the domain if no other COS is selected.

Domain status

The domain status is active in the normal state. You may want to change this status when you setting up the domain if you are not ready to use the domain.

Admin help URL

If you are created a customized administrator Help, enter the URL. This replace the Help link in the administrator console.

Delegated admin help URL

If you created a customized delegated administrator help, enter the URL.

 

  1. In the Voice and Chat section, if you set up the voice feature select the unified communication type.

  2. Click Save.
