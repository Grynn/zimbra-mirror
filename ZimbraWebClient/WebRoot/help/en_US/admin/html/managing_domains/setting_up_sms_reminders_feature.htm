Setting Up SMS Reminders Feature

The ZWC  Preferences>Notification page lets users configure a SMS alert to their mobile device to receive a reminder message for a task or for a meeting on their calendar. You can enable the SMS notification feature.

Users select a region and a carrier when setting up their SMS alert. The list of SMS/email gateways is in ZmSMS.properties. You can customize this list to add SMS/email gateways that are not listed.

The SMS Reminders feature can be enabled on the domain, COS, or for individual accounts. The feature cannot be enabled on both a domain and a COS.

Enable from the Domain

  1.  Go to Configure > Domains, and select the domain to edit.

  2. Open the Features page and check SMS reminders.

  3. Click Save.

Enable from the COS

  1.  Go to Configure > Class of Service, and select the COS to edit.

  2. Open the Features page and in the Calendar Features section, check SMS reminders.

  3. Click Save.

Enable from an account

  1.  Go to Manage > Accounts, and select the account to edit.

  2. Open the Features page and in the Calendar Features section, check SMS reminders.

  3. Click Save.
