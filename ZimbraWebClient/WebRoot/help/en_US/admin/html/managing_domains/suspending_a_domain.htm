Suspending a Domain

The domain status can be changed from active to suspended to prevent access to the domain. When the domain status is set to Suspended, users cannot log in, user email is queued at the MTA, and accounts and distribution lists cannot be created, deleted, or modified.

To suspend a domain

  1. In the Domains list select the domain to be suspended.

  2. Click Edit in the Content pane toolbar.

  3. From the General tab Domain Status, select Suspended.

  4. Click Save.
