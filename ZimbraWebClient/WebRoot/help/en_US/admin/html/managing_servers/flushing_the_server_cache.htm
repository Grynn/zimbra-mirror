Flushing the Server Cache

You can flush the server cache for the following types of entries.

  • Zimlets

  • Themes

  • Locale

 

When you add or change skin (themes) properties files and local resource files or update a customized Zimlet, you must flush the cache to reload the new content. Until you do this, the new themes, locales, or Zimlets are not available on the server.

Select the server and click Flush Cache on the toolbar.

    • Flushing the cache may cause a temporary slowdown of mail delivery.
