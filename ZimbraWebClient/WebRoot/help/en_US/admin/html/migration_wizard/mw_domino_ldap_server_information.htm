Domino LDAP server information

To log into the LDAP server enter the following information:

  • LDAP Server. The external or internal LDAP server name.

  • LDAP Port. Enter the port number.  The default is 389.

  • User DN and Password. Enter the DN (Distinguished Name). If the LDAP is external, the DN is the LDAP user name and password. If the LDAP is on the Domino server, enter your user name and password.
