Creating Advanced Server Statistics

In Monitoring>Sever Statistics the Advanced Statistics tab includes advanced graphing options that lets you generate various charts based on statistical information for the CPU, IO, mailboxd, MTA queue, MySQL and other components. Logs are processed into a format that is readily displayed in the admin console, indicating real-time service status, MTA/spam/virus traffic and performance statistics.

You can specify charts by server for the following groups. Each group has a list of specific counters. You can select what information you want graphed.

  • cpu.csv: CPU utilization is tracked both at the server level and the process level.
  • convertd.csv. Collects CPU statistics for the convertd process for Network Edition only.
  • df.csv. Disk utilization is tracked for each disk partition
  • fd.csv: Captures file descriptor usage on the system
  • mtaqueue.csv: ZCS server and JVM statistics.  
  • io-x.csv and io.csv: Store data from the iostat(1) command (io-x.csv with iostat -x)
  • vm.csv: Linux VM statistics (from the vmstat command). The output of vmstat is recorded for reviewable statistics.

 

The .csv files zmmtzstats.csv and zmstatuslog.csv are used to generate the Server Status page and MTA stats tab.

How to Add a Chart

To generate images, Adobe Flash Player must be installed

When you open the Advanced Statistics tab, the Add Chart fields are displayed. You select a combination of server, group and counters to create the chart and you set a start and end time for information you want charted.

  1. Select the Server and the statistics Group you want to graph.

  2. In the Counters field select the type of information you want to graph. If you select more than one counter type, the counter details overlay on the graph.

  3. Enter the start and end time to display the information. You can enter either as date (mm/dd/yy) or as Unix AT command, for example, Start  now-1d -  End now would generate a graphic for information 1 day previous to now to now.

The chart is generated with the information you requested. Click Update Chart to refresh the information. These charts are not saved, but you can rerun by updating the start and end times at any time.

If you want to look at more than one type of chart, click Add Chart and a new set of chart fields are displayed.
