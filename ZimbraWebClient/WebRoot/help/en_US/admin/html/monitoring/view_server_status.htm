View Server Status

The Server Status content pane lists all installed servers and services, including the services for MTA, mailbox, logger, SNMP, anti-spam, anti-virus, and LDAP. The status,  for on or for off, is displayed, along with the time the status last changed.  

In order to see the status data, the Zimbra logger package must have been installed on one of the Zimbra mailbox servers.

Each server can have one or more services running on it. The services that are monitored include:

  • LDAP

  • Mailbox

  • MTA

  • SNMP

  • Anti-virus

  • Anti-spam

  • Logger

  • Spell

 

    • You can turn services on or off from the Servers>Services tab.
