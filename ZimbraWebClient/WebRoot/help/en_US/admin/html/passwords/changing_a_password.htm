Changing Passwords from the Admin Console

If you use internal authentication, you can change an account's password quickly from the Accounts toolbar.  

  1. Select the account and click Change Password.
  2. Enter the new password.
  3. Reenter the password.
  4. The user must be given the new password.

 

    • Passwords cannot contain accented characters, such as these examples:   ñ, é, ó, etc.

    Must Change password can be enabled for the account.  The user must change the password the next time he logs on.
