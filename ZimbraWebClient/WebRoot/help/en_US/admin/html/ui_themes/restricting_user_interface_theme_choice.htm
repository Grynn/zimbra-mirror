Restricting User Interface Theme Choice

The ZWC user interface theme that displays as the default on the client and the themes that users can select can be configured in Classes of Service or in individual accounts. The following theme usage options can be configured:

  • Limit users to one theme.

  • Let users access any of the installed themes.

  • Select specific themes that users can use.

See Making UI Themes available by COS and Making UI Themes Available by Account
