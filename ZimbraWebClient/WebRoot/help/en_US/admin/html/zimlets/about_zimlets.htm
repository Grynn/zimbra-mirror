Managing Zimlets

Zimlets™ is a mechanism for integrating the Zimbra Collaboration Server with third party information systems and content. Zimlets are also used to intermix user interface within the Zimbra Web Client for Email, Calendar, Address Book, and Documents applications. You can create other Zimlets so that users can interact with your company resources or other defined applications from the Zimbra Web Client.

  • For information about how to develop and configure Zimlets see the Zimlet Development on the Zimbra wiki.  Go to the Zimbra gallery, http:// gallery.zimbra.com, to see the type of Zimlets that can be used within the ZCS.  

From the administration console, you can add new Zimlets, set access privileges by COS and by individual accounts, and disable and uninstall Zimlets from ZCS.  Users can disable and enable Zimlets that are not mandatory from their Preferences>Zimlets page.

If you create a Zimlet that needs additional configuration after deployment, you will need to use the ZCS CLI to access and change the Zimlet configuration. See the Working with Zimlets chapter in the Administrator's Guide for details.

Zimlets that are deployed are listed on the Zimlets contents pane. You can remove Zimlets, disable and enable them from the Zimlets content pane.

Default Zimlets Enabled on ZCS

ZCS includes pre-configured Zimlets that enhance the user experience while working in the Zimbra Web Client. Some of Zimlets are already deployed and made available from the COS.  You can manage which Zimlets are available to users from the COS Zimlets tab. The following is a list of default Zimlets.:

  • Attach Contacts. Allows attaching contacts when composing a new message.

  • Email Attacher. Attach email messages when composing a new message.

  • Date. Highlights dates, previews associated appointments and creates a link to the calendar.

  • Drag-n-Drop Attachments. Provides ability to drag-n-drop file attachments when composing an email message.

  • Email Contact. Details Highlights and previews associated contact details for an email address.

  • LinkedIn. Hooks on to email Zimlet; shows LinkedIn search result for a given email.

  • Social. Access social services like Twitter, Facebook, Digg and TweetMeme from Zimbra.

  • Search Highlighter. After a mail search, this Zimlet highlights Search terms with yellow color.

  • URL Links. Highlights Web URLs for linking in email messages.

  • WebEx. Easily schedule, start or join WebEx meetings.
