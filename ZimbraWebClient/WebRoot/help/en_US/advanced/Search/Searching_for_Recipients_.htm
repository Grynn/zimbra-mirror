Searching for messages sent to or from specific recipients

You can search for messages from or to a recipient of a message in one of the following ways:

  • From the Sent or Drafts folders To column

  • Using Advanced Search, Basic Search pane To/Cc field

When you search within a folder, only results for messages within that folder are displayed.  When you use the Advanced Search option, you can select which folders to search. Results include messages from the folders selected for the search.

Search Sent and Drafts folders

Note: Select a message, not a conversation to perform this search.

  1. In the Sent and Drafts folders, select a message with the name of the recipient you want to find.

  2. Right-click the message and select Find Emails.

  3. Select either Received From Recipient or Sent To Recipient. The search is executed and messages that meet your request display in the Content pane.

To find messages you sent to more than one email recipient

To find all messages you sent to one or more recipients, you can click the To heading in the Sent>Content pane and enter multiple addresses in the dialog. By default, every name in the Addresses field. You can either search for all the names in the Addresses field, or you can uncheck Include messages sent to all addresses to perform an OR search, finding all messages sent to any of the recipients.

  1. In the Sent and Drafts folders, click To in the Content pane heading. A Find Emails To Recipients dialog displays.

  2. Enter recipient addresses, separated by a semicolon .

  3. Click Find. The search is executed and messages sent to listed recipients display in the Content pane.

Uncheck Include messages sent to all addresses to perform an OR search, finding messages from either one address or the other.

Using Advanced Search

  1. Click Advanced to open the Advanced search area.

  2. In the  Basic Search pane To/Cc field  or From field enter the name to search.  The more specific the name the better the results. By default the Inbox is searched.

  3. To search other folders, click Folder and select the folders to search.

As soon as you enter a criteria in a search pane, or as soon as you have made any selection change in any other search pane, the search executes.

Items matching the search criteria are listed in the lower right. If the Conversation feature is enabled, conversations containing messages that match the search criteria are listed.
