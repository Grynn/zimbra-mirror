Create tasks lists

You can create different task lists and configure the lists to sync with an  iCal's To do list.

To create a task list:

  1. Select the Tasks tab to open the Task application.

  2. In the Overview pane, right-click the Tasks header and select New Task List.

  3. In the Create New Task List dialog, type the task list name and select the color to display the task.

  4. Check Synchronize tasks from the remote service to sync the new list to an iCal To Do list. Enter the iCal URL.

  5. Click OK. The list is added to Tasks in the Overview pane.
