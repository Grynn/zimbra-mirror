Importing  Items into your account

The Preferences Import/Export folder can be used to import your complete account data, specific folders, calendars, and contact lists.

Importing Contacts

You can import contact lists  that are saved in a comma-separated (.csv) text file format. Import is available from your Preferences>Import/Export folder.

  1. From the Preferences tab, click the Import/Export folder.

  2. In the Import section,  select the type as Contacts.

  3. In the drop-down next to Contacts, select the type of contact file that is being imported:

    • Zimbra contacts

    • Yahoo! contacts

    • GMail contacts

    • Outlook 2000 contacts

    • Outlook 2003 contacts

    • Outlook contacts

    • Thunderbird contacts

 Contact lists from different accounts may not display contact information in the same order. Picking the correct type of contact file that is being imported  identifies the order of the fields so that the information displays correctly in your Address Book.

  1. Browse to locate the contact .csv file to import.

  2. Click Import. In the Choose Address Book dialog either select an existing address book to add the contacts or create a new address book to import the file.

  3. Click OK. The content of the contacts file is imported.

Importing your iCal calendar

You can import iCalendars (iCal) calendars that are saved in the .ics format. iCalendars are the standard for calendar data exchange over the Internet, and .ics is the standard format for iCalendar information. Import is available from your Preferences > Import/Export tab.

You can import your iCal calendar to an existing calendar or create a new calendar and import the .ics file to it.

To import an iCal calendar:

  1. From the Preferences tab, click the Import/Export tab to open it.

  2. In the Import section's Type field,  select Calendar.

  3. In the File field, browse to locate the .ics file to import.

  4. Click Import. In the Choose folder dialog, select the folder to import the calender, either select one of your existing calendars or create a new calendar.

  5. Click OK. The calendar is imported into our account.

iCal calendars are updated periodically according to the remote calendar polling intervals set by the system administrator.
