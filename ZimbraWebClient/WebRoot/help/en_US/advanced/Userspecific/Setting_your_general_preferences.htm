Setting your general preferences

Your default user preferences are configured when your account is created. These preferences define how your @ClientNameShort@ features work. The Preferences>General page includes the following settings that you can change.

  • You are logged into the Advanced web client, but you can define your default client type here.

    • Advanced offers the full set of Web collaboration features. This Web Client works best with newer browsers and faster internet connections.

    • Standard is recommended when internet connections are slow, when using older browsers, or for easier accessibility.  

  • Theme is the background colors for your mailbox interface. In the General page, Themes displays a list of different background color you can select. Carbon is the default.

  • You can change the Language that displays in Zimbra Web Client to one you are more comfortable reading.

  • Default Timezone determines the date and time for messages that you send or receive and the time to display for Calendar appointments.  This can be different from your computer time zone configuration. If the time zone displayed is not correct, change it here.  

  • If Change password is displayed, you can change your password. If you have trouble changing your password, contact you system administrator for the password rules. Contact your system administrator if you do not have this option and want to change your password.

  • In Searches, you can choose to automatically include the Spam and Trash folders in any search you perform. By default these folders are not searched.

  • In Search Language, check Show advanced search language in search toolbar  to show search string details in the search text box on the client page.  

  • Other

    • To see more than 50 lines when scrolling the Content page, modify Number of items to fetch when scrolling.

    • Settings.  Enable Display check boxes to quickly select items in lists (requires refresh) to display a checkbox for each item in the Content pane. When this is enabled, use the checkbox to select one or more items to perform the same action on, such as delete, move, mark as read/unread.

    • Print Font Size. The default font size used when a message is printed is 12 points. You can select another print font size from the list.

    • Email Addresses. When this is enabled, only the address name is displayed, not the complete email address. You can click on the name to see the complete address.  

    • Bubbles. The Bubbles option is on by default. When this is checked, names in the address field display in a "bubble" rather than as text.  The bubbles are more interactive than email addresses displayed as text.

  • System Defaults check Use Zimbra Desktop as default mail application to always use @DesktopName@ when you send email messages from a non-email program.
