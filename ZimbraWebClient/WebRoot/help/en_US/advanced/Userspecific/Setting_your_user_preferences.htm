Setting your user preferences

Your default user preferences are configured when your account is created. These preferences define how your @ClientNameLong@ features work.

You can change these settings from the Preferences tab. The preference pages display in the Overview pane.  If an option described here is not visible in your view, it is not available for your account.

You can set preferences in the following folders. Click the folder name for a description of the content.

  • General

    In the General page you can

    • Configure the default version of the ZWC to use, advanced or standard, select your mailbox theme, language and default time zone

    • Define Search settings to include searching Junk and Trash Folders in your searches

    • Displaying check boxes in the Content pane

    See Setting your general preferences

  • Mail

    In the Mail page you can:

    • Configure how often to check for new mail. The default is 5 minutes

    • Select the behavior of the Get Mail button to either run your default search or to update your current view

    • Select how messages are displayed, as HTML or as plain text

    • Set up how your mailbox handles messages that are received, including message preview options, reading pane and move options

    • Set up message alerts

    • Create an auto-reply message that can be used when you are away

    • Turn on Read Receipt reply

    • If you are set up to use a POP3 client, you can configure the POP access behavior when messages are downloaded to @ClientNameShort@

    • Set Spam Mail Options to block messages to block and to allow messages from addresses you specify

    See Setting your mail preferences

  • Composing

    In the Compose page you can

    • Select the text editor you want as the default, either HTML or plain text

    • Select whether to include the original text in the body of your reply message

    • Make spell check mandatory

    • Specify how to mark forwarded or replied to messages

    • By default, the Save copies of messages to Sent folder is enabled. You can change this.

    See Setting your preferences for the email compose window

  • Signatures

    You can create signatures for your email messages.  If you create multiple identities, you can create different signatures and assign them to specific addresses.

  • Address Book

    In the Address Book page you can

    • Enable the feature to automatically add addresses to your Emailed Contacts address book when you send mail

    • Enable to use the Global Address List when viewing contacts in the Contact Picker

    • Include the Global Address List when autocompleting addresses

    • Include addresses in shared address books when auto completing addresses 

    See Setting your address book preferences

  • Accounts

    The accounts page is used to create and manage your personas. Persona is the mail identity used in the From field of email messages you send. You can set up your  and POP accounts

  • Mail Filters

    Use the Mail Filters folder to define mail filtering rules. To learn how to set up mail filter rules, see How filtering works.

  • Calendar

    In the Calendar page you can

    • Define your default view of the calendars

    • Set parameters to use when creating an appointment

    • Select to show the  mini-calendar

    • Enter an address to forward your invites from a shared Calendar

    • Set the reminder notice for meetings

    • Set permissions for free/busy views and who can invite you to meetings

    See Setting your Calendar preferences

  • Sharing

    The Sharing page shows items that are shared with you and items that are shared by you. See Managing items you share

  • Notifications

    You can configure an email address or SMS alert to your mobile device to receive a reminder message for a task or a meeting on your calendar.

  • Mobile Devices

    Mobile devices you configured to sync with ZWC are listed here.  You can also wipe your device from this page. See Managing your mobile devices

  • Import / Export

    The Import/Export page can be used to import or export your complete account data, specific folders, calendars, and contact lists. This is useful if you want to backup your account or a specific folder locally.

  • Shortcuts

    In the Shortcuts page, you can view the list of keyboard shortcuts and print the list.

  • Zimlets

    You can manage which Zimlets you want to use from the Zimlets folder. Zimlets are add-on applications that enhance the functionality of your client.  The system administrator enables the Zimlets. Zimlets that you can use are displayed in the Overview pane.
