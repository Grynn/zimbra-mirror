Saving the Content of Your  Account

You can export all of your account data, including email messages and attachments, contacts, calendar, tasks, Documents notebooks and Briefcase folders, or you can export specific items in your account.  You can then save the data to your computer or other location as a backup. The account data is saved as a tar-gzipped ( tgz) archive file so that it can be imported to restore your account.

 Note: When you export individual contact folders, they are saved as .csv files; individual calendar folders are saved as .ics files.

The data are copied, not removed from your account.

You export the file from Preferences> Import/Export.

To export data

  1. From the Preferences tab , click the Import/Export folder.

  2. Either export all account data or selected a specific type of folder to export.

    • To export all data for the account, select Account. Note: Trash and Junk folders are not exported.

    • To export a contact list or a specific calendar, select that option. When you select a calendar or a contact to export, select the format type to be exported as. See Exporting Address Books and Exporting your iCal calendar.

    • If you want to select several specific types of items to export at one time, check Advanced Settings to expand the view. Check the boxes for the data items to export. All items of the chosen type within all folders under the chosen folder are included. If you want specific data, you can use the search filter field to specify the specific type of data to export. See Query language description and Search examples.  

  1. Click Export and select where to save the file.

  2. Click OK.

Note: The exported account data file can be viewed with an archive program such as WinRAR archiver.

To import saved account data

You restore an account from the file that you exported.  

  1. From the Preferences tab, click the Import/Export folder.

  2. In the Import section, browse to the file that you want to import.  If you import Calendars or Contacts, specify the format that is being imported. See Importing Contacts and Importing your iCal calendar.  When  Account is selected, only the tgz file that you have already exported can be imported to your account.  

Select specific types of items from the saved tgz file to import in the Data Types section. When you click Import only those item types are imported.

  1. Click Import.
