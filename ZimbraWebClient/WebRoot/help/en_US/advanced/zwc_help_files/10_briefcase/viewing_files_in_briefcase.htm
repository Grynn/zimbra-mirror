Viewing Files in Briefcase

  1. Go to the Briefcase page.

  2. Click View on the Task toolbar to select how to display the file in the Preview Pane.

 

Option

Description

Preview Pane At The Bot­tom

The Briefcase preview pane displays at the bottom of the pane.

Preview Pane On the Right

The Briefcase preview pane displays on the right.

Preview Pane Off

The Preview Pane does not display
