Set Preferences for Email Replies

To reply to the message in the same format it is received, select Reply/Forward using format of the original message. This ensures that the recipient gets the message in the most readable format.

  1. Go to the Preferences>Mail page, Composing Messages section.

  2. To reply to the message in the same format it is received, select Reply/Forward using format of the original message.

  3. Select whether to include the original text in your reply message. If you are including the original message:

    4. •   Select whether to mark the previous responses with a character or include the original details in the header (The To, From, and Subject fields). You can also include the original message as an attachment.

    •    If you want to prefix the previous responses, select either the angle bracket (>) or pipe (|).

  4. Click Save.
