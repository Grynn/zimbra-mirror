Creating a Filter from an Email Message

You can create a filter based on an email message in your Inbox. The filter information is extracted from the message, and you can then modify the conditions and actions.

  1. Right-click an email message and choose New Filter.

    2. The Add Filter dialog box appears populated with the information from the email message.

  2. In the Filter Name text box, type a unique name for the filter.

  3. Change the conditions and actions as necessary using the drop-down menus.

  4. Click OK.
