Viewing Filters

Viewing Existing Filters

You can view the filters that you have defined and their status.

  1. Go to the Preferences>Filters page.

  2.    To view information about a particular filter, double-click it.

Viewing Filter Order

Active filters are applied in the order that they are listed. You can change the order.

  1. Go to the Preferences>Filters page.

  2. Use the Move Up and Move Down buttons to change the order.

Note:   An email message is evaluated by each active filter unless you specify that a message is treated only by the first filter that it matches. Usually, you do not want other filters applied to a message once there is a match because the message can undergo multiple, perhaps contradictory, actions.
