Adding Members to a Contact Group List

  1. Go to the Address Book page and select the contact group to add members.

  2. Right-click the contact group and select Edit Group.

  3. To find a new group member, type a name into the Find text box.

  4. In the in drop-down menu, select the address book you want to search.

    5. Names that match your entry are listed in the box below.

  5. Select the names to enter and click Add or Add All. The names are added to the Members list.

    6. If the address you are looking for is not in the list, you can type the email address in the Or enter addresses below text box. Separate addresses with either a comma (,) or semicolon (;), or pressing Return.

  6. Click Save.
