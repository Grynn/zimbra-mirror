Printing a Task or Task List

You can print all the tasks in your list or you can select specific tasks to print.

  1. Go to the Tasks page.

  2. In the Overview pane, select the task or task list that you want to print. To select multiple tasks, use the check boxes next to the task.

  3. In the Content pane, select the task list or tasks to print.

  4. On the Task toolbar, click Print and select whether to Print selected task(s) or to Print Task Folder.

  5. In the Print dialog, click OK.
