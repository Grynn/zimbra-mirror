Forwarding a mail message

You can use Forward to send a mail message on to someone else. Forwarding a message includes all file attachments, whereas using Reply preserves the body text and does not include the attachments.

How your message is forwarded can be set up in your Preferences>Mail>Compose page. You can select whether to include the original text of the message and message header  To/From and Subject header details in your forwarded message, or you can select to include the original message as an attachment to your email.

To forward a message:

    1. Open the message to be forwarded.

    2. On the bar above the message field, click Forward. .  A compose window opens. The text of the forwarded message appears in the body.   

    3. Enter addressees in the To:, CC:, or BCC: fields.

    4. Optionally, add a few lines of new text at the top such so that the recipient knows why you decided to forward it.

    5. Click Send.
