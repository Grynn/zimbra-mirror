Link to Shared Address Books

If you receive an email notice that you have been granted access to share another person's address book, you will need to link to the that address book. You will need to know the owner's full email address and the exact name of the address book that is being shared.

To link to a shared address book:

  1. In the Address Book header in the Overview pane, click edit.

  2. In the toolbar, select Link to Address Book.

  3. Enter a name for the shared address book. This is the name that displays in your address book list.

  4. Enter the owner's email address and the owner's calendar name.

  5. Select the color for this shared address book.

  6. Click Create Address Book.
