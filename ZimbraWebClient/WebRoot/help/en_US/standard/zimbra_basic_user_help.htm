Getting Started with @CollaborationProductShortName@ Standard Edition

The standard Web client offers you an easy to use email account with the following features.

Note: Your account may not include all the features mentioned. Check with your system administrator to see which features are enabled for your account.

See Navigating around your mailbox.

Email

    • Compose and send new mail messages

    • Include attachments with your message

    • Add signatures to your messages

    • Track mail exchanges using the Conversation feature

    • Create filters to route incoming mail to designated folders

Address Book

    • Create and manage multiple address books

    • Import and export contact lists

Calendar

    • Create and manage multiple calendars

    • Create appointments, meetings, and events

    • Import and export other calendar programs

Tasks

    • Create multiple Tasks lists

    • Manage a task, set the priority and track the progress

 

In addition, you can manage how mailbox works and configure account settings from the Preferences tab, and use the Find field to search for items.
