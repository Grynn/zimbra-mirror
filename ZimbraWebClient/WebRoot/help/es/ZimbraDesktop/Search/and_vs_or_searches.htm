Búsquedas con AND frente a búsquedas con OR

Para las búsquedas en las que quieres incluir varios criterios, puedes usar estas variables para encontrar elementos que coinciden con un solo criterio o que coinciden con todos. Puedes realizar ambos tipos de búsqueda con la búsqueda avanzada.

Las reglas para todos los paneles de búsqueda son:

  • El panel Búsqueda básica permite crear una consulta de búsqueda con la variable AND. Si abres dos paneles de Búsqueda básica, crearás una consulta de búsqueda con la variable OR entre ambos.

  • Para buscar mensajes que coinciden con cualquiera de los criterios especificados, utiliza OR (o) y, si el mensaje contiene el criterio X o Y, se considerará una coincidencia. Para los paneles de la búsqueda avanzada con múltiples casillas de verificación, al seleccionar varias opciones dentro de un solo panel, se utilizará la variable OR para esas opciones.

  • Para buscar mensajes que contienen los criterios X e Y, utiliza AND (y) para indicar que el mensaje debe cumplir todos los criterios especificados para que se considere una coincidencia. En los paneles de Búsqueda avanzada Etiquetas y Estado, si abres varias instancias del mismo panel y activas diferentes casillas de verificación en cada panel, se realizará una búsqueda con la variable AND.

Nota: Sólo verás "OR" en la búsqueda. Si seleccionaste como opción mostrar la consulta de búsqueda en la barra de búsqueda, se actualizará el cuadro de texto de búsqueda para mostrar la consulta resultante. Con la variable AND, la palabra AND no aparecerá.
