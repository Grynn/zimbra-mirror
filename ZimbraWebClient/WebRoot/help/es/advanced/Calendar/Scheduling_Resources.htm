Programar recursos

Al programar tus reuniones, puedes reservar una ubicación o equipamiento para la reunión.  Cuando quieras reservar estos recursos, se solicitan para una reunión. Si está disponible el recurso, se reservará para la reunión.  

Recibirás un mensaje de correo electrónico de aceptación o rechazo de la reunión. Se rechazará si ese recurso ya está asignado para otra reunión a esa hora o para reuniones recurrentes si la nueva reunión coincide con la fecha u hora de cualquiera de las reuniones de la serie.  Recibirás un mensaje indicando las fechas donde ha surgido el conflicto.

El administrador configura las cuentas de estos recursos con direcciones de corre electrónico y configura su disponibilidad. No podrás reservar algunos recursos para más de una reunión por invitación.
