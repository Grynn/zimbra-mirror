Aceptar acceder a un Bloc de notas

Si recibes una notificación por correo electrónico de que se te ha otorgado acceso para ver el bloc de notas compartido de otro, puedes aceptar o rechazar la oferta.

Si aceptas, se visualizará el bloc de notas compartido en tu lista de blocs de notas.

Para aceptar

  1. Haz clic en Aceptar compartir en el mensaje de correo electrónico. Se abrirá el cuadro de diálogo Aceptar compartir que describe el rol que se te ha asignado.

  2. Antes de aceptar, puedes personalizar el nombre del bloc de notas.

  3. Activa la casilla para Enviar mensaje estándar para enviar una confirmación al dueño. (opcional)

  4. Pulsa en Sí. Se añadirá el bloc de notas a tu lista de blocs de notas. Se moverá el mensaje a la carpeta Papelera.
