Crear un nuevo documento

Puedes crear un documento en el Maletín y diseñar la distribución de contenido mediante las herramientas de formato de texto enriquecido. Puedes comprobar la ortografía de la página y utilizar palabras claves para crear fácilmente enlaces a otras páginas.

  1. Haz clic en la carpeta del maletín en la que quieres trabajar. En el panel de Lista de documentos se muestran los archivos que ya se encuentran en el maletín

  2. En la lista Nuevo en la barra de herramientas, selecciona Documento. Se abrirá una página de Documentos de Zimbra en blanco.

  3. En la barra de herramientas, introduce el nombre del documento.

  4. Redacta el contenido de la página. Puedes usar el editor de texto enriquecido predeterminado para añadir estilos, color y tablas a tu página.

  1. Haz clic en Guardar para guardar el archivo y continuar trabajando. Para cerrar el documento sin guardarlo, haz clic en la X de la pestaña del navegador. Haz clic en Guardar y cerrar para guardar el archivo y vuelve a la página Lista de documentos.

Cuando guardes un documento, la versión de este cambiará.

Nota: Si has creado el documento en un maletín equivocado, podrás mover el archivo al documento correcto seleccionando el documento y arrastrándolo hasta la carpeta correcta.
