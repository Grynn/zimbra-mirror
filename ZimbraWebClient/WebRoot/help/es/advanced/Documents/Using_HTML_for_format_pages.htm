Utilizar código HTML para dar formato a las páginas de blocs de notas

Si estás familiarizado con el código HTML, puedes añadir etiquetas HTML para crear el contenido.  Se permiten todos los elementos básicos de HTML.  Para crear páginas con HTML, pulsa en Formato en la barra de herramientas y selecciona Fuente de HTML.

Al modificar la página, puedes cambiar entre Texto enriquecido y HTML. En la esquina derecha superior, pulsa en Formato para indicar qué editor quieres usar.
