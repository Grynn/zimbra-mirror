Crear listas de tareas

Puedes crear listas de tareas diferentes y configurar las listas para sincronizar listas de tareas desde servicios remotos como la lista de tareas pendientes de iCal.

Para crear una lista de tareas:

  1. Haz clic en la pestaña Tareas para abrir la aplicación de tareas.

  2. En el panel de resumen, haz clic con el botón derecho del ratón en el encabezado de tareas y selecciona Nueva lista de tareas.

  3. En el cuadro Crear nueva lista de tareas, introduce el nombre de la lista de tareas y selecciona el color que quieres asociar a ella.

  4. Marca la opción Sincronizar tareas desde el servicio remoto para sincronizar la nueva lista con una lista de tareas pendientes de iCal. Introduce la URL de iCal.

  5. Haz clic en Aceptar. Se añadirá la lista a la sección Tareas en el panel de resumen.
