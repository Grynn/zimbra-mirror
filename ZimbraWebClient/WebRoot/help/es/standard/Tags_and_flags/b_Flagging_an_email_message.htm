Marcar un mensaje de correo

La banderita al lado de los mensajes es un indicador para saber si se ha marcado o no ese mensaje. Puedes usar esta herramienta para indicar que se trata de un elemento importante o para distinguir un mensaje o conversación de otros mensajes y conversaciones menos importantes.

Para añadir una marca de seguimiento, una bandera, selecciona el mensaje o conversación y, en la lista de Más acciones, selecciona Añadir marca.

Para desactivar la marca de seguimiento, selecciona el mensaje o conversación y, en la lista de Más acciones, selecciona Quitar marca.
