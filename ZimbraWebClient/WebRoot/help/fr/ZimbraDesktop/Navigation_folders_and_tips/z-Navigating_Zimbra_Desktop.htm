Navigation Zimbra Desktop

Lorsque vous lancez Zimbra Desktop, vous verrez un affichage plein écran similaire à celui-ci.

La page principale de Zimbra Desktop inclut les zones suivantes :

  1. Ligne Barre de recherche. Cette zone affiche la Rechercher et Recherche avancée. Les liens Configuration et Aide de compte sont à droite.

  2. Onglet d'applications. Les applications accessibles sont répertoriées dans les onglets situés dans la partie supérieure.

  3. Barre d'outils Affiche les actions disponibles pour l'application que vous utilisez actuellement. Dans cet exemple, la barre d'outils de l'applicationMail s'affiche.

  4. Panneau de lectureLe contenu de cette zone change selon l'application utilisée. Dans l'affichage Boîte de réception, le panneau affiche tous les mails ou conversations dans votre Boîte de réception. Dans l'écran sur cette page, le Panneau de lecture est la zone en dessous des messages dans votre Boîte de réception. Lorsque vous cliquez sur un mail, celui-ci s'affiche dans un panneau de lecture au-dessous de la liste des mails.  Pour régler la taille du panneau de lecture, cliquez puis déplacez la ligne de séparation du panneau vers le haut ou vers le bas. Par défaut, l'affichage des mails dans le panneau de lecture est activé. Vous pouvez déplacer le panneau de lecture vers la gauche ou désactiver le Panneau de lecture à partir du lien Affichage de la barre d'outils.

  5. Panneau d'aperçu. Affiche vos différents comptes. Chaque nom de compte apparaît. Cet exemple affiche la présentation Toutes les boîtes aux lettres. Tous les messages à partir de chacun de vos comptes pour les dossiers Boîte de réception, Envoyés et Brouillons sont combinés sous cette section. Cette préférence peut être définie à partir de la page Préférences>Général.

    Chaque compte inclut les dossiers suivants :

    • Les Dossiers, y compris les dossiers système Boîte de réception, Envoyés, Brouillons, Spam, et Corbeille, ainsi que les dossiers personnalisés que vous avez créés (le cas échéant).

    • Les Recherches que vous avez sauvegardées.

    • LesTags que vous avez créés.

    • Les liens Zimlet créés pour intégrer les applications tierces à votre boîte aux lettres.

  6. Un mini-calendrier peut s'afficher au-dessous de la fenêtre d'aperçu. L'affichage du mini-calendrier est facultatif et peut être activé sur la page Préférences>Calendrier.

Ouverture de Zimbra Desktop

Quand vous vous connectez à Internet, si votre compte est défini pour la synchronisation automatique, de nouveaux mails sont téléchargés sur votre Zimbra Desktop. Il n'est pas nécessaire d'ouvrir Desktop pour effectuer cette opération. Une icône enveloppe s'affiche dans Windows Tray (ou dans la ligne Mac Dock) affichant le nombre total de nouveaux messages dans tous vos comptes depuis la dernière synchronisation de Zimbra Desktop. Cliquez sur l'icône en forme d'enveloppe pour ouvrir l'application de bureautique.

 

Mail et Notification de rendez-vous

Il est possible de configurer une notification de rendez-vous (envoi automatique de mail). Plusieurs options de notification sont disponibles.

  • Pour Windows, la notification est une icône en forme d'enveloppe. Pour Mac, vous devez installer au préalable Growl avant de pouvoir recevoir la notification de mail et de rendez-vous. Vous devrez également activer Afficher un message de notification dans Préférences>Mail et >Calendrier. Remarque : Cette fonction n'est pas disponible dans Linux.

 

  • Dans Préférences>Mail et >Calendrier, vous pouvez également configurer d'autres actions de notification, y compris émettre un son, contraster l'onglet Mail ou Calendrier ou faire clignoter le titre du navigateur. Les messages de notification sont des paramètres spécifiques à chaque compte. Certains comptes peuvent être activés, d'autres pas.
