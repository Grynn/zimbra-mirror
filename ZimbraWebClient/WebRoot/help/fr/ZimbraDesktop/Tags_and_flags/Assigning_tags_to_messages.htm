Affecter des tags aux mails et à d’autres objets

Le marquage avec un tag est similaire au marquage avec un drapeau. La différence, c'est que vous créez et appliquez plusieurs tags à un seul élément afin de le classer simultanément dans plusieurs catégories. Après la création d’un tag, vous pouvez l’appliquer à des conversations, des mails ou des contacts.

Remarque : les tags sont créés dans le compte dans lequel vous travaillez. Ces tags ne sont pas partagés avec les autres comptes que vous créez.

Pour affecter un tag à un élément en utilisant l’option glisser-déposer :

Sélectionnez et faites glisser le tag du panneau d’aperçu situé à gauche vers l’élément dans la fenêtre de contenu.

Pour appliquer un tag à un élément à l’aide des menus :

  1. Faites un clic droit sur la fenêtre de contenu.

  2. Choisissez Marquer le mail (ou Marquer la conversation ou Marquer le contact, etc.).

  3. Choisissez le nom du tag dans la liste qui apparaît. Seuls les tags qui ne sont pas encore appliqués à l'élément sélectionné s'affichent.

Pour créer un tag et l’appliquer à un élément :

  1. Faites un clic droit dans la liste puis choisissez l’option du menu Tag qui s’affiche.

  2. Choisissez Nouveau Tag.

  3. Donnez un nom au tag puis cliquez sur OK.

Pour supprimer un tag d’un élément :

  1. Faites un clic droit dans la liste puis choisissez l’option du menu Tag qui s’affiche.

  2. Choisissez Retirer Tag. Si l’élément est marqué de plusieurs tags, vous pouvez choisir de retirer un tag spécifique ou l’ensemble des tags.

Pour appliquer/retirer un tag à plusieurs éléments à la fois :

  1. Choisissez tous les objets à marquer d’un tag à l’aide de Ctrl-Clic ou de Maj.-Clic.

  2. Faites un clic droit sur l’un des éléments sélectionnés, puis choisissez l’option du menu Tag qui s’affiche.

  3. Choisissez le tag à ajouter ou à retirer.

Pour rechercher rapidement les mails marqués d’un tag :

Cliquez sur un tag dans le panneau d’aperçu.

Alternative : vous pouvez aussi effectuer une recherche de tags. Dans le champ Rechercher saisissez la requête sous la forme tag : <"nom du tag">, puis cliquez sur Rechercher. Une recherche de tag :Pourpre indiquerait l’ensemble des éléments qui sont marqués d’un tag nommé Pourpre.
