Configuration de vos préférences Desktop

L'onglet Préférences contient des pages de fonctions qui vous permettent de personnaliser le fonctionnement de votre boîte aux lettres, carnet d'adresses et calendrier, pour chaque compte que vous configurez sur Zimbra Desktop. Le comportement par défaut est défini lorsque vous configurez un compte sur Zimbra Desktop.

Les modifications que vous apportez dans le dossier Préférences générales affectent la présentation de base Zimbra Desktop pour tous les comptes.

Global

Les modifications apportées à cet onglet affectent tous les comptes de messagerie de Zimbra Desktop.

  • Thème affiche une liste des différentes skins de l'interface (UI). Vous pouvez à tout moment modifier la couleur de fond de votre messagerie.

  • Langue. La langue utilisée pour afficher le texte dans Zimbra Desktop. (Disponible pour le client Web de Zimbra)

  • Si le fuseau horaire affiché sur cet onglet est incorrect, modifiez le Fuseau horaire par défaut. Il détermine la date et l'heure des mails que vous envoyez ou recevez, et l'heure à afficher pour les rendez-vous de votre calendrier. Celui-ci peut être différent de la configuration de fuseau horaire de votre ordinateur.

  • Dans Paramètres de recherche, vous pouvez choisir d'inclure automatiquement dans les dossiers Spam et Corbeille toutes les recherches que vous effectuez. Par défaut, ces dossiers ne sont pas pris en compte lors des recherches.

  • Dans Langue de recherche, cochez Afficher les fonctions évoluées dans la barre d'outils de recherche pour afficher les détails de cette chaîne dans la zone de texte de recherche.

  • Activez Afficher les cases à cocher qui facilitent la sélection des objets dans les listes pour afficher une coche pour chaque objet dans le panneau Contenu. Utilisez les cases à cocher pour sélectionner un ou plusieurs éléments sur lesquels exécuter la même action. Par exemple : supprimer, déplacer, marquer comme lu/non lu.

  • Valeurs par défaut. Activez cette option pour faire de Zimbra Desktop votre application de messagerie par défaut. Lorsque vous envoyez un mail ou cliquez sur un mail pour ouvrir un lien vers une page Web, la fenêtre Écrire de Zimbra Desktop s'ouvre.

Mail

  • Afficher. Sélectionnez le nombre d'éléments (mails ou conversations) à afficher par page. Par défaut, l'affichage est de 25 éléments par page, mais vous pouvez choisir 10, 50, ou 100 éléments par page.

  • Afficher le mail. Spécifiez si voulez afficher les mails au format HTML pour les messages formatés en HTML. En désactivant cette option, les mails apparaissent désormais en format texte simple, affichant les balises HTML au lieu de les appliquer en tant que code.

  • Aperçu des mails.

    • Afficher un extrait du contenu de chaque mail. Lorsque cette option est activée, les extraits de contenu de chaque mail sont affichés lorsque le curseur est placé sur l'objet d'un mail.

    • Lorsque vous double-cliquez sur un mail, celui-ci s'ouvre dans une nouvelle fenêtre si l'option Double-cliquez pour ouvrir le mail dans une nouvelle fenêtre est activée.

  • Images. Activez Images pour télécharger automatiquement les images de votre mail en HTML. Lorsque cette fonction est activée, il n'est pas nécessaire de cliquer sur l'image du mail pour voir l'image.

  • Lorsque j'affiche un mail dans le panneau de lecture définit le comportement pour marquer les mails consultés dans le panneau de lecture comme lus ou non lus. Vous pouvez configurer vos mails pour qu'ils soient marqués comme lus immédiatement, comme lus après un nombre de secondes défini de consultation dans le panneau de lecture, ou toujours conserver comme non lus les mails consultés dans le panneau de lecture.

  • Recherche Mail par défaut. Ce paramètre définit la recherche à exécuter lorsque vous vous connectez. La recherche par défaut est dans votre Boîte de réception. Lorsque vous ouvrez Desktop, les résultats de votre Boîte de réception s'affichent. Vous pouvez modifier ce champ pour qu'un autre dossier, tag, ou recherche enregistrée apparaissent lorsque vous ouvrez le Desktop pour la première fois. Consultez la section Description du langage de demande pour connaître la syntaxe.

  • Réception des mails.

Lorsqu'un mail arrive :

    • Émettre un son. Un signal sonore ("bip") vous avertit lorsque vous recevez un mail dans votre Boîte de réception. Si vous avez mis le volume de votre ordinateur en sourdine, vous n'entendrez aucun son.

    • Activer l'onglet Mail. Si vous n'êtes pas en train d'utiliser l'onglet Mail, ce dernier est contrasté lorsque vous recevez un mail dans votre Boîte de réception.

    • Afficher brièvement le nom du navigateur. Lorsque vous recevez un mail dans votre Boîte de réception, le navigateur clignote jusqu'à ce que vous ouvriez l'onglet Mail.

    • Afficher un message de notification. Lorsque vous recevez un mail, une boîte de dialogue miniature affiche les informations de ce mail.

Les options suivantes sont uniquement disponibles sur le compte Zimbra Web Client.

  • Spécifiez une adresse vers laquelle faire suivre votre mail. Vous pouvez supprimer le mail original de votre messagerie Zimbra.

  • Spécifiez si vous souhaitez envoyer une réponse automatique par mail à un mail entrant, et définissez la date de début et d'arrêt. Cette fonction est fréquemment utilisée pour les messages d'absence. Vous pouvez également spécifier une date de début et d'arrêt pour utiliser ce message.

  • Mails envoyés par moi-même. Configurez la section Mails envoyés par moi-même pour traiter les mails que vous envoyez et qui contiennent votre adresse parmi les destinataires du mail, ou une adresse telle qu'une liste de diffusion à laquelle vous appartenez.

Création de mail

  • Définissez l'éditeur que vous voulez utiliser par défaut, HTML ou texte simple.

  • Définissez les paramètres de police par défaut à utiliser pour rédiger en HTML dans Mail, Documents et Calendrier. Vous pouvez définir le style, la taille et la couleur des polices à utiliser par défaut. La police par défaut est : Arial, 12 pt, noir.

  • Toujours ouvrir une nouvelle fenêtre pour créer un mail lance une nouvelle fenêtre de mail, séparée de votre boîte aux lettres, lorsque vous rédigez un mail. Lorsque la fenêtre de création de mail apparaît, vous pouvez consulter vos mails et naviguer dans votre messagerie.

  • Les mails que vous rédigez sont automatiquement enregistrés en tant que brouillons après quelques minutes. Si vous préférez que les mails ne soient pas automatiquement enregistrés, désactivez cette fonction.

  • Choisissez d'inclure le texte original dans le corps de votre message de réponse. Vous pouvez choisir d'inclure le texte original, de ne pas l'inclure, de l'inclure sous forme de pièce jointe, de l'inclure dans le corps avec un préfixe de votre choix, ou d'inclure uniquement le mail le plus récent. (autrement dit, il reprendra le texte écrit par la personne qui a envoyé le mail, mais pas le texte précédent.)

  • Si vous choisissez d'inclure le texte original, précisez si vous voulez ajouter un préfixe à chaque ligne avec > pour les mails précédents que vous avez fait suivre ou auxquels vous avez répondu.

  • Par défaut, l'option Garder une copie des mails dans le dossier Envoyés est activée. Une copie des mails que vous envoyez est enregistrée dans votre dossier Envoyés.

  • Spécifiez comment inclure le texte original d'un mail lorsque vous le faites suivre : dans le corps du mail, dans le corps avec un préfixe, ou en tant que pièce jointe.

Signatures

Vous pouvez créer des signatures pour vos mails. Votre signature peut inclure votre nom et du texte supplémentaire comportant le nombre maximum de caractères autorisé par votre compte. Si vous créez plusieurs identités, vous pouvez créer plusieurs signatures et les attribuer à plusieurs adresses.

Contacts

  • Choisissez si vous préférez voir les contacts sous forme de liste ou de cartes professionnelles.

  • Sélectionnez le nombre de contacts à afficher par page. Le nombre par défaut est 25.

  • Activez la fonction pour ajouter automatiquement des adresses vers le dossier Contacts mail lorsque vous envoyez un mail.

    Le carnet d'adresses Personnes contactées par mail se remplit chaque fois que vous envoyez un mail à une adresse qui ne fait pas partie de vos autres carnets d'adresses. Vous pouvez désactiver cette fonction affichant Préférences, sous l'onglet Carnet d'adresses. Décochez la case sur Paramètres, Ajouter les nouveaux contacts dans "Personnes contactées par mail".

Avatars

L'onglet Comptes sert à créer et à gérer vos avatars. Un avatar est une identité mail utilisée dans le champ Envoyé par des mails que vous envoyez.

Le style par défaut est défini dans les paramètres de compte principaux. Lorsque vous créez de nouvelles identités, vous pouvez utiliser les mêmes paramètres que l'identité par défaut, ou vous pouvez configurer des préférences différentes.

Filtres de mails

Utilisez l'onglet Filtres de mails pour définir les règles de filtre de vos mails. Pour apprendre à configurer les règles de filtre des mails, consultez la rubrique Fonctionnement du filtre.

Calendrier

  • Choisissez le mode d'affichage de votre calendrier Zimbra. Par défaut, l'affichage défini est par semaine ouvrée.

  • Spécifiez avec quel jour de la semaine vous souhaitez débuter la semaine de votre calendrier.

  • Un mini- calendrier s'affiche dans Calendrier. Vous pouvez activer le mini-calendrier pour qu'il soit visible depuis n'importe quelle vue.

  • Vous pouvez activer la suppression automatique d'une invitation après y avoir répondu ou, dans le cas contraire, l'invitation reste dans votre Boîte de réception.

  • Configurer la notification de rappel pour les réunions. Vous pouvez définir la date de rappel d'une réunion sur Jamais pour être averti jusqu'à 60 minutes avant le début de la réunion. Vous pouvez choisir le type de rappel, sonore, afficher brièvement le nom du navigateur, ou afficher un message de notification.

  • Si vous créez fréquemment des rendez-vous qui n'incluent pas de participants invités, cochez la case Utilisez la boîte de dialogue QuickAdd/Ajout rapide.

  • Si vous changez de fuseau horaire au cours de vos voyages, il est possible d'afficher la liste des fuseaux horaires dans la vue des rendez-vous. Ainsi, lorsque vous programmez des rendez-vous, vous pouvez modifier le fuseau horaire si nécessaire.

Importer/Exporter

  • Consulter la section Sauvegarde du contenu de votre compte pour exporter tous les dossiers de votre compte ou spécifier les dossiers de votre compte.

  • Importer à nouveau dans votre compte les dossiers exportés

  • Rechercher des types de fichier spécifiques à exporter

  • Importez et exportez des programmes de calendrier.

  • Importez et exportez des listes de contacts.

Raccourcis

Les raccourcis clavier sont des touches ou combinaisons de touches qui vous permettent de sélectionner rapidement une action lorsque la séquence de touche d'accès est saisie.  

Allez dans votre dossier Raccourcis Préférences pour voir la liste complète des raccourcis clavier ou cliquez sur Ctrl Q à partir de tout affichage et la liste des raccourcis clavier s'affiche.
