Gestion de l’affichage de vos mails

Vous pouvez afficher vos mails comme des messages simples ou conversations et vous pouvez consulter les messages avec le Panneau de lecture affiché au bas de la page ou sur la droite de la page.

Pour modifier la manière dont les messages sont regroupés dans votre boîte aux lettres, dans la barre d'outils, cliquez sur Affichage et choisissez la vue par conversation ou par message et l'emplacement d'affichage du Panneau de lecture.  

Affichage des mails

L’affichage traditionnel présente une liste de mails en commençant par le mail le plus récent. Les mails non lus sont affichés en gras. Les informations affichées sur une ligne contiennent les éléments suivants : colonnes de priorité (drapeaux), libellés (tags), De, présence de pièces jointes, objet, emplacement du dossier, taille du mail, y compris pièces jointes et heure à laquelle le mail a été reçu.

Affichage de conversation

La vue de conversation affiche vos mails regroupés par objet. Le nombre de messages dans la conversation s’affiche entre parenthèses après l’objet. Chaque conversation contient le message d'origine et l'ensemble des réponses. Les conversations contenant des messages non lus sont affichées en gras.

 

Double-cliquez sur la conversation pour afficher les messages dans la conversation. Cliquez sur un message pour afficher son contenu. Dans le lien Affichage, vous pouvez choisir d'afficher d'abord le mail le plus ancien ou le plus récent.

Tous les messages associés à la conversation sont affichés, même s'ils sont conservés dans d’autres dossiers. La colonne Dossier affiche le dossier dans lequel le message est conservé.

Navigation dans le panneau de contenu des mails

En affichage message ou conversation, chaque mail dispose d'une ligne distincte qui affiche les colonnes d'informations suivantes :

  • Indicateur de conversation. Ce mail fait-il partie d'une conversation ?

  • Indicateur de drapeau. L'expéditeur a-t-il affecté un drapeau à ce message ?

  • Indicateur de priorité. Ce mail a-t-il été affecté d'une haute priorité ?

  • Tags. Le mail a-t-il été marqué d'un Tag que vous avez défini ?

  • Statut. Les enveloppes ouvertes ou fermées indiquent si le mail a été lu ou non. Le texte en gras indique également que le message n'a pas été lu.

  • De. Nom ou adresse mail de l'expéditeur.

  • Indicateur de pièce jointe. L'icône d'un trombone indique qu'un mail contient une pièce jointe.

  • Objet. Courte description du mail. Si les extraits de contenu sont activés dans votre compte, la première ligne du mail s'affiche.

  • Dossier. Affiche le nom du dossier dans lequel le message est conservé. Pour les conversations, le nom du dossier ne s'affiche pas parce que les mails peuvent inclure plusieurs dossiers d'intervalles.

  • Taille. Taille du mail ou nombre de mails contenus dans la conversation.

  • Reçu. Date ou heure à laquelle le mail a été reçu.

Trier les mails dans le panneau de contenus

Vous pouvez trier les messages en utilisant les colonnes De, Objet, Taille ou Reçu.

Pour trier par rapport à une colonne spécifique, cliquez sur le titre de la colonne. La colonne affiche une flèche indiquant si le tri est effectué par ordre croissant (flèche Haut) ou par ordre décroissant (flèche Bas).

Vous pouvez trier une colonne à la fois.
