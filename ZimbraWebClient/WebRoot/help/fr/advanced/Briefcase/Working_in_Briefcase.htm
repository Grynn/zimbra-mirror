Travailler dans un Porte-documents

Le Porte-documents peut être utilisé pour partager et gérer les documents que vous créez dans votre Porte-documents, ainsi que les documents et les images que vous y placez. Le Porte-documents vous permet de sauvegarder des fichiers dans votre compte pour que vous puissiez accéder à ces fichiers lorsque que vous vous connectez à votre compte depuis n'importe quel ordinateur.

Vous pouvez uploader des documents, des feuilles de calculs, des présentations, des images et des fichiers pdf à partir de votre ordinateur ou de votre réseau informatique, et vous pouvez sauvegarder dans votre Porte-documents des pièces jointes envoyées avec vos mails.  Vous pouvez créer plusieurs dossiers de Porte-documents afin d’organiser les fichiers. Vous pouvez également partager le dossier Porte-documents avec d'autres utilisateurs.

Les fichiers contenus dans le Porte-documents sont des copies des fichiers originaux.  Les modifications apportées aux fichiers de votre porte-documents ne modifient pas le fichier original.

Création de dossier de Porte-documents

  1. Activez l’onglet Porte-documents.

  2. Sélectionnez Porte-documents dans la Nouvelle liste  de la barre d'outils.

  1. La boîte de dialogue Créer un porte-documents s’affiche. Saisissez un nom pour le nouveau dossier de Porte-documents.

Ce nom est doit être unique dans la hiérarchie des dossiers de votre boîte aux lettres. Les dossiers de Porte-documents ne peuvent pas avoir le même nom qu'un dossier de niveau supérieur dans votre mail @CompanyName@, calendrier ou carnet d'adresse. Par exemple, si vous avez un calendrier intitulé Vacances, vous ne pouvez pas nommer un Porte-documents de niveau supérieur Vacances. Vacances peut être le nom d’un porte-documents dans un autre porte-documents.

  1. Sélectionnez la  hiérarchie des dossiers. Vous pouvez créer un porte-documents de niveau supérieur, ou vous pouvez l’insérer dans un porte-documents existant.

  2. Cliquez sur OK. Le nouveau porte-documents s’affiche dans le panneau d’aperçu.

Uploader des fichiers

Quand vous uploadez un fichier, vous transférez un fichier de votre ordinateur vers votre compte. Cela permet de rendre le fichier disponible à chaque fois que vous vous connectez à votre compte. Le quota de votre compte détermine combien de fichiers peuvent être uploadés dans votre Porte-documents.

Pour uploader des fichiers

  1. Ouvrez Porte-documents, puis cliquez sur Uploader fichier dans la barre d’outils

  2. Dans la boîte de dialogue Uploader un nouveau fichier dans le Porte-documents, cliquez sur Parcourir pour trouver les fichiers à uploader. Vous pouvez sélectionner plusieurs fichiers à uploader une fois que les fichiers sont dans le même dossier. Pour ajouter d'autres fichiers, cliquez sur Ajouter et recherchez le fichier.

Vous ne pouvez pas uploader de fichier portant le même nom dans le même porte-documents. Dans la boîte de dialogue Upload, vous pouvez spécifier comment les fichiers ayant le même nom doivent être traités lorsque vous tentez d'uploader le fichier. Si vous modifiez cette option, la modification est enregistrée dans la boîte de dialogue.

    • Si vous voulez être averti lorsque vous uploadez un fichier portant le même nom, choisissez l'option Me demander. Si un fichier porte le même nom qu’un fichier précédemment uploadé, la boîte de dialogue Conflit d’upload s’affiche. Vous avez le choix de conserver votre fichier existant ou de le remplacer par le fichier uploadé. Vous pouvez également consulter le nouveau fichier. Cette option est définie par défaut.

    • Si vous voulez toujours que le fichier que vous uploadez remplace le fichier existant portant le même nom, choisissez Le remplacer par mon fichier. Le nouveau fichier est uploadé et remplace le fichier existant. Vous ne recevrez pas de notification de cette opération.

    • Si vous ne voulez pas que les fichiers de votre porte-documents soient remplacés, choisissez Conserver le fichier existant. Le fichier n’est pas téléchargé et vous ne serez pas averti de cette opération.

  1. Lorsque tous les fichiers sont uploadés, cliquez sur OK. Le fichier s’affiche dans le porte-documents.  

Remarque : Il est possible que votre compte impose une taille limite pour uploader des fichiers. Si le fichier est trop volumineux, un avertissement s’affiche.
Les fichiers que vous uploadez ont une répercussion sur le quota de votre compte.  

Ajouter des pièces jointes de mails à un fichier du porte-documents

Les pièces jointes de mails peuvent être ajoutées rapidement à votre porte-documents.

  1. Pour ajouter la pièce jointe, cliquez sur le lien Porte-documents qui se trouve dans l'en-tête du mail à côté du lien de la pièce jointe.

  2. Dans la boîte de dialogue Ajouter au porte-documents, choisissez le dossier de destination ou créez un nouveau dossier de porte-documents.

  3. Cliquez sur OK.

Afficher le contenu dans les panneaux d'aperçu

Vous pouvez sélectionner le mode d'affichage de vos fichiers de vos dossiers du Porte-documents dans un panneau d'aperçu.

  • Panneau d'aperçu en bas/Panneau d'aperçu à droite. Tous les fichiers au format reconnu sont affichés dans le panneau d'aperçu. Vous pouvez désactiver le panneau d'aperçu.

Partager votre Porte-documents

Les dossiers de Porte documents peuvent être partagés avec d'autres utilisateurs dans votre société et en dehors. Vous pouvez autoriser d'autres utilisateurs  à créer et à modifier le contenu d'un Porte-documents ou simplement consulter les pages.

Pour partager votre porte-documents, consultez la section Partager votre Porte-documents
