Accepter l’accès à un calendrier partagé

Si vous recevez une notification par mail indiquant que vous êtes autorisé à partager le calendrier d'un autre utilisateur, vous pouvez alors accepter ou refuser le partage.

Si vous acceptez le partage, le calendrier partagé s’affiche dans votre liste du Calendrier.

Pour accepter l’accès à un calendrier partagé

  1. Cliquez sur Accepter le partage dans le mail. La boîte de dialogue Accepter le partage s’affiche et décrit le rôle qui vous a été attribué.

  2. Avant d’accepter, vous pouvez personnaliser le nom du calendrier et choisir la couleur d’affichage des messages.

  3. Cochez le mail envoyé qui traite de ce partage, et envoyez une confirmation au cédant. (facultatif).

  4. Cliquez sur Oui. Le nouveau calendrier partagé a été ajouté à votre liste du Calendrier. Ce message a été déplacé vers le dossier Corbeille.
