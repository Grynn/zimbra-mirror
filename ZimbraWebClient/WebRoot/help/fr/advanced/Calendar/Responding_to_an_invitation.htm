Répondre à une invitation de réunion

Lorsque vous recevez une invitation à une réunion, vous recevez un mail de notification dans votre boîte de réception, et si l'option Ajouter automatiquement dans l'agenda les rendez-vous reçus est activée sous Préférences > Calendrier, la réunion est également ajoutée à votre calendrier par défaut et marquée comme Nouveau. Vous pouvez répondre rapidement à l’invitation depuis la Boîte de réception du mail ou depuis le panneau de la Liste du calendrier.

Remarque : Lorsque vous recevez une invitation par mail en tant que pièce jointe au format .ics, cliquez sur Ajouter dans le calendrier dans l'en-tête pour enregistrer l'invitation dans votre calendrier  Sélectionnez le calendrier où l'invitation devrait être enregistrée.

Pour répondre à une invitation

  1. Ouvrez le mail qui contient la demande de réunion, soit dans la Boîte de réception, soit en cliquant avec le bouton droit sur le rendez-vous dans le Calendrier.

  2. Pour répondre à l'invitation, cliquez sur  Accepter, À confirmer, ou Refuser. Une réponse est automatiquement envoyée si le créateur l'a exigée. Vous pouvez ajouter des commentaires avant d’envoyer votre réponse. Vous pouvez sélectionner le calendrier sur lequel programmer la réunion. Si vous ne sélectionnez pas de calendrier spécifique, votre réunion sera enregistrée sur le Calendrier par défaut.

Pour ajouter des commentaires à une réponse lorsque vous acceptez une invitation en vue Calendrier, faites un clic droit sur l'invitation, puis sélectionnez Modifier la réponse.

Si l'heure de la réunion ne vous convient pas mais que vous souhaitez tout de même y participer, vous pouvez proposer un autre horaire. Cliquez sur Proposer un nouvel horaire pour sélectionner un autre horaire. L'organisateur de la réunion peut décider d'accepter ou de décliner la proposition.

Après avoir fait votre choix, le message est envoyé à la Corbeille.

Le message est marqué par défaut comme public. Si le créateur a marqué le message comme privé, l'invitation sera reçue de manière privée ou, si vos préférences sont définies de manière à marquer tous les messages comme privés, le message sera marqué comme privé sur le calendrier.

Dans votre calendrier, cliquez sur la notification de réunion pour consulter à nouveau les invitations ou pour voir les pièces jointes qui ont été envoyées.

Vous pouvez déplacer une invitation vers un autre calendrier si vous ne voulez pas qu'elle apparaisse sur votre calendrier par défaut. Dans le mail, cliquez sur Calendrier en haut du message, puis sélectionnez le calendrier.

Remarque : si le créateur modifie ou annule une réunion que vous avez acceptée ou marquée comme "à confirmer", la modification est automatiquement effectuée sur votre calendrier.  Le créateur peut choisir de vous avertir ou non de ces modifications.

Modifier vos préférences de calendrier pour les invitations de réunion

Si vous ne souhaitez pas que les réunions que vous n'avez pas acceptées soient automatiquement ajoutées à votre calendrier, vous pouvez désactiver l'option Ajouter automatiquement dans l'agenda les rendez-vous reçus depuis la page Préférences > Calendrier. Les réunions sont ajoutées à votre calendrier uniquement si vous cliquez sur Accepter ou À confirmer dans le mail d'invitation. En cliquant sur Refuser, le rendez-vous ne sera pas ajouté à votre calendrier.

Si l'option Ajouter automatiquement dans l'agenda les rendez-vous reçus est activée, les rendez-vous refusés seront affichés sur votre calendrier dans une vue estompée, afin de vous rappeler la réunion à laquelle vous avez refusé de participer.  Vous pouvez supprimer les messages refusés à tout moment.

Par défaut, les messages d'invitation à une réunion de votre boîte de réception sont déplacés vers votre dossier Corbeille lorsque vous y répondez.  Si vous souhaitez que le message reste dans votre boîte de réception, désactivez l'option Après réponse à une invitation...Après réponse, supprimer l'invitation depuis la page Préférences > Calendrier.
