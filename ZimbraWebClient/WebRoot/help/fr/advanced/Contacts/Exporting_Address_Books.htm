Exportation des listes de contact du carnet d'adresses

Lorsque vous exportez une liste de contacts, celle-ci est enregistrée au format .csv (dans le fichier généré, les différents contacts sont séparés par une virgule).

Pour exporter votre carnet d'adresses

  1. Sélectionnez Préférences, puis cliquez sur Importer / Exporter

  2. Dans la section "Export", sélectionnez le type Contacts.

  3. Dans le champ Source, cliquez pour sélectionner la liste de contacts à exporter.

  1. Cliquez sur Exporter.

  2. Sélectionnez l'emplacement de destination du fichier exporté, puis cliquez sur OK. Pour plus de détails sur l'importation de ces contacts dans l'application cible, consultez la documentation de celle-ci.
