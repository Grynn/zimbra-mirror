Utilisation de l’éditeur HTML

Vous pouvez rédiger vos mails en utilisant HTML en tant qu’éditeur de format Rich Text (RTF).  Les mails créés avec HTML génèrent un formatage, une numérotation, des points, des fonds colorés, des tableaux, et des liens qui facilitent leur lecture.

  • Il est possible que certains clients de messagerie n’acceptent pas les mails formatés en HTML.

Dans l'onglet Préférences, Générales du client Web, vous définissez l’éditeur HTML comme votre éditeur par défaut et configurez les réglages de polices par défaut, y compris le style, la taille et la couleur.
