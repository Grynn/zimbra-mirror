Joindre des fichiers à votre mail

Lorsque vous rédigez un message, vous pouvez joindre des pièces jointes depuis votre ordinateur, des fichiers du porte-documents, des contacts de votre carnet d'adresses et des mails.

Remarque : Les destinataires de votre mail doivent disposer du logiciel approprié pour pouvoir ouvrir et lire le fichier. Pour les fichiers créés avec des logiciels communément utilisés, tels que Microsoft Office, les utilisateurs d'un système bureautique classique pourront les ouvrir. Les fichiers de formats courants, tels que HTML, et les fichiers images, tels que .GIF ou .JPG, peuvent être ouverts à partir de différents programmes. Néanmoins, cela dépend du type de système de l'utilisateur.

Pour joindre des fichiers en utilisant le lien Ajouter pièce jointe :

  1. Rédigez le mail comme décrit dans la section Création d'un mail.

  2. Cliquez sur  . La boîte de dialogue Joindre un ou plusieurs fichiers affiche les onglets pour Mon ordinateur, Porte-documents, Joindre mail et Contacts. Sélectionnez l'onglet correspond au type de pièce à joindre.

    • Mon ordinateur. Accédez à un dossier sur votre ordinateur, puis sélectionnez le fichier. Sélectionnez le fichier, puis cliquez sur Ouvrir.  Vous pouvez également faire glisser des fichiers depuis le dossier pour saisir l'en-tête du mail.

    • Porte-documents. Sélectionnez le dossier Porte-documents. Les fichiers contenus dans le porte-documents s'affichent sur la droite. Choisissez les fichiers à joindre.

    • Mail. Si ce mail n'est pas contenu dans votre boîte de réception, sélectionnez le dossier. Vous pouvez rechercher un mail dans le dossier que vous avez sélectionné. Les mails s'affichent à droite. Sélectionnez le mail à joindre.

    • Contacts. Sélectionnez le carnet d'adresses. Les noms du carnet d'adresses s'affichent sur la droite. Vous pouvez parcourir votre carnet d'adresses pour rechercher un nom. Choisissez les noms à joindre. Ils sont joints au message dans des fichiers .vcf.

Remarque : Utilisez les combinaisons de touches Ctrl-clic ou Maj-clic pour ajouter plusieurs pièces jointes à la fois. Plusieurs pièces jointes sont ajoutées à une ligne dans la boîte de dialogue Joindre les fichiers.

  1. Une fois toutes les pièces jointes ajoutées dans cet onglet, cliquez sur Joindre.

  2. Cliquez sur Envoyer pour envoyer le message et les pièces jointes.

Pour ajouter des pièces jointes à partir de votre ordinateur par glisser-déposer

  1. Sélectionnez un ou plusieurs fichiers et maintenez le curseur sur le ou les fichiers à joindre.

  2. Faites glisser le ou les fichiers vers la zone d'en-tête du mail et libérez le curseur. Les noms de fichiers s'affichent dans l'en-tête.

 

 

 

Suppression d'une pièce jointe

Pour supprimer une pièce jointe, désélectionnez la case de la pièce jointe avant d'envoyer le mail.
