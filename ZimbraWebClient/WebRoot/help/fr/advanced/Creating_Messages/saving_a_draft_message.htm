Enregistrement de mails en tant que brouillons

Vous pouvez enregistrer votre message sous forme de brouillon à terminer et l'envoyer ultérieurement.  Les messages sont enregistrés dans votre dossier Brouillons jusqu’à ce que vous décidiez de les envoyer ou les supprimer de ce dossier.

Pour créer un brouillon à modifier ultérieurement :

  • Rédigez votre mail comme décrit dans la section Création d’un mail.

  • Cliquez sur Enregistrer le brouillon.

  • Le brouillon du mail est enregistré dans votre dossier Brouillons.

Pour récupérer un brouillon

  • Ouvrez le dossier Brouillons , puis cliquez sur le message.

  • Vous pouvez rédiger et envoyer le message terminé de la même manière qu’un mail ordinaire. Après son envoi, le message est supprimé du dossier Brouillons.
