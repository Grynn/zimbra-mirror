Créer un document

Vous pouvez créer un document dans Porte-documents, concevoir la présentation de contenu en utilisant les outils en format RTF. Vous pouvez procéder à une vérification orthographique et utiliser des mots-clés pour créer des liens vers d'autres pages.

  1. Cliquez sur le dossier de porte-documents dans lequel vous souhaitez travailler. Le panneau Liste de documents affiche les fichiers déjà contenus dans le porte-documents.

  2. Dans la liste Nouveau sur la barre d'outils, sélectionnez Document. Une page Zimbra Docs vierge s'affiche.

  3. Dans la barre d'outils, entrez le nom du document.

  4. Rédiger le contenu de la page. Vous pouvez utiliser l'éditeur de format RTF pour ajouter des styles, des couleurs et des tableaux à votre page.

  1. Cliquez sur Enregistrer pour sauvegarder le fichier. Pour fermer le document sans sauvegarder, cliquez sur la croix (X) dans l'onglet du navigateur. Cliquez sur Enregistrer et fermer pour sauvegarder le fichier et revenir à la page Liste de documents.

Lorsque vous sauvegardez un document, sa version est modifiée.

Remarque : Si vous avez créé le document dans le mauvais porte-documents, vous pouvez déplacer le fichier dans le bon porte-documents en sélectionnant le document et en le faisant glisser vers le bon dossier.
