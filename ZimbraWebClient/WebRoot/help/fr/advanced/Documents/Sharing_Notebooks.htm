Partage de carnets

Vous pouvez partager vos carnets avec les personnes ou groupes suivants :

  • Utilisateurs ou groupes internes. Vous sélectionnez le type de privilèges à accorder aux utilisateurs internes, c’est-à-dire :

    • Visionneur. Le bénéficiaire peut consulter le contenu du dossier des carnets mais ne peut pas le modifier.

    • Gestionnaire. Le bénéficiaire dispose de l’autorisation complète pour consulter, modifier le contenu d'un carnet, ajouter des pages et en supprimer.

  • Invités externes. Vous créez un mot de passe pour accéder au dossier des carnets, et les invités doivent le saisir pour pouvoir consulter les pages. Ils ne sont pas autorisés à apporter des modifications aux informations.

  • Public. Toute personne ayant connaissance de l’URL menant au carnet peut consulter les pages.  Elle n'est pas autorisée à apporter de modifications aux informations.

Pour partager votre carnet

Vous pouvez choisir le niveau de carnet à partager. Si vous choisissez de partager le carnet de niveau supérieur, tous les carnets inclus dans celui-ci seront partagés.

  1. Cliquez avec le bouton droit sur le carnet que vous souhaitez partager.

  1. Cliquez sur Partager carnet. La boîte de dialogue Propriétés partagées s’affiche. Sélectionnez les personnes avec qui partager le dossier : utilisateurs ou groupes internes, invités externes, ou public.

  2. Selon le type d'utilisateur avec qui vous partagez le dossier, effectuez l'une des opérations suivantes :

  • Pour les utilisateurs ou groupes internes, saisissez l’adresse mail, puis dans la section Rôle choisissez les privilèges d’accès.

  • Pour les invités externes, saisissez l’adresse des invités et le mot de passe permettant de consulter votre carnet.

  • Pour le public, indiquez aux utilisateurs l’URL qui s’affiche dans le champ URL. Par exemple, vous pouvez copier et envoyer par mail cette URL aux personnes souhaitant consulter la page.

  1. Dans la zone Mail , choisissez le type de mail à envoyer. Ce message peut être envoyé aux utilisateurs internes et aux invités externes. Les options sont les suivantes :

    • Ne pas envoyer un mail à propos de ce partage.. Aucun mail n’est envoyé.

    • Envoyer un mail standard. Le mail standard de notification de partage est envoyé à l’adresse mail.  Ce mail inclut le nom du carnet à partager, votre adresse, l’adresse du bénéficiaire et le rôle attribué avec une liste d'actions autorisées.

    • Ajouter une note au mail standard. Vous pouvez ajouter des informations complémentaires au mail standard.

    • Envoyer le mail dans une nouvelle fenêtre. Écrire et envoyer un mail.

  1. Cliquez sur OK.

  2. Rappel : si vous avez choisi l’option Public, vous devrez indiquer l’URL aux utilisateurs.
