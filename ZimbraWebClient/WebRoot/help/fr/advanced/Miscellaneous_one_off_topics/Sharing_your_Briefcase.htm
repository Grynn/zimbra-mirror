Partager votre Porte-documents avec d’autres personnes

Vous pouvez partager vos Porte-documents avec :

  • Les utilisateurs ou groupes internes.  Vous choisissez le type de privilèges à accorder, ou :

    • Visionneur. Le bénéficiaire peut consulter les fichiers de votre porte-documents mais ne peut pas les modifier.

    • Gestionnaire. Le bénéficiaire dispose de l’autorisation complète pour modifier et supprimer les fichiers de votre porte-documents.

    • Aucune est une option pour désactiver temporairement l’accès au porte-documents d’un cédant sans retirer les privilèges de partage. Le dossier s'affiche encore dans la liste du bénéficiaire mais ce dernier ne peut plus consulter ni gérer les fichiers.

  • Invités externes. Vous créez un mot de passe pour accéder au porte-documents partagé. Les invités doivent saisir ce mot de passe pour pouvoir consulter les fichiers. Ils ne peuvent apporter de modifications aux fichiers.

  • Public. Tout utilisateur ayant connaissance de l’URL menant au porte-documents partagé peut consulter son contenu. Ces utilisateurs ne peuvent pas apporter de modifications au porte-documents.

Pour partager votre porte-documents

Cliquez avec le bouton droit sur le dossier porte-documents à partager, puis choisissez Partager dossier.

Sélectionnez la personne avec qui partager le calendrier sur la boîte de dialogue Propriétés du partage.

Utilisateurs ou groupes internes

  1. Si vous choisissez l’option Interne, vous avez la possibilité de sélectionner des adresses dans la liste qui s’affiche lorsque vous commencez à les saisir – saisie automatique – ou vous pouvez saisir des adresses qui ne figurent pas dans la liste. L’utilisateur ou le groupe doit disposer d’une adresse valide dans le serveur mail de votre société.

  2. Dans la section Rôle, sélectionnez les autorisations d’accès.

  1. Dans la zone Message, sélectionnez le type de message à envoyer.  Les options sont les suivantes :

  • Envoyer un mail standard. Le mail standard de notification de partage est envoyé à l’adresse mail.  

  • Ajouter une note au mail standard. Vous pouvez ajouter des informations complémentaires au mail standard.

  1. Cliquez sur OK.

 

Invités externes

  1. Si vous choisissez des invités Externes, saisissez leurs adresses mail.   Vous avez la possibilité de saisir plusieurs adresses mails dans le champ de messagerie. Insérez un point-virgule (;) entre les adresses.

  2. Saisissez le mot de passe que les utilisateurs devront entrer pour consulter votre dossier Porte-documents.

  1. Dans la zone Mail , choisissez le type de mail à envoyer.  L’URL menant à votre dossier ainsi que le mot de passe sont inclus dans le contenu du mail.

    • Envoyer un mail standard. Le mail standard de notification de partage est envoyé à l’adresse mail.

    • Ajouter une note au mail standard. Vous pouvez ajouter des informations complémentaires au mail standard.

  2. Cliquez sur OK.

Public

Si vous choisissez l’option Public, tout personne disposant de l’URL peut consulter votre porte-documents.  L’utilisation d’un mot de passe est facultative.

Modification ou retrait de l’autorisation

Vous pouvez modifier ou retirer les autorisations à tout moment.

  1. Cliquez avec le bouton droit sur le dossier porte-documents partagé.

  2. Cliquez sur Propriétés d'édition. Les utilisateurs partageant le dossier sont répertoriés.

  3. Cliquez sur Modifier pour modifier les autorisations, ou
    sur Retirer pour supprimer les autorisations.

  4. Si un mail décrivant les modifications doit être envoyé, sélectionnez l'option qui s'applique.

  5. Cliquez sur OK.
