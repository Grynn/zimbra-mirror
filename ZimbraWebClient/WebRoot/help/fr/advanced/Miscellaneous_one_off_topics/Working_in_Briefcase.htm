Travailler dans un Porte-documents

La fonction Porte-documents vous permet d’enregistrer des fichiers que vous utilisez dans le porte-documents. Vous pouvez accéder à ces fichiers dès lors que vous êtes connecté à votre compte, sur n’importe quel ordinateur.  

Vous pouvez télécharger des documents, des feuilles de calculs, des présentations, des images, des fichiers pdf à partir de votre ordinateur ou de votre réseau informatique, et vous pouvez sauvegarder dans votre porte-documents des pièces jointes envoyées avec vos mails.  Vous pouvez créer plusieurs dossiers de porte-documents afin d’organiser les fichiers. Vous pouvez également partager le dossier Porte-documents avec d'autres utilisateurs.

Les fichiers contenus dans le Porte-documents sont des copies des fichiers originaux.  Les modifications apportées aux fichiers de votre porte-documents ne modifient pas le fichier original.

Création de dossier de porte-documents

  1. Activez l’onglet Porte-documents.

  2. Cliquez sur Nouveau Porte-documents dans la barre d’outils.

  3. La boîte de dialogue Créer un porte-documents s’affiche. Donnez un nom au nouveau dossier de porte-documents

Ce nom est doit être unique dans la hiérarchie des dossiers de votre boîte aux lettres. Les dossiers de porte-documents ne peuvent pas porter le même nom qu'un dossier de niveau supérieur de votre messagerie, calendrier, ou carnet d'adresses Zimbra. Par exemple, si l'un de vos calendriers s’appelle Vacances, vous ne pouvez pas nommer un porte-documents de niveau supérieur du même nom. Vacances peut être le nom d’un porte-documents dans un autre porte-documents.

  1. Sélectionnez la  hiérarchie des dossiers. Vous pouvez créer un porte-documents de niveau supérieur, ou vous pouvez l’insérer dans un porte-documents existant.

  2. Cliquez sur OK. Le nouveau porte-documents s’affiche dans le panneau d’aperçu.

Uploader des fichiers

Quand vous uploadez un fichier, vous transférez un fichier de votre ordinateur vers votre compte ZWC. Cela permet de rendre le fichier disponible à chaque fois que vous vous connectez à votre compte.

Pour uploader des fichiers

  1. Ouvrez Porte-documents puis cliquez sur Uploader fichier dans la barre d’outils.

  2. Dans la boîte de dialogue Upload, cliquez sur parcourir pour rechercher le fichier à transférer. Pour ajouter un autre fichier, cliquez sur Ajouter.

Vous ne pouvez pas uploader de fichier portant le même nom dans le même porte-documents. Vous pouvez préciser dans la boîte de dialogue Uploader la manière dont les fichiers portant le même nom doivent être traités lorsque vous essayez d’uploader le fichier. Si vous modifiez cette option, la modification est enregistrée dans la boîte de dialogue.

  • Si vous voulez être averti lorsque vous uploadez un fichier portant le même nom, choisissez l'option Me demander. Si un fichier porte le même nom qu’un fichier précédemment uploadé, la boîte de dialogue Conflit d’upload s’affiche.  Vous avez le choix de conserver votre fichier existant ou de le remplacer par le fichier uploadé. Vous pouvez également consulter le nouveau fichier. Cette option est définie par défaut.

  • Si vous voulez toujours que le fichier que vous uploadez remplace le fichier existant portant le même nom, choisissez Le remplacer par mon fichier. Le nouveau fichier est uploadé et remplace le fichier existant. Vous ne recevrez pas de notification de cette opération.

  • Si vous ne voulez pas que les fichiers de votre porte-documents soient remplacés, choisissez Conserver le fichier existant. Le fichier n’est pas téléchargé et vous ne serez pas averti de cette opération.

  1. Lorsque tous les fichiers sont uploadés, cliquez sur OK. Le fichier s’affiche dans le porte-documents.  

  2. Il est possible que votre compte impose une taille limite pour uploader des fichiers. Si le fichier est trop volumineux, un avertissement s’affiche.
    Les fichiers que vous uploadez ont une répercussion sur le quota de votre compte.  

Ajouter des pièces jointes de mails au porte-documents

Les pièces jointes de mails peuvent être ajoutées rapidement à votre porte-documents.

  1. Pour ajouter la pièce jointe, cliquez sur le lien ajouter au porte-documents qui se trouve dans l’en-tête du mail à côté du lien de pièce jointe.

  2. Dans la boîte de dialogue Ajouter au porte-documents, choisissez le dossier de destination ou créez un nouveau dossier de porte-documents.

  3. Cliquez sur OK.

Consulter des fichiers du Porte-documents

Vous pouvez sélectionner le mode d’affichage des fichiers dans vos dossiers de Porte-documents. Dans la barre d’outils de Porte-documents, choisissez l'une des options dans le menu Affichage.

  • Affichage Explorer. Les fichiers s’affichent comme des vignettes.

  • Vue Détails. Les fichiers s’affichent sur une ligne horizontale comportant la taille du fichier, la date de l'upload, l'auteur, s'il s'agit d'un fichier partagé qui ne vous appartient pas, et le dossier dans lequel le fichier est sauvegardé.

  • Affichage du navigateur en colonnes. Les fichiers s’affichent de manière verticale. Dans cet affichage, vous pouvez cliquer sur un fichier pour consulter ses informations, notamment sa taille, sa date de création, son auteur, sa date de modification et l'auteur des modifications.

Ouvrir des fichiers du Porte-documents

Vous pouvez ouvrir des fichiers de vos porte-documents depuis n'importe quel ordinateur. Vous devez disposer de l’application logicielle adéquate pour pouvoir ouvrir, lire ou mettre à jour le fichier. Des formats de fichiers courants comme les fichiers texte et HTML, et les fichiers images comme les .gif ou .jpg peuvent être ouverts à partir de différents programmes.

  • Pour ouvrir un fichier, cliquez avec le bouton droit sur le fichier puis choisissez Ouvrir. Si le fichier est de type inconnu, une boîte de dialogue s’affiche vous demandant ce que vous voulez faire : l'enregistrer sur l'ordinateur ou l'ouvrir avec un programme spécifique.

Déplacer des fichiers

Vous pouvez déplacer des fichiers d’un porte-documents à un autre.

  1. Dans la barre d’outils, cliquez sur .

  2. Choisissez le dossier de destination ou créez un nouveau dossier de porte-documents.

  3. Cliquez sur OK.

Marquer des  fichiers

Vous pouvez marquer des fichiers.

  1. Cliquez sur le fichier, puis dans la barre d’outils, et enfin sur .

  2. Choisissez le tag. Vous pouvez uniquement afficher les tags dans la Vue Détails.

  3. Cliquez sur OK.

Envoyer un mail contenant un lien vers un fichier

Vous pouvez envoyer un mail qui contient un lien vers un fichier du porte-documents. Les destinataires doivent disposer du logiciel adéquat pour pouvoir ouvrir le fichier.

  1. Cliquez avec le bouton droit sur le fichier sélectionné.

  2. Choisissez Envoyer. Une boîte de dialogue de confirmation s’affiche.

  3. Cliquez sur Oui.
