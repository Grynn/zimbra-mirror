Personnaliser votre vue

Vous pouvez personnaliser la vue de votre compte en modifiant  la couleur de fond affichée lorsque vous vous connectez à votre compte, réorganiser le panneau Contenu de la vue Boîte aux lettres, de la vue Calendrier et de la vue Tâches pour afficher les informations dans l'ordre de votre choix.

Changer la couleur de fond

Vous pouvez changer la couleur de fond de votre boîte aux lettres dans Préférences, onglet Généralités.  Dans la section Options de connexion, le champ Thème contient la liste des thèmes disponibles.

Modifier l'affichage des mails

Par défaut, votre boîte aux lettres s'ouvre dans la vue Boîte de réception, mais vous pouvez modifier cette configuration :

  • Changer le dossier qui est affiché à chaque connexion.  La vue par défauit est la boîte de réception qui affiche tous les mails qui s'y trouvent.  Vous pouvez sélectionner tout autre dossier de mails comme vue initialise lorsque vous vous connectez. Vous pouvez sélectionner des critères de recherche, tels que les mails non lus.  Dans ce cas, seuls les mails non lus seront affichés lorsque vous vous connectez.  Voir Personnalisation de votre page de messagerie initiale.

  • Indiquez si vous souhaitez afficher le panneau de lecture et s'il doit s'afficher dans lebas de la page ou sur la droite dans la page. Pour effectuer ces modifications, cliquez sur le lien Vue dans la barre d'outils des mails.

  • Si le panneau de lecture est désactivé ou affiché dans le bas de la page, vous pouvez sélectionner les colonnes à afficher dans la liste des messages du panneau Contenu.  Vous n'êtes pas obligé d'afficher tous les détails relatifs aux mails dans le panneau Contenu. Pour afficher la liste des colonnes disponibles, cliquez avec le bouton droit de a souris sur les en-têtes des colonnes. Pour masquer une colone dans la vue du panneau Cotenu, cliquez sur son nom. Cette fonction étant une bascule, vous pouvez afficher de nouveau la colonne en cliquant à nouveau sur son nom dans la liste.

Modifier l'affichage du calendrier

Vous pouvez spécifier la vue dont vous souhaitez disposer lorsque vous ouvrez votre calendrier (Jour, Semaine ouvrée, Semaine de sept jours, Mois, Programme ou Liste). Vous pouvez également sélectionner le jour de début de vos semaines de calendrier. Voir Vues Calendrier.

Modifier l'affichage des tâches

Vous pouvez sélectionner les colonnes à afficher dans le panneau Contenu lorsque vous consultez vos tâches. Pour afficher la liste des colonnes disponibles, cliquez avec le bouton droit de la souris dans l'en-tête de la colonne Tâches. Pour masquer une colonne, cliquez sur la colonne dans la vue du panneau Contenu.  Cette fonction étant une bascule, vous pouvez replacer la colonne en cliquant de nouveau sur son nom dans la liste.

Indiquez si vous souhaitez afficher le panneau d'aperçu et s'il doit apparaît dans le bas ou sur la droite. Pour effectuer ces modifications, cliquez sur le lien Vue affiché dans la barre d'outils.

Modifier l'affichage du porte-documents

Vous pouvez sélectionner les colonnes à afficher dans le panneau Contenu lorsque vous consultez votre porte-documents. Pour afficher le menu des titres de colonne, cliquez avec le bouton droit de la souris dans l'en-tête de la colonne Porte-documents. Pour masquer une colonne, cliquez sur la colonne dans la vue du panneau Contenu.  Cette fonction étant une bascule, vous pouvez replacer la colonne en cliquant de nouveau sur son nom dans la liste.

Indiquez si vous souhaitez afficher le panneau d'aperçu et s'il doit apparaît dans le bas ou sur la droite. Pour effectuer ces modifications, cliquez sur le lien Vue affiché dans la barre d'outils.
