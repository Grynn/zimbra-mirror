Appliquer des actions à plusieurs sélections

Lorsque vous affichez un dossier en cliquant sur le nom du dossier à gauche, le contenu du dossier s'affiche avec l'un des éléments sélectionnés (mis en évidence). Vous pouvez sélectionner un ou plusieurs éléments pour exécuter les actions suivantes :

  • Cliquez pour sélectionner un seul élément à la fois. Si vous cliquez sur un autre élément, l'élément précédent est désélectionné.

  • Double cliquez pour ouvrir un élément.

  • Cliquez avec le bouton droit de la souris sur un élément pour afficher le menu d'actions applicables à l'élément. Vous pouvez également effectuer cliquer avec le bouton droit de la souris après avoir sélectionné plusieurs éléments afin d'appliquer simultanément la même action à tous les éléments.

  • Ctrl-clic sélectionne plusieurs éléments. Il peut également être utilisé pour désélectionner l'élément actuel et fonc tout désélectionner.

  • Maj.-clic sélectionne plusieurs éléments séquentiellement.  Cliquez sur un élément et utilisez Maj.-clic sur un autre élément plus bas dans la liste pour sélectionner tous les éléments qui se trouvent entre les deux éléments sélectionnés.

Vous pouvez sélectionner plusieurs éléments en cliquant avec le bouton droit de la souris, choisissez de les marquer comme non lus, d'ajouter un drapeau, de les supprimer ou de tous les déplacer.
