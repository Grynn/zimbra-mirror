Utilisation des navigateurs

Le @ClientNameLong@ est conçu pour fonctionner dans un navigateur Web. Microsoft Internet Explorer, Mozilla Firefox et Macintosh Safari sont parmi les navigateurs les plus populaires. Tous les navigateurs ne sont pas supportés, par conséquent contactez votre administrateur si vous avez des questions.

Instructions générales lors de l’utilisation du @ClientNameLong@ dans un navigateur :

  • Le bouton Retour du navigateur vous permet de revenir à la page précédente que vous consultez. Vous pouvez également utiliser le bouton Suivant.

  • Pour vous déconnecter, cliquez sur . Si vous consultez un site différent sans vous être déconnecté, il est possible que votre session reste active un certain temps. Si vous utilisez un ordinateur partagé, d’autres utilisateurs peuvent accéder à votre compte tant que la session est active.

  • N’utilisez pas le bouton Recharger (Actualiser) du navigateur. Si vous l'utilisez, le client sera téléchargé et votre session redémarrée, ce qui n'est probablement pas ce que vous aviez l’intention de faire.
