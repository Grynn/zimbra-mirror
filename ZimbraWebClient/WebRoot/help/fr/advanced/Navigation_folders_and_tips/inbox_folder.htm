Dossiers du système de messagerie

Les dossiers système sont Boîte de réception, Envoyés, Brouillons, Spams et Corbeille. Vous ne pouvez pas déplacer, renommer ni supprimer ces dossiers. Les dossiers système s'affichent toujours dans la partie supérieure de votre liste de dossiers dans le panneau d'aperçu des mails, suivis des dossiers que vous avez créés.

Si vous déplacez le curseur sur un dossier, le nombre de mails présents dans le dossier et la taille de ce dossier s'affichent.

Boîte de réception

Les nouveaux mails sont réceptionnés dans la boîte de réception. Par défaut, votre boîte de réception s'affiche immédiatement après votre connexion.

Envoyés

Une copie de tous les mails que vous avez envoyés est enregistrée dans le dossier Envoyés.

Brouillons

Vous pouvez enregistrer les mails que vous avez écrits, mais pas encore envoyés dans le dossier Brouillons. Lorsque vous ouvrez un mail dans le dossier Brouillons, il s'affiche dans la fenêtre Écrire.

Spams

Le filtrage de la plupart des mails automatiques non sollicités (appelés "spam" ou "courrier indésirable") est effectué par un filtre anti-spam avant que ces mails ne soient placés dans votre boîte de réception. Les mails supposés être du courrier indésirable peuvent être placés automatiquement dans le dossier Spams. Vous pouvez consulter ces mails, les placer dans un autre dossier ou les supprimer. Si vous ne les supprimez pas, le dossier Spam est vidé après un laps de temps défini par votre administrateur de messagerie.

Remarque :  si vous pensez que vous recevez un grand nombre de mails indésirables, contactez votre administrateur de messagerie. En rapportant les mails marqués comme du spam, vous permettez à votre administrateur de messagerie d'affiner le filtrage des spams en vigueur sur votre site.

Si vous trouvez un message qui est clairement non pertinent, vous pouvez le sélectionner et cliquer sur Spam dans la barre d'outils. Ce message a été placé dans le dossier Spam.  Il peut arriver q'un mail soit placé dans le dossier Spam alors qu'il est légitime Faites glisser le message vers un autre dossier.  

Vous pouvez vider le dossier Spams en cliquant avec le bouton droit sur le dossier dans le panneau d'aperçu et en sélectionnant Vider le dossier Spams. Notez aussi que le dossier Spam est vidé automatiquement au bout d'un certain nombre de jours défini par l'administrateur.

Corbeille

Les éléments supprimés sont placés dans le dossier Corbeille et sont conservés jusqu'à ce que vous vidiez manuellement la corbeille ou que le dossier soit automatiquement vidé. Mail est vidé dans ce dossier après une période spécifiée par l'administrateur.

Vous pouvez vider le dossier Spam en cliquant avec le bouton droit de la souris sur le dossier Aperçu et en choisissant Vider la corbeille.
