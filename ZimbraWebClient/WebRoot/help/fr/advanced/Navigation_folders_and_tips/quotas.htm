Quotas de la messagerie

Afin conserver l’usage du disque et des ressources à un niveau acceptable, votre administrateur Zimbra peut imposer des quotas pour limiter le volume de stockage utilisable pour votre messagerie.

Le bandeau d’affichage sous votre nom dans le client Web de Zimbra comporte une barre qui indique le volume de stockage utilisé de votre quota.

Si vous avez dépassé le quota, voici ce qui peut se passer :

  • Vous recevez une notification.

  • Votre interface mail risque d'afficher une notification lorsque vous vous connectez.

  • Les mails qui vous sont envoyés peuvent être renvoyés à l’expéditeur (rejetés), accompagnés d’une notification leur indiquant que votre boîte aux lettres est saturée.

Pour réduire la taille de votre messagerie, vous pouvez effecter l'une des opérations suivantes :

  • Supprimer les mails les plus anciens qui n'ont plus guère d'utilité pour vous.

  • Rechercher les mails comportant des pièces jointes volumineuses et les supprimer. Sauvegardez localement les pièces jointes si vous voulez les conserver.

  • Supprimer les carnets, pages, images et pièces jointes dont vous n'avez plus besoin.

  • Vider le dossier Corbeille.

Zimbra supporte également le nettoyage automatique des mails datant d'un certain nombre de jours. Votre administrateur de messagerie établit quand les mails doivent être purgés, leur degré d'ancienneté, et les dossiers à purger.
