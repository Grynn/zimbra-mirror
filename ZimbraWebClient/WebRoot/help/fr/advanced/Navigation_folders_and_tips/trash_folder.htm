Vider votre dossier Corbeille

Chaque application dans @ClientNameShort@ dispose d'un dossier Corbeille. Lorsque vous ouvrez la corbeille d'une application, seuls les éléments supprimés de l'application en question sont affichés. Mais quand vous sélectionnez Vider la corbeille à partir d'une application, les objets de toutes les applications sont supprimés.

Avant de vider le dossier Corbeille, vous pouvez récupérer les éléments de la Corbeille en les faisant glisser et en les déposant dans un autre dossier.

Si Restaurer objets supprimés s'affiche quand vous cliquez avec le bouton droit sur le dossier Corbeille, vous pouvez récupérer les mails que vous aviez supprimés dans la Corbeille. Il est possible de récupérer les mails supprimés de la Corbeille dans un délai de 30 jours après la suppression de la Corbeille. Passés ces 30 jours, les objets sont supprimés de manière permanente.

Attention : il est possible que la Corbeille soit vidée de manière automatique, ce qui implique que les mails qu'elle contient seront définitivement supprimés.
