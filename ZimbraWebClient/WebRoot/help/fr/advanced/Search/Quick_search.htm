Recherche rapide

La fonction de recherche de Zimbra est un outil puissant. Bien que vous puissiez apprendre à créer des requêtes de texte complexes, comme indiqué dans Description de la langue de recherche, il existe un raccourci vous permettant de saisir simplement un nom ou un mot, sans aucun mot clé. Par exemple, pour rechercher tous les mails contenant le mot "tim" n’importe où dans une ligne d’objet, dans les champs À :, De :, Cc : ou Bcc :, dans le corps du mail, ou dans les pièces jointes, vous pouvez simplement saisir tim dans la zone de recherche puis cliquez sur Recherche.

Vous pouvez saisir n'importe quelle valeur, comme par exemple un mot, un prénom, un nom de famille, un numéro de téléphone, ou le nom d'un domaine. Pour rechercher des contacts dans vos carnets d’adresses, vous devez saisir un mot complet dans le champ. Par exemple, pour rechercher des contacts en fonction de leur numéro de téléphone, vous devrez saisir le numéro de téléphone complet tel qu'il apparaît dans l'entrée du contact. Vous ne pouvez pas effectuer une recherche avec l'indicatif régional uniquement.

Avant de pouvoir utiliser la fonction de recherche rapide, assurez-vous que vous avez sélectionné le type d’élément que vous voulez rechercher. La flèche déroulante située à gauche de la zone de recherche vous permet de choisir le type d’éléments à rechercher. Vous pouvez rechercher des mails, des contacts personnels, des contacts personnels et partagés, les contacts de la liste d’adresses globale, les pages et documents dans le dossier Documents, ou de tous les types. Lorsque vous recherchez plusieurs types d’éléments, l’icône affichée dans la liste vous permet de connaître le type des éléments inclus.

Pour rechercher rapidement des contacts dans votre carnet d’adresses :

  1. Cliquez sur la flèche du menu Rechercher, , choisissez Contacts personnels.

  2. Saisissez un nom ou d’autres éléments de données du contact dans la zone Rechercher.

  3. Cliquez sur Recherche, pour lancer la recherche.

Pour effectuer une recherche rapide des contacts dans le répertoire général :

  1. Cliquez sur la flèche du menu Rechercher, , choisissez Liste d’adresses globale (GAL).

  2. Saisissez un nom ou d’autres éléments de données du contact dans la zone Rechercher.

  3. Cliquez sur Recherche, pour lancer la recherche.

Pour effectuer une recherche rapide dans les mails :

  1. Cliquez sur la flèche du menu Rechercher, , choisissez Mails.

  2. Saisissez un mot, un nom ou toutes autres données relatives au mail dans la zone Rechercher.

  3. Cliquez sur Recherche, pour lancer la recherche.
