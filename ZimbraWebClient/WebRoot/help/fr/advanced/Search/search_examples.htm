Exemples de recherches

Les différents volets de la zone de recherche avancée peuvent être utilisés pour effectuer une recherche selon différents critères. Les exemples suivants incluent les panneaux de recherche avancée que vous pouvez utiliser.

Voici quelques exemples de recherches :

  • Pour rechercher un type spécifique de pièce jointe. Un mail ayant en pièce jointe une feuille de calculs du budget de l'année prochaine. Vous ne vous rappelez plus qui est l'expéditeur ni à quelle date il a été envoyé, mais vous savez que le fichier a l'extension .XLS.

Ouvrez simplement le panneau des pièces jointes, puis choisissez Pièce jointe spécifique. Une liste de cases à cocher apparaît alors au-dessous de la case d'option. Faites défiler, puis cochez la case à côté de Microsoft Excel.

  • Pour rechercher un contact travaillant dans une entreprise spécifique. Un contact qui travaille pour la société "Rivendell".

Utilisez le menu déroulant pour choisir Rechercher un contact. Dans la zone de recherche avancée, ouvrez le panneau de recherche simple, puis saisissez "Rivendell" dans le champ de contenu.

  • Pour rechercher un type de pièce jointe. Mail comportant des pièces jointes PDF ou JPEG.

Dans le panneau Pièce jointe, sélectionnez Pièce jointe spécifique, puis cochez "Adobe PDF" et "Image JPEG".

  • Pour rechercher un élément marqué par un tag. Description des nouvelles procédures de votre groupe de travail, que vous avez marquées avec la mention "Haute priorité".

Ouvrez le panneau Tag, puis cochez la case à cocher du tag Haute priorité. Remarque : les tags sont définis par l'utilisateur. Par conséquent, cela implique que vous avez précédemment 1) créé la définition du tag, et 2) l'avez appliqué au mail.

  • Pour rechercher selon un intervalle de date. Un mail qu'une personne prétend vous avoir envoyé "au début du mois de décembre", mais que vous pensez n'avoir jamais reçu.

Pour définir une période, ouvrez deux instances du panneau Date. Dans la première, définissez la date sur is before, puis sélectionnez la date de fin. Utilisez le second panneau de Date pour spécifier "is after", puis une date de début. Vous pouvez également effectuer une recherche par expéditeur.

  • Pour rechercher un type spécifique de pièce jointe qui inclut une phrase spécifique. Un mail ayant un fichier PDF contenant la phrase "noisettes broyées."

Dans le panneau de recherche simple sous Contenu, saisissez "noisettes broyées". Dans le panneau Pièce jointe, sélectionnez Pièces jointes spécifiques, puis cochez "Adobe PDF". Notez que la recherche n'établit pas de distinction entre le contenu trouvé dans le corps d'un mail et le contenu trouvé dans une pièce jointe.
