Exemples de recherches

Les différents volets de la zone de recherche avancée peuvent être utilisés pour effectuer une recherche selon différents critères. Les exemples suivants incluent les panneaux de recherche avancée que vous pouvez utiliser.

Voici quelques exemples de recherche :

  • Pour rechercher un type spécifique de pièce jointe. Un mail ayant en pièce jointe une feuille de calculs du budget de l'année prochaine. Vous ne vous rappelez plus qui est l’expéditeur ni à quelle date il a été envoyé, mais vous savez que le nom du fichier a une extension .XLS.

Ouvrez simplement le panneau des pièces jointes puis choisissez "Pièce jointe spécifique." Une liste de cases à cocher apparaît alors au-dessous de la case d’option. Faites défiler puis cochez la case à côté de "Microsoft Excel."

  • Pour rechercher un contact travaillant dans une entreprise spécifique. Un contact qui travaille pour une société appelée "Rivendell".

Utilisez le menu déroulant pour choisir Rechercher un contact. Dans la zone de recherche avancée, ouvrez le panneau de recherche simple, puis saisissez "Rivendell" dans le champ de contenu. (Vous pouvez également effectuer cette recherche sans utiliser Search Builder.)

  • Pour rechercher une pièce jointe qui peut être d’un type ou d’un autre. Mail comportant des pièces jointes PDF ou JPEG.

Dans le panneau Pièces jointes, choisissez "Pièce jointe spécifique" puis cochez à la fois "Adobe PDF" et "Image JPEG".

  • Pour rechercher un élément marqué par un tag. Une description des nouvelles procédures de votre groupe de travail, que vous avez marquée avec la mention "Haute priorité".

Ouvrez le panneau Tags puis cochez Haute priorité. Remarque : les tags sont définis par l’utilisateur. Par conséquent, cela implique que vous avez précédemment 1) créé la définition du tag, et 2) l’avez appliqué au mail.

  • Pour rechercher selon un intervalle de date. Un mail qu'une personne prétend vous avoir envoyé "au début du mois de décembre", mais que vous pensez n'avoir jamais reçu.

Pour configurer un intervalle de date, ouvrez deux instances du panneau de Date. Dans la première, configurez la date sur "avant le" puis sélectionnez la date de fin. Utilisez le second panneau de Date pour spécifier "après le" puis une date de début. Vous pouvez également effectuer une recherche par expéditeur.

  • Pour rechercher un type spécifique de pièce jointe qui inclut une phrase spécifique. Un mail ayant un fichier PDF contenant la phrase "noisettes broyées"

Dans le panneau de recherche simple sous "Contenu", saisissez "noisettes broyées". Dans le panneau Pièces jointes, choisissez "Pièces jointes spécifiques" puis cochez "Adobe PDF". Remarque : la recherche n’établit pas de distinction entre le contenu trouvé dans le corps d’un mail et le contenu trouvé dans une pièce jointe.
