Utilisation de la fonction de recherche

La fonction de recherche vous permet de rechercher des mails, des rendez-vous de contacts, des pages de documents et des fichiers.  Vous pouvez effectuer une recherche avec des mots spécifiques, par date, heure, URL, taille, tag, si le mail a été lu ou pas, s’il comporte des pièces jointes ou des pièces jointes d’un type spécifique, et bien plus encore.

Zimbra met à votre disposition deux outils de recherche :

  • Rechercher. Il s'agit d'une fonction de recherche rapide qui lance toute recherche s'affichant actuellement dans la zone de recherche. La flèche déroulante située à gauche de la zone de recherche vous permet de choisir le type d’élément à rechercher. Vous pouvez choisir d'effectuer la recherche dans les mails, dans vos contacts, y compris dans les listes de contacts de votre société, dans les tâches, dans les rendez-vous, dans les pages et les fichiers. Il s’agit d’une aide pour les recherches rapides comme indiqué dans la section Paramètres de recherche rapide.

  • La recherche avancée ouvre un nouveau panneau et facilite le lancement de recherches plus complexes.  Vous pouvez enregistrer vos recherches avancées et les relancer ultérieurement.

Rechercher des mails

Si vous êtes familier avec les recherches de correspondances ou avec les fonctions de traitement de texte, comme par exemple Rechercher/Remplacer dans Microsoft Word, veuillez noter que la recherche de contenu dans le client Web de Zimbra est légèrement différente d’une recherche de correspondance littérale.

La syntaxe de recherche sur Zimbra fonctionne de la manière suivante :

  • Vous pouvez rechercher des phrases, mais chaque mot de cette phrase correspond littéralement à un mot entier. Les variantes orthographiques ne sont pas admises. Par exemple, si vous recherchez bananes, les mails comportant le mot banane ne sont pas considérés comme une correspondance. Vous pouvez effectuer une recherche par nom de domaine, y compris la "." (période)

  • La recherche ne tient pas compte des maj./min. ; "Sud", "sud", et "SUD" correspondent au même mot.

  • Les caractères spéciaux qui suivent ne peuvent pas être utilisés dans le texte de votre recherche. ~ ' ! @ # $ % ^ & * () _- + ? / { }[ ] ; : "

  • L’astérisque * servant de caractère de substitution après un préfixe est supporté. Une recherche du type ca* donne comme résultat les éléments avec le mot cache, carotte, etc.

  • La recherche d'un contenu permet de rechercher le corps d'un mail, et, le cas échéant, les pièces jointes (consultables sur le système) qu'il contient. Une pièce jointe consultable sur le système est un type de fichier qui peut être converti en texte affichable au format HTML. Ce type inclut les documents Microsoft Office (Word, PowerPoint, ou Excel), ainsi que les fichiers texte, mais pas les fichiers images ou audio.
