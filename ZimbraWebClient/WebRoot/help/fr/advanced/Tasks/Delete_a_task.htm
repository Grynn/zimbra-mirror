Supprimer une tâche

Vous pouvez supprimer une tâche dans une liste de tâches de plusieurs manières : Cliquez sur la liste de tâches pour afficher les tâches dans le panneau de contenu

  •  Choisissez la tâche à supprimer puis cliquez sur l’icône de la corbeille dans la barre d’outils.

  • Faites un clic droit sur la tâche puis sélectionnez Supprimer Cliquez sur Oui.

La tâche est supprimée de la liste.
