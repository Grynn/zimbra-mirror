Ajouter des comptes externes

Au lieu de vous connecter séparément à chacun de vos comptes de messagerie externes, vous pouvez consulter les mails de ces comptes directement depuis ce compte. Voir la section Accès à vos comptes externes.

Pour ajouter des comptes externes à votre compte du client Web

  1. Activez l'onglet Préférences, Comptes puis cliquez sur Ajouter un compte externe. Un nouveau nom de compte externe est ajouté à vos comptes. Maintenant, vous devez le configurer.

  2. Dans la section Paramètres du compte externe, dans le champ Noms de comptes, saisissez un nom pour identifier ce compte. Vous pouvez sélectionner le nom de ce compte à partir de la liste d'adresses De lorsque vous rédigez de nouveaux mails.

  3. Dans le champ Adresse mail saisissez l'adresse exacte du compte externe.

  4. Vous pouvez maintenant configurer les paramètres du compte. Adressez-vous à votre fournisseur de messagerie pour obtenir ces informations.

    • Cochez le type de compte, POP ou IMAP

    • Nom d'utilisateur pour ce compte. Saisissez le nom de ce compte de messagerie.

    • Serveur de messagerie. Saisissez le nom d'hôte du serveur de messagerie.

    • Mot de passe – Saisissez le mot de passe que vous utilisez pour vous connecter à ce compte externe. Pour vérifier que le mot de passe que vous avez saisi est correct, cochez Afficher mot de passe.

    • Options avancées Si votre fournisseur de messagerie utilise un port différent de celui défini par défaut, modifiez-le maintenant. Si vous utilisez une connexion sécurisée, activez Utiliser une connexion cryptée ( SSL) avec ce serveur.

  5. Dans la section Télécharger les messages dans, sélectionnez les mails à enregistrer. Par défaut, un nouveau dossier est créé et permet de recevoir les mails.  Mais si vous voulez recevoir les mails dans votre Boîte de réception, sélectionnez Boîte de réception.

  6. Activez Supprimer les mails du serveur après les avoir téléchargés, si vous voulez que le mail soit supprimé de votre compte externe lorsqu'il est envoyé dans votre compte ZWC.

  7. Dans la zone de texte sous De : Saisissez le nom qui doit s'afficher dans le champ De de vos mails sortants (par exemple, John). Il s'agit du nom convivial qui s'affiche devant votre adresse mail.

  8. Si les réponses doivent être envoyées à un nom et une adresse différents de ceux que vous avez configurés dans le champ De, cochez Répondre à. Saisissez le nom et l'adresse qui doivent recevoir les réponses pour cette identité. Par exemple, les réponses aux mails de formation de John doivent être envoyées à l'Équipe de formation à formation@exemple.com.

  9. Vous pouvez choisir une signature à utiliser pour cet avatar. Si vous n'avez pas créé de signature adéquate, vous pouvez revenir en arrière et modifier ultérieurement les données de ce compte. Si vous ne voulez pas que la signature soit appliquée de manière automatique, sélectionnez Ne pas inclure de signature.

  10. Cliquez sur Enregistrer pour sauvegarder ce compte externe.

Si vous devez créer une signature, affichez la page Signature après avoir cliqué sur Enregistrer. Voir la section Utiliser une signature.
