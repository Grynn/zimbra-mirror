Accéder à vos comptes externes

Au lieu de vous connecter séparément à chacun de vos comptes de messagerie externes, vous pouvez consulter les mails de ces comptes directement depuis le client Web de Zimbra. Lorsque vous configurez votre compte externe sur ZWC, vous pouvez spécifier l’emplacement de sauvegarde des mails et créer l’avatar que vous voulez utiliser lorsque vous répondez ou transférez un mail reçu depuis ce compte externe.

Informations nécessaires pour configurer votre compte externe

Lorsque vous configurez votre compte externe, vous devez connaître les informations suivantes concernant ce compte : il est possible que vous deviez contacter votre fournisseur de messagerie pour obtenir ces informations.

  • Type de compte. Il s’agit du type de mail que votre fournisseur utilise,  POP ou IMAP

  • Nom d’utilisateur du compte. Le nom d’utilisateur associé à ce compte de messagerie. Parfois votre nom d’utilisateur correspond à la partie de votre adresse mail précédant le caractère @. Dans d'autres cas, il s’agit de votre adresse complète.

  • Serveur de messagerie. Il s’agit du nom du serveur de votre compte externe.

  • Paramètres avancés. Votre fournisseur doit vous prévenir si votre compte n’utilise pas les ports standard IMAP (143) ou POP (110). Activez Avec ce serveur, utiliser une connexion cryptée (SSL) lorsque vous devez utiliser une connexion sécurisée au serveur externe.  Les numéros de port changeront.

Recevoir des mails sur votre compte

Lorsque vous récupérez vos mails externes, les mails et pièces jointes sont enregistrés dans le dossier que vous avez spécifié lors de la configuration des informations de votre compte.

Les mails de votre compte externe ne sont pas automatiquement reçus dans le compte de votre client Web de Zimbra.

  •  Si vous avez configuré la Boîte de réception comme dossier pour sauvegarder les mails de votre compte externe, cliquez sur Actualiser la réception du courrier dans la barre d’outils. Votre dossier Boîte de réception est mis à jour avec tous les nouveaux mails – y compris les mails de vos comptes externes.

  • Si vous avez configuré un autre dossier pour vos comptes externes, sélectionnez ce dossier puis cliquez sur le bouton Actualiser la réception du courrier dans la barre d’outils.  Les mails sont réceptionnés dans ce dossier. Cliquez sur Actualiser pour consulter les derniers mails.
