Configurer une adresse de redirection

Vous pouvez identifier une adresse pour faire suivre vos mails, et choisir de les conserver dans votre messagerie ou de les supprimer automatiquement après les avoir transférés.

Pour faire suivre votre mail

  1. Activez l’onglet Préférences puis l’onglet  Mail.

  2. Faites défiler jusqu’à la section Réception des mails puis, dans le champ Lorsqu’un mail arrive : Faire suivre une copie à, ajoutez l’adresse complète de redirection.

  3. Si vous voulez supprimer automatiquement le mail de votre compte, activez Ne pas garder de copie locale des mails.

  4. Cliquez sur Enregistrer.
