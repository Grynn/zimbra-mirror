Configurer des alertes et des rappels dans Zimbra Desktop

Vous pouvez créer des alertes pour les notifications de nouveaux mails et des rappels de rendez-vous. Pour recevoir des alertes, votre boîte aux lettres doit être ouverte.

  • Alertes Mails. Vous pouvez définir vos préférences sur la page >Préférences Mail. Voici vos options d'alertes à l'arrivée d'un mail :

    • Émettre un son. Un bip vous avertit lorsque vous recevez un mail dans votre boîte de réception.  Si vous avez coupé le son de votre ordinateur, aucun bip n'est émis. Le logiciel QuickTime ou Windows Media doit être installé pour pouvoir entendre le signal sonore.

    • Activer l'onglet Mail. Si vous n'êtes  pas en train d'utiliser l'onglet Mail, ce dernier est mis en évidence lorsque vous recevez un mail dans votre boîte de réception.

    • Afficher brièvement le nom du navigateur. Lorsque vous recevez un mail dans votre Boîte de réception, le navigateur clignote jusqu'à ce que vous ouvriez le dossier Mail.

    • Afficher un message de notification.   Lorsqu'un nouveau mail arrive, un message s'affiche. Vous devez disposer de Yahoo! BrowserPlus doit être installé.

  • Afficher les alertes de rappel sur les calendriers. Vous pouvez définir vos préférences sur la page Préférences>Calendrier.  Pour pouvoir configurer une alerte, vous devez définir l'heure du rappel de votre réunion.  Une boîte de dialogue de rappel s'affiche à l'heure indiquée avant la réunion. Si vous définissez les rappels sur Jamais, vous ne recevrez pas de rappel au sujet de la réunion, même si l'option de rappel est activée.

Le rappel ne s'affichera pas en haut de la page si vous n'êtes pas en train d'utiliser le Calendrier. Vous pouvez activer les options suivantes pour bénéficier du message de notification :

    • Émettre un son.Lorsqu'un message de rappel de rendez-vous s'affiche, vous êtes averti par un bip. Si vous avez coupé le son de votre ordinateur, aucun bip n'est émis.

    • Afficher brièvement le nom du navigateur. Lorsqu'un message de rappel de rendez-vous s'affiche, le navigateur clignote jusqu'à ce que vous fermiez le message de notification.

    En plus des alertes envoyées à votre ordinateur, vous pouvez configurer une notification par mail ou SMS depuis le rendez-vous.  Vous pouvez configurer l'adresse mail et le numéro SMS dans la page >Préférences>Notification.
