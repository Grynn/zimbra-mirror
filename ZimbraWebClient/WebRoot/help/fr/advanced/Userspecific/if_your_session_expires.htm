Si votre session a expiré

Il est possible que votre système soit configuré avec un délai de session qui vous déconnecte automatiquement si votre session est restée inactive pendant un certain laps de temps.

Votre administrateur peut également configurer une durée maximale de session (par exemple, une durée maximum de connexion, que votre session soit active ou pas).

Par conséquent, il est possible que votre écran de connexion s’affiche périodiquement, même si vous êtes déjà connecté.

Si c’est le cas, reconnectez-vous et continuez votre travail.
