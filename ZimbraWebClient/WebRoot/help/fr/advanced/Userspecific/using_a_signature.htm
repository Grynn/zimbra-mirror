Ajouter automatiquement votre signature

Vous pouvez créer différentes signatures et les attribuer à une adresse spécifique que vous avez créée. Voir la section Identités mail.

Si vous utilisez le format HTML pour formater votre signature, vous pouvez lier le nom de votre société au site Web et vous pouvez ajouter des images, telles qu'un logo, à votre signature.

Exemple d'une signature

John Smith
Vice Président du département technique
Widgets Division
Acme Corporation, Inc.   
303-555-1212 x111  
john.smith@exemple.com

 

Pour créer une signature :

  1. Ouvrez Préférences, puis sélectionnez Signatures.

  2. Cliquez sur Ajouter signature et tapez le nom de cette signature dans le champ Nom de la signature.

  3. Cliquez sur Format HTML pour personnaliser votre signature si vous souhaitez créer un style de signature.

  4. Dans la zone de texte Signature, saisissez les informations de la signature telles que vous voulez qu’elles s’affichent dans vos mails. Si vous utilisez le format HTML, vous pouvez sélectionner les polices, la taille et la couleur du texte et ajouter des liens et des images à votre signature. Les images peuvent être téléchargées ou vous pouvez les lier à l'URL d'une image. 

  5. Dans la section Utilisation des signatures, sélectionnez l’emplacement de la signature dans vos mails. Choisissez Au-dessus du mail inclus pour ajouter votre signature sous le texte que vous avez rédigé, ou auquel vous avez répondu, ou bien encore que vous avez fait suivre. Choisissez Au-dessous des mails inclus pour ajouter la signature au bas du mail.

  6. Cliquez sur Enregistrer.

  7. Pour appliquer cette signature à votre nom de compte, allez dans le dossier Comptes , sélectionnez le nom d'avatar du compte et dans le champ Signature, sélectionnez le nom de la signature dans la liste déroulante.
