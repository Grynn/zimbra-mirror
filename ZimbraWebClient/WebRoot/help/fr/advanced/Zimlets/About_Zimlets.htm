À propos des Zimlets

Les Zimlets sont des petits programmes sous forme de mécanisme qui intégrent des informations et contenus tiers avec les fonctions du client Web de Zimbra. Les Zimlets vous permettent d’interagir avec différents types de contenus dans vos mails.  

  • Cliquez sur l’URL d’un mail pour ouvrir une fenêtre de navigateur vous permettant d'afficher cette page. Déplacez-vous sur l’URL pour avoir un aperçu du site.

  • Faites un clic droit sur le numéro de téléphone pour passer un appel depuis votre ordinateur : Téléphonie sur PC - comme par exemple avec Skype ou avec un téléphone VOIP Cisco.

  • Faites un clic droit sur la date d’un mail pour consulter votre calendrier et planifier une réunion sans laisser de message.

  • Faites un clic droit sur un nom, une adresse, ou un numéro de téléphone pour mettre à jour votre carnet d’adresses.

D’autres Zimlets peuvent être disponibles dans le panneau d’aperçu.

Remarque : Si aucun Zimlet n’apparaît dans votre panneau d’aperçu, cela signifie que votre administrateur système les a désactivés.
