Affichage HTML dans les mails

Vous pouvez afficher les mails selon deux formats.

  • HTML. Les mails en HTML peuvent comporter un formatage, une numérotation, des points, un fond coloré, et des liens qui peuvent faciliter la lecture des mails. Il est possible que certains clients de messagerie n’acceptent pas les mails formatés en HTML.

  • Texte. Les mails en texte simple sont des messages qui ne contiennent pas de code de formatage HTML. Les mails envoyés au format texte simple peuvent toujours être lus par d'autres clients de messagerie. Vous pouvez consulter des mails au format texte simple même s’ils contiennent des balises HTML.

Pour toujours afficher vos mails au format HTML, depuis Préférences>Mail, sélectionnezFormat HTML (si possible).  Les mails envoyés au format HTML ont une présentation formatée.
