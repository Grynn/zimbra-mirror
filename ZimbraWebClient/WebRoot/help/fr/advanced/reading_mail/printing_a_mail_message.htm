Impression d’un mail

Pour envoyer un mail vers l’imprimante :

  1. Ouvrez la conversation, sélectionnez le mail, puis cliquez sur Imprimer.

Le mail choisi apparaît dans une fenêtre séparée, accompagné d’une boîte de dialogue d’impression standard.

  1. Dans la fenêtre de dialogue, sélectionnez une imprimante puis cliquez sur Imprimer, pour envoyer le message vers l’imprimante sélectionnée.

Vous pouvez également sélectionner une conversation complète à imprimer En revanche, vous ne pouvez pas imprimer une sélection de plusieurs mails individuels d'une conversation.

  • N’utilisez pas la fonction Imprimer Fichier du navigateur. La page ne sera pas formatée de manière correcte, et la lecture de votre message pourrait s'avérer difficile.
