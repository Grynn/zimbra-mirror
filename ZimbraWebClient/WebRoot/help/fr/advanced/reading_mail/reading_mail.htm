Lecture d'un mail

Par défaut, tous les nouveaux mails sont enregistrés dans votre Boîte de réception.  Si vous avez configuré des règles de filtre de mails, les nouveaux messages peuvent apparaître dans d'autres dossiers que votre Boîte de réception. Le nombre entre parenthèses indiqué à côté de chaque dossier correspond au nombre de mails non lus contenus dans ce dossier.

Si vous utilisez l'affichage de conversation, les mails sont regroupés en conversations.   Si vous déplacez la souris sur la conversation, une info-bulle affiche la première ligne du corps du mail le plus récent.

Pour lire un mail dans la fenêtre de conversation

  1. Double-cliquez sur la conversation contenant le mail. Le panneau de contenu change, et affiche à présent les mails individuels de la conversation. La colonne Fragment affiche la première ligne du corps du message.

  2. Pour ouvrir le mail, cliquez sur la ligne qui contient le mail que vous souhaitez lire.

Si le Panneau de lecture est activé, le corps du message s'affiche dans le Panneau de lecture. Sinon, double-cliquez sur le message. Le message s'affiche dans le Panneau de contenu.

Pour lire un message dans l'affichage Message simple

  • Si la fonction Panneau de lecture est activée, cliquez sur le message, puis lisez-le dans le Panneau de lecture.

  • Si la fonction Panneau de lecture est désactivée, double-cliquez sur le message. Le message s'affiche dans le Panneau de contenu.
