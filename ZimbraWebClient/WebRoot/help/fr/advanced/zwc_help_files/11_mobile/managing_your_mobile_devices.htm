Gestion de vos périphériques mobiles

Remarque : cette fonction est uniquement disponible pour ZCS Network Edition

Si les fonctions mobiles sont disponibles, la page Mobiles est affichée dans l'onglet Préférences. Cette page affiche les mobiles que vous avez configurés pour être synchronisés avec votre compte @ClientNameShort@.

À partir de cette page, vous pouvez synchroniser votre compte et votre mobile ou suspendre la synchronisation. Vous pouvez également supprimer les mobiles que vous n'utilisez plus.
