Importer un calendrier iCalendar

Vous pouvez importer les calendriers iCalendars (iCal) enregistrés au format .ics. iCalendars est le standard pour les échanges des données de calendrier via Internet, et .ics est le format standard des informations iCalendar.

Vous pouvez importer votre calendrier iCal dans un calendrier existant ou créer un calendrier et importer le fichier dans ce nouveau calendrier.

Pour importer un calendrier iCal :

  1. Activez l'onglet "Calendrier".

  2. Dans le panneau "Aperçu du calendrier", cliquez sur Modifier.

  3. Dans le panneau "Aperçu du calendrier", déroulez la liste des calendriers et sélectionnez celui qui recevra le fichier importé ou créez un calendrier qui recevra le fichier importé.

  4. Cliquez sur Créer calendrier. Dans le champ Importer de :  .ICS, retrouvez le fichier .ics à importer.

  5. Cliquez sur Importer.

  6. Cliquez sur Fermer.
