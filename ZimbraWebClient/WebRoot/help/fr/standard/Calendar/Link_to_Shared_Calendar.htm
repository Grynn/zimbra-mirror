Connecter à un calendrier partagé

Si vous recevez par mail l'autorisation de consulter et de partager le calendrier d'une autre personne, vous devez établir un lien avec ce calendrier (ce qui a pour effet de le faire apparaître dans votre liste de calendriers).  Pour créer ce type de lien, vous devez connaître l'adresse mail du propriétaire et le nom exact du calendrier à partager.

Pour établir un lien avec un calendrier partagé (dans les options qui suivent, le terme "calendrier" doit être compris comme "agenda") :

  1. Activez l'onglet "Calendrier" et cliquez sur Modifier.

  2. Dans la barre d'outils, sélectionnez Connecter à un calendrier partagé.

  3. Dans le panneau "Nouveau calendrier", affectez le calendrier partagé du nom de votre choix (ce nom apparaîtra dans votre liste de calendriers).

  4. Pour accéder au calendrier distant, entrez l'adresse mail du propriétaire et le nom du calendrier (nom utilisé dans le compte du propriétaire).

  5. Sélectionnez la couleur de fond qui sera affectée (dans votre compte) à ce calendrier partagé.

  6. Cliquez sur Créer calendrier.
