Créer de nouveaux calendriers

Un calendrier a été créé pour vous ; vous pouvez en créer des nouveaux pour effectuer un suivi des différents types de fonctions.  Par exemple, vous pouvez créer un calendrier pour le travail, un calendrier personnel, un calendrier pour les activités de la société, vous pouvez vous inscrire à un calendrier accessible sur Internet, et vous pouvez établir un lien vers un calendrier partagé.

Vous pouvez utiliser la Vue programmée pour consulter au moins deux de vos calendriers, l'un à côté de l'autre.

Pour créer un calendrier

  1. Sélectionnez l’onglet Calendrier.

  2. Dans le panneau Aperçu du Calendrier, cliquez sur modifier.

  3. Dans la barre d’outils, cliquez sur Nouveau Calendrier.

  4. Saisissez le nom du calendrier, choisissez une couleur pour afficher les activités du Calendrier. Si vous ne voulez pas que d'autres utilisateurs aient accès aux activités du calendrier,  cochez Exclure ce calendrier au sujet de vos disponibilités.

  5. Par défaut, l’interface est décochée. Les activités de ce calendrier s’afficheront lorsque vous ouvrez l'affichage du Calendrier.  Si vous ne voulez pas afficher ce calendrier en même temps que l’autre calendrier, décochez la case. Dans ce cas, vous devez sélectionner le calendrier et l'ouvrir afin de consulter ses activités.

  6. Cliquez sur Créer un Calendrier.

S'inscrire à un autre calendrier

  1. Sélectionnez l’onglet Calendrier .

  2. Dans le panneau Aperçu du Calendrier, cliquez sur modifier. Les informations par défaut du calendrier s’affichent.  

  3. Dans la barre d’outils, cliquez sur S’inscrire au Calendrier.  

  4. Saisissez le nom du calendrier, entrez l’URL d’un fichier iCal correct pour le calendrier distant, puis choisissez une couleur pour l’affichage des activités du calendrier.

  5. Cliquez sur Créer un Calendrier.
