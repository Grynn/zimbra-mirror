Créer des réunions récurrentes

Les réunions récurrentes sont des rendez-vous qui se répètent en vertu d'une certaine forme de planification. Vous pouvez programmer des réunions récurrentes pour qu'elles se répètent de manière quotidienne, hebdomadaire, mensuelle ou annuelle. De plus, vous pouvez personnaliser des planifications de rendez-vous récurrents.

  1. Lorsque vous créer un rendez-vous, cliquez sur Répétition dans la barre d’outils afin d'ouvrir la page de configuration des rendez-vous récurrents.

  2. Dans la section Basique, section Répétition cliquez sur le bouton radial pour sélectionner la fréquence à laquelle vous souhaitez que la réunion se répète – tous les jours, toutes les semaines, tous les mois ou tous les ans.

  3. Si votre réunion récurrente dispose d’une date de fin, choisissez dans la section Avancé, boîte Fin, de finaliser les réunions après un nombre spécifique de réunions ou à une date spécifique.

  4. La section Avancé vous permet de programmer plus de modèles spécifiques.

    • Quotidienne. Personnalisez les répétitions quotidiennes pour les configurer chaque jour ou à une autre fréquence (par exemple, tous les deux ou quatre jours).

    • Hebdomadaire. Personnalisez les répétitions hebdomadaires en choisissant s'il s'agit d'un rendez-vous récurrent hebdomadaire ou d'un rendez-vous suivant une autre configuration hebdomadaire – par exemple, tous les quinze jours.  Vous pouvez organiser la réunion un jour de semaine spécifique.

    • Mensuelle. Personnalisez les répétitions mensuelles en choisissant s'il s'agit d'un rendez-vous récurrent mensuel ou d'un rendez-vous suivant une autre configuration mensuelle – par exemple, tous les deux mois.  Un calendrier s’affiche et vous pouvez sélectionner le ou les jours pour lesquels vous voulez que la réunion soit programmée.

    • Annuelle. Personnalisez les répétitions annuelles pour choisir le mois et la date.

  1. Cliquez sur Terminé pour revenir à la page de rendez-vous.
