Lien avec un carnet d'adresses partagé

Si vous recevez par mail l'autorisation de consulter et de partager le carnet d'adresses d'une autre personne, vous devez établir un lien avec ce carnet d'adresses. Pour créer ce type de lien, vous devez connaître l'adresse mail du propriétaire et le nom exact du carnet d'adresses à partager.

Pour établir un lien avec un carnet d'adresses partagé :

  1. Sélectionnez "Carnet d'adresses" et cliquez sur Modifier.

  2. Dans la barre d'outils, cliquez sur Connecter au Carnet d'adresses.

  3. Dans le panneau "Contacts", affectez le carnet d'adresses partagé du nom de votre choix (ce nom apparaîtra dans votre liste de carnets d'adresses).

  4. Entrez l'adresse mail du propriétaire et le nom du carnet d'adresses (nom utilisé dans le compte du propriétaire).

  5. Sélectionnez la couleur de fond qui sera affectée (dans votre compte) à ce carnet d'adresses partagé.

  6. Cliquez sur Nouveau carnet d'adresses.
