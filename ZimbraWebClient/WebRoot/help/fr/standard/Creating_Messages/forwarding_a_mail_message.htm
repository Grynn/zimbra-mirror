Faire suivre un mail

Vous pouvez utiliser la fonction Faire suivre pour envoyer un mail à un autre utilisateur. Le transfert de mail inclut toutes les pièces jointes, tandis que l’utilisation de la fonction Répondre conserve la partie texte mais n’inclut pas les pièces jointes.

Le mode de transfert de votre mail peut être configuré dans Préférences de messagerie. Vous pouvez choisir d'inclure ou non le texte original du mail, l'en-tête du message À/De et les détails de l'en-tête Objet dans le mail transféré, ou vous pouvez choisir d'inclure le mail original en pièce jointe dans votre message.

Pour faire suivre un mail :

    1. Ouvrez le mail à faire suivre.

    2. Sur la barre située au dessus du champ Mail, cliquez sur Faire suivre. .  Une fenêtre de création de mail s’affiche. Le texte du mail transféré s’affiche dans le corps du message .   

    3. Entrez les adresses dans les champs De :, Cc: ou Cci : .

    4. Vous pouvez éventuellement ajouter quelques lignes du nouveau texte en haut afin que le destinataire sache pourquoi vous avez décidé de le transférer.

    5. Cliquez sur Envoyer.
