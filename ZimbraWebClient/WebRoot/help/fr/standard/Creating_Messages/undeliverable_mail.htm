Distribution de mail impossible

Il arrive qu'un mail soit "renvoyé". Cela signifie qu'il ne peut pas être distribué. Ce problème peut avoir les origines suivantes :

  • La destinataire ne correspond pas à une adresse mail correcte. Vérifiez que vous disposez de l'adresse mail correcte.

  • Le système de messagerie du destinataire refuse le mail pour une quelconque raison.

  • La boîte de réception du destinataire est saturée.

Si un mail est refusé, vous recevez un message "Distribution impossible" comportant au moins une partie de votre mail d'origine, ainsi que certains messages d'erreur générés par le système vous fournissant plus d'informations sur les raisons de ce rejet. Essayez de résoudre le problème et de renvoyer le mail.

Si le mail est renvoyé pour cause d'adresse mail incorrecte alors que vous êtes certain que l'adresse est correcte, vous devez contacter le destinataire par un autre moyen et lui demander de résoudre ce problème avec son administrateur système.

Contactez votre administrateur système si :

  • Chaque mail que vous envoyez est refusé.

  • Le mail est renvoyé avec les messages "utilisateur inconnu" ou "adresse incorrecte" pour des adresses provenant de votre liste d'adresses globale.

  • Vous commencez à recevoir des messages de renvoi pour un mail que vous n'avez jamais envoyé.
