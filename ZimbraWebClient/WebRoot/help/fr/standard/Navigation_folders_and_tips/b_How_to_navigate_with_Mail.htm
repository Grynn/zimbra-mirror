Naviguer dans l’onglet Mail

Lorsque vous ouvrez l’onglet Mail, tous vos mails sont répertoriés. Dans cet affichage, vous pouvez ouvrir et lire vos mails.

Il s’agit de l’affichage classique de la Boîte de réception quand vous ouvrez votre boîte aux lettres.

  1. Cette ligne comprend :

  • Cliquez sur Actualiser pour récupérer les nouveaux mails.

  • Cliquez sur Écrire pour rédiger un mail. Il s’affiche dans le panneau Contenu.

  • Cliquez sur Imprimer pour imprimer le message ou la conversation.

  • Cliquez sur Supprimer pour supprimer un message. Le message est déplacé vers le dossier Corbeille.

  • Utilisez la barre d’action Déplacer pour déplacer des mails vers le dossier sélectionné.

  • Utilisez la barre d’action Actions pour marquer les mails sélectionnés comme lus ou non lus, par un drapeau ou sans drapeau, les marquer avec un tag déjà créé, ou ne pas ajouter de tag.

  • Rassemblez les mails par conversation ou en tant que messages individuels.

  • Dans Afficher, choisissez d'afficher ou non un aperçu de votre message dans une sous-fenêtre de lecture.  

  1. Barre d’en-tête pour le panneau Contenu. Si vous cochez la case dans l’en-tête, tous les mails présents dans cette page seront également cochés.

  2. Dans le panneau Contenu, vos mails s’affichent comme des conversations ou de simples messages. Cochez pour sélectionner le mail ou la conversation pour lesquels vous voulez exécuter la tâche Déplacer ou Autres actions.
