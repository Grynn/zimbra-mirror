Utiliser la recherche

La fonction de recherche de @CompanyName@ est un outil puissant. Bien que vous puissiez apprendre à créer des requêtes de texte complexes, comme indiqué dans Description de la langue de recherche, il existe un raccourci vous permettant de saisir simplement un nom ou un mot, sans aucun mot clé. Par exemple, pour rechercher tous les mails contenant le mot " tim" n’importe où dans une ligne d’objet, dans les champs À : De :, Cc : ou Cci : , dans le corps du mail, ou dans les pièces jointes, vous pouvez simplement saisir tim dans la zone Rechercher, puis cliquez sur Recherche.

Vous choisissez l’emplacement où effectuer la recherche : mail, contacts, rendez-vous, tâches, pages de documents et fichiers dans Porte-documents. Vous pouvez également sélectionner pour rechercher tous les dossiers pour information.

Pour rechercher des contacts dans votre carnet d’adresses

Pour rechercher des contacts dans vos carnets d’adresses, vous devez saisir un mot complet pour que la demande soit correcte. Par exemple, pour rechercher des contacts par numéro de téléphone, vous devez saisir le numéro de téléphone complet tel qu'il apparaît dans les données du contact ; vous ne pouvez pas rechercher simplement par indicatif.

  1. Dans la ligne de recherche, dans la liste déroulante à côté du champ de recherche, sélectionnez Contacts personnels. Cette action permet de rechercher dans tous vos carnets d'adresses.

  2. Saisissez un nom complet ou toutes autres données du contact dans la zone Rechercher.

  3. Cliquez sur Recherche, pour lancer la recherche.

Pour effectuer une recherche rapide dans les autres types

Vous pouvez saisir n'importe quelle valeur, comme par exemple un mot, un prénom, un nom de famille, un numéro de téléphone, ou le nom d'un domaine. Pour plus d’options, vous pouvez utiliser l’astérisque * en tant que caractère de substitution.  

Utiliser le caractère * de substitution dans une recherche

L’astérisque * peut servir de caractère de substitution pour rechercher dans une zone de texte des mots ayant une orthographe similaire.  

Utiliser l’astérisque * en tant que caractère de substitution après un préfixe. Une recherche du type ca* donne comme résultat les éléments avec le mot cacher, carotte, etc.

  1. Dans la ligne de recherche, dans la liste déroulante à côté du champ de recherche, sélectionnez l'objet à rechercher ou sélectionnez Tous les types d'objet pour rechercher dans tous les dossiers dans votre compte.

  2. Saisissez un mot, un nom ou toutes autres données relatives au mail dans la zone Rechercher.

  3. Cliquez sur Recherche, pour lancer la recherche.
