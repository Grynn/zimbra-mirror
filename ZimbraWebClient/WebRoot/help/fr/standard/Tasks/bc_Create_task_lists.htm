Créer des listes de tâches

Pour créer une liste de tâches

  1. Activez l’onglet Tâches.

  2. Dans le titre Tâches, cliquez sur modifier.

  3. Sélectionnez Nouvelle liste de tâches.

  4. Dans la Nouvelle liste de tâches, saisissez le nom de la liste de tâches puis choisissez la couleur à utiliser pour afficher la tâche.

  5. Cliquez sur Créer liste de tâches. La liste est ajoutée aux tâches dans le panneau d’aperçu.

  6. Cliquez sur Fermer.
