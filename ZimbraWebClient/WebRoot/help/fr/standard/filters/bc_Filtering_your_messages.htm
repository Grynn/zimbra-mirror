Filtrer vos messages

Le filtre applique un ensemble de règles de correspondance au mail entrant, puis exécute une action spécifiée.

Vous pouvez filtrer vos mails entrants pour les trier dans des dossiers, pour les marquer automatiquement, pour les faire suivre, ou les supprimer. Par exemple, vous pouvez créer une règle de la manière suivante :

  • Tous les mails envoyés par votre directeur sont rangés dans le dossier "Directives commerciales".

  • Tous les mails provenant de la liste de diffusion "événements de la société" sont marqués avec le tag "Événements".

Pour créer ou modifier un règle de filtre :

  1. Activez l’onglet Préférences  .

  2. Affichez l’onglet Filtres de mails puis cliquez sur Nouveau filtre.

  3. Dans Nom du filtre, saisissez le nom de la règle.

  4. Dans la zone Si --- les conditions ci-dessous sont satisfaites, sélectionnez une condition.  Vous pouvez maintenant spécifier la condition.

Vous pouvez répéter les étapes suivantes afin de configurer plusieurs conditions et actions dans un seul filtre.

  • Choisissez dans la première liste déroulante une condition à utiliser.  

  • Choisissez un procédé de comparaison . Les options affichées dépendent de votre choix dans la liste déroulante précédente. Par exemple, "contient" ou "ne contient pas" peuvent s’afficher.

  • Saisissez un mot ou une phrase à comparer dans la zone de texte.

Cliquez sur ajouter pour ajouter des conditions. Vous pouvez continuer à ajouter des conditions ou procéder à l'étape suivante, qui consiste à ajouter une ou plusieurs actions.

  1. Dans le champ Exécuter les actions suivantes, choisissez une action dans la liste déroulante.

  • Spécifiez un dossier ou le nom d'un tag, si nécessaire. Ce que vous spécifiez dépend de l’action que vous choisissez.

Cliquez sur ajouter pour ajouter des actions. Vous pouvez continuer à ajouter des actions ou cliquer sur OK pour terminer.

  1. Cochez la case actif à côté de Nom du filtre pour activer la règle de filtre.

Le filtre s’appliquera de manière automatique dès la réception des nouveaux mails.

Pour plus d’informations sur les conditions et les actions, reportez-vous à la section Conditions et actions de filtre supportées.
