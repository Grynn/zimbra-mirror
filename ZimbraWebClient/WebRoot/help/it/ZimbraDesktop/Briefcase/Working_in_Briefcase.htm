Uso della Valigetta

La Valigetta consente di salvare file nell'account in modo da potervi accedere ogni volta che si entra nell'account da qualsiasi computer.

Puoi caricare documenti, fogli di calcolo, presentazioni, immagini e file pdf dal tuo computer o dalla tua rete di computer e salvare allegati inviati con i tuoi messaggi email nella cartella Valigetta.  È possibile creare diverse cartelle Valigetta per organizzare i file.

I file caricati nella Valigetta sono copie del file originale nel computer.  Pertanto, le modifiche apportate ai file nella Valigetta non si riflettono sul file originale.

Creazione di cartelle Valigetta

  1. Clicca sulla scheda Valigetta.

  2. Seleziona Valigetta dall'elenco Nuovo nella  barra degli strumenti.

  1. Viene visualizzata la finestra di dialogo Crea nuova cartella Valigetta. Digita un nome per la nuova cartella Valigetta.

Questo nome deve essere univoco all'interno della gerarchia delle cartelle della casella di posta. Le cartelle Valigetta non possono avere lo stesso nome di cartelle di livello superiore delle applicazioni Zimbra Mail, Agenda o Rubrica. Se ad esempio hai un'Agenda chiamata Festività, non puoi assegnare lo stesso nome a una Valigetta di livello superiore. Si può però denominare Festività una Valigetta all'interno di un'altra Valigetta.

  1. Seleziona la gerarchia di cartelle. Puoi creare una nuova Valigetta di livello superiore o inserirla in una Valigetta esistente.

  2. Clicca su OK. La nuova Valigetta viene visualizzata nel riquadro di navigazione.

Caricamento dei file

Il caricamento di un file comporta il trasferimento di file dal tuo PC all'account email. In questo modo il file è disponibile ogni volta che entri nel tuo account. La quantità di file che è possibile caricare nella Valigetta dipende dallo spazio disponibile nell'account.

Per caricare i file

  1. Apri la Valigetta e clicca su Carica file nella barra degli strumenti.

  2. Nella finestra di dialogo Carica il nuovo file nella Valigetta clicca su Sfoglia per individuare i file da caricare. Puoi selezionare più file da caricare contemporaneamente se si trovano nella stessa directory. Per aggiungere altri file, clicca su Aggiungi per individuarli.

Non puoi caricare due file con lo stesso nome nella stessa Valigetta. Puoi specificare nella finestra di dialogo Carica come trattare i file con lo stesso nome quando tenti di caricare il file. Quando modifichi questa opzione, la modifica viene salvata nella finestra di dialogo.

    • Se desideri che venga segnalato il caricamento di un file con lo stesso nome, seleziona Chiedimi. Se un file ha lo stesso nome di un file già caricato, viene visualizzata la finestra di dialogo Conflitto di caricamento. Puoi mantenere il file esistente o sostituirlo con il file caricato. Puoi anche visualizzare il nuovo file. Questa è l'impostazione predefinita.

    • Se desideri sempre sostituire il file esistente con il file caricato che ha lo stesso nome, seleziona Sostituisci con il mio file. Il nuovo file viene caricato e sostituisce quello esistente. L'operazione viene segnalata con un messaggio di notifica.

    • Se non vuoi che i file presenti nella Valigetta vengano sostituiti, seleziona Tieni il file esistente. Il file non viene caricato e appare un messaggio di notifica.

  1. Dopo aver caricato tutti i file, clicca su OK. Il file viene visualizzato nella Valigetta.  

Visualizzazione di contenuti nei riquadri di anteprima

È possibile scegliere la modalità di visualizzazione dei file delle cartelle Valigetta in un riquadro di anteprima.

  • Riquadro anteprima in basso/Riquadro anteprima a destra. Nel riquadro di anteprima vengono visualizzati i file di tipo riconosciuto. Puoi scegliere di disattivare il riquadro di anteprima.

Condivisione della Valigetta

Le cartelle Valigetta possono essere condivise con altre persone sia all'interno che all'esterno della tua organizzazione. È possibile concedere autorizzazioni ad altre persone per creare e modificare i contenuti di una Valigetta o semplicemente per visualizzare le pagine.
