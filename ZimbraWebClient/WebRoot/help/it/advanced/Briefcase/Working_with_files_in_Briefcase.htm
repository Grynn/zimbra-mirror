Uso dei file nella Valigetta

Apertura dei file nella Valigetta

I file contenuti nella Valigetta possono essere aperti da qualsiasi computer. È necessario disporre dell'applicazione software appropriata per aprirli, leggerli o aggiornarli. I formati di file comuni, come i file di testo o HTML e i file di immagini come . gif o . jpg possono essere aperti con diversi programmi.

Per aprire un file, clicca su di esso con il tasto destro del mouse e scegli Apri.

Nota: per aprire un file è inoltre possibile cliccare due volte sulla relativa icona.

  • Se il tipo è sconosciuto, viene visualizzata una finestra di dialogo in cui viene richiesto se salvare il file su disco o aprirlo con un'applicazione specifica. Se la Valigetta è condivisa con altri utenti, è possibile estrarre un file (checkout) in modo da evitare che altri utenti vi apportino modifiche nella Valigetta.

    • Clicca con il tasto destro del mouse sul file e scegli Checkout del file per le modifiche. Accanto al file verrà visualizzata l'icona di un lucchetto chiuso.

    • Se si desidera estrarre il file e apportare modifiche, nella finestra di dialogo Scarica file seleziona Checkout del file per le modifiche. Clicca su  Scarica.

    • Dopo aver apportato le modifiche, carica il file. Nella finestra di dialogo Conflitto di caricamento viene richiesto se caricare la nuova versione o mantenere la versione già presente nella Valigetta. Seleziona Nuovo e clicca su OK.

  • Se il file è un documento di Zimbra, clicca due volte per aprirlo in una pagina del browser. Non è possibile modificare un documento di Zimbra in questa visualizzazione.  

Modifica del file di un documento di Zimbra

  • Clicca con il tasto destro del mouse sul file del documento di Zimbra e scegli Modifica. Il file verrà aperto in una nuova pagina del browser.

  • Modifica la pagina e clicca su Salva e chiudi per chiudere la pagina. Se non desideri salvare i dati, chiudi la scheda del browser.

Nota: quando si clicca su Salva o su Salva e chiudi, viene creata una nuova versione del file.

Spostamento dei file

È possibile spostare i file da una Valigetta all'altra.

  1. Seleziona uno o più file da spostare

  2. Nella barra degli strumenti, clicca su .

  3. Seleziona la cartella di destinazione oppure crea una nuova cartella Valigetta.

  4. Clicca su OK.

Applicazione di tag ai file

È possibile contrassegnare i file con dei tag.

  1. Clicca sul file e nella barra degli strumenti clicca su .

  2. Seleziona il tag. I tag sono visibili solo nella Vista dettagliata.

  3. Clicca su OK.
