Creazione di nuovi Blocchi Note

  1. In Zimbra Web Client, clicca sulla scheda Documenti.

  2. Clicca con il tasto destro del mouse sulla barra Blocchi Note e scegli Nuovo Blocco Note.


     

  3. Viene visualizzata la finestra di dialogo Crea nuovo Blocco Note. Digita il nome del nuovo Blocco Note.

Questo nome deve essere univoco all'interno della gerarchia delle cartelle della casella di posta. I Blocchi Note creati nella barra Blocchi Note  non possono avere lo stesso nome delle cartelle di livello superiore delle applicazioni Zimbra Mail, Agenda o Rubrica. Se ad esempio hai un'Agenda chiamata Festività, non puoi assegnare lo stesso nome a un Blocco Note di livello superiore. Si può però denominare Festività un Blocco Note all'interno di un altro Blocco Note.

  1. Seleziona  la gerarchia di cartelle. Puoi creare un nuovo Blocco Note di livello superiore o inserirne uno in un Blocco Note esistente.

  2. Clicca su OK. Il nuovo Blocco Note viene visualizzato nel riquadro di navigazione.
