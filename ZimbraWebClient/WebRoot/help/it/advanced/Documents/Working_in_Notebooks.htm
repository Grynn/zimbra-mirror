Uso di Documenti Zimbra

L'applicazione Documenti Zimbra è un programma di collaborazione e condivisione dei documenti che consente di sviluppare, organizzare e condividere le informazioni a partire da una posizione centrale in Zimbra Web Client.  Nelle pagine di Documenti è possibile incorporare, immagini, fogli di calcolo e altri oggetti con contenuto web. È possibile caricare anche documenti esterni. Le pagine vengono visualizzate nel browser.

Documenti Zimbra è composto da Blocchi Note contenenti pagine singole organizzate in un sommario. Un Blocco Note, chiamato Blocco Note, viene creato automaticamente. Puoi crearne altri, anche all'interno di un Blocco Note esistente, per organizzare le tue informazioni.  Le pagine vengono create all'interno di un Blocco Note e ne viene quindi sviluppato il contenuto mediante un editor Rich Text o la formattazione HTML.

I Blocchi Note in Documenti possono essere condivisi con altre persone sia all'interno che all'esterno della tua organizzazione. È possibile concedere autorizzazioni ad altre persone  per creare e modificare pagine di un Blocco Note o semplicemente per visualizzarle.

Operazioni consentite in Documenti

  • Creare più Blocchi Note

  • Creare Blocchi Note all'interno di altri Blocchi Note

  • Progettare pagine mediante un editor Rich Text o HTML

  • Caricare e collegare file e immagini in pagine diverse

  • Condividere Blocchi Note, assegnando autorizzazioni di sola visualizzazione o di visualizzazione, aggiunta, modifica ed eliminazione

  • Eseguire ricerche in tutti i Blocchi Note, pagine e allegati

  • Inviare rapidamente tramite email un link al Blocco Note dal sommario
