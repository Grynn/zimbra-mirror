Uso della Valigetta

La funzione Valigetta ti consente di salvare i file su cui stai lavorando nella Valigetta e di accedervi ogni volta che entri nel tuo account, da qualsiasi computer.  

Puoi caricare documenti, fogli di calcolo, presentazioni, immagini e file pdf dal tuo computer o dalla tua rete di computer e salvare allegati inviati con i tuoi messaggi email nella Valigetta. È possibile creare diverse cartelle Valigetta per organizzare i file e condividerle con altri utenti. 

I file presenti nella Valigetta sono copie del file originale. Pertanto, le modifiche apportate nella Valigetta non si riflettono sul file originale. 

Creazione di cartelle Valigetta

  1. Clicca sulla scheda Valigetta.

  2. Clicca su Nuova Valigetta nella barra degli strumenti.

  3. Viene visualizzata la finestra di dialogo Crea nuova cartella Valigetta. Digita un nome per la nuova cartella Valigetta.

Questo nome deve essere univoco all'interno della gerarchia delle cartelle della casella di posta. Le cartelle Valigetta non possono avere lo stesso nome di cartelle di livello superiore delle applicazioni Zimbra Mail, Agenda o Rubrica. Se ad esempio hai un'Agenda chiamata Festività, non puoi assegnare lo stesso nome a una Valigetta di livello superiore. Si può però denominare Festività una Valigetta all'interno di un'altra Valigetta.

  1. Seleziona  la gerarchia di cartelle. Puoi creare una nuova Valigetta di livello superiore o inserirla in una Valigetta esistente.

  2. Clicca su OK. La nuova Valigetta viene visualizzata nel riquadro di navigazione.

Caricamento di file

Il caricamento di un file comporta il trasferimento di file dal tuo PC all'account di ZWC. In questo modo il file è disponibile ogni volta che entri nel tuo account.

Per caricare i file

  1. Apri la Valigetta e clicca su Carica file nella barra degli strumenti.

  2. Nella finestra di dialogo Carica, clicca su Sfoglia per trovare il file da caricare. Per aggiungere un altro file, clicca su Aggiungi.

Non puoi caricare due file con lo stesso nome nella stessa Valigetta. Puoi specificare nella finestra di dialogo Carica come trattare i file con lo stesso nome quando tenti di caricare il file. Quando modifichi questa opzione, la modifica viene salvata nella finestra di dialogo.

  • Se desideri che venga segnalato il caricamento di un file con lo stesso nome, seleziona Chiedimi. Se un file ha lo stesso nome di un file già caricato, viene visualizzata la finestra di dialogo Conflitto di caricamento.  Puoi mantenere il file esistente o sostituirlo con il file caricato. Puoi anche visualizzare il nuovo file. Questa azione è l'impostazione predefinita.

  • Se desideri sempre sostituire il file esistente con il file caricato che ha lo stesso nome, seleziona Sostituisci con il mio file. Il nuovo file viene caricato e sostituisce quello esistente. L'operazione viene segnalata con un messaggio di notifica.

  • Se non vuoi che i file presenti nella Valigetta vengano sostituiti, seleziona Tieni il file esistente. Il file non viene caricato e appare un messaggio di notifica.

  1. Dopo aver caricato tutti i file, clicca su OK. Il file viene visualizzato nella Valigetta.  

  2. È possibile che nel tuo account sia stata impostata una dimensione massima dei file da caricare. Se un file è troppo grande, viene visualizzato un avviso.
    Le dimensioni dei file caricati influiscono sullo spazio disponibile nel tuo account.  

Aggiunta di allegati email alla Valigetta

Gli allegati email possono essere aggiunti velocemente alla tua Valigetta.

  1. Per aggiungere l'allegato, clicca sul link aggiungi alla Valigetta che si trova nell'intestazione del messaggio accanto al link dell'allegato.

  2. Nella finestra di dialogo Aggiungi alla Valigetta, seleziona la cartella di destinazione oppure crea una nuova cartella Valigetta.

  3. Clicca su OK.

Visualizzazione dei file nella Valigetta

Sono disponibili diverse opzioni per visualizzare i tuoi file nelle cartelle Valigetta. Scegli quella che preferisci dal menu Visualizza sulla barra degli strumenti Valigetta.

  • Vista Gestione elementi. I file vengono visualizzati come miniature.

  • Vista dettagliata. I file vengono visualizzati su una riga orizzontale in cui sono indicate le dimensioni del file, la data di caricamento, il proprietario (che nel caso di un file condiviso potresti non essere tu) e la cartella in cui si trova il file.

  • Vista a colonne. I file vengono visualizzati verticalmente. In questa vista puoi cliccare su un file per visualizzarne le relative informazioni, incluse le dimensioni, la data di creazione, l'autore, la data di modifica e l'autore delle modifiche.

Uso della Valigetta

Apertura dei file nella Valigetta

I file contenuti nella tua Valigetta possono essere aperti da qualsiasi computer. È necessario disporre dell'applicazione software appropriata per aprirli, leggerli o aggiornarli. I formati di file comuni, come i file di testo o HTML e i file di immagini come .gif o .jpg, possono essere aperti con svariati programmi.

  • Per aprire un file, cliccaci sopra con il tasto destro del mouse e scegli Apri. Se il tipo è sconosciuto, viene visualizzata una finestra di dialogo in cui viene richiesto se salvare il file su disco o aprirlo con un'applicazione specifica.

Spostamento dei file

È possibile spostare i file da una Valigetta all'altra.

  1. Nella barra degli strumenti, clicca su .

  2. Seleziona la cartella di destinazione oppure crea una nuova cartella Valigetta.

  3. Clicca su OK.

Applicazione di tag  ai file

È possibile contrassegnare i file con dei tag.

  1. Clicca sul file e nella barra degli strumenti clicca su .

  2. Seleziona il tag. I tag sono visibili solo nella Vista dettagliata.

  3. Clicca su OK.

Invio di una mail con un link a un file

È possibile inviare una mail contenente un link a un file presente nella Valigetta. I destinatari devono disporre del software appropriato per aprire il file.

  1. Clicca con il tasto destro del mouse sul file selezionato.

  2. Seleziona Invia. Viene visualizzata una finestra di dialogo di conferma.

  3. Clicca su Sì.
