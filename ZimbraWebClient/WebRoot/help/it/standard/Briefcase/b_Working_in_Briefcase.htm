Uso della Valigetta

La Valigetta può essere utilizzata per condividere e gestire i documenti creati nella Valigetta stessa o documenti e immagini che vi vengono caricati. La Valigetta consente di salvare file nell'account in modo da potervi accedere ogni volta che si entra nell'account da qualsiasi computer.

Puoi caricare documenti, fogli di calcolo, presentazioni, immagini e file pdf dal tuo computer o dalla tua rete di computer e salvare allegati inviati con i tuoi messaggi email nella cartella Valigetta.  Per organizzare i tuoi file, puoi creare diverse cartelle Valigetta

I file caricati nella Valigetta sono copie del file originale nel computer.  Pertanto, le modifiche apportate ai file nella Valigetta non si riflettono sul file originale.

Creazione di cartelle Valigetta

  1. Seleziona la scheda Valigetta.

  2. Nell'intestazione Cartelle nel riquadro Panoramica, clicca sull'icona Modifica.

  3. Clicc su Nuova Valigetta.

  1. Verrà visualizzata la pagina Nuova Valigetta. Digita un nome per la nuova cartella Valigetta.

Questo nome deve essere univoco all'interno della gerarchia delle cartelle della casella di posta. Le cartelle Valigetta non possono avere lo stesso nome di cartelle di livello superiore delle applicazioni @CompanyName@ Mail, Agenda o Rubrica. Se ad esempio hai un'Agenda chiamata Festività, non puoi assegnare lo stesso nome a una Valigetta di livello superiore. Si può però denominare Festività una Valigetta all'interno di un'altra Valigetta.

  1. In Cartella principale, selezionare la gerarchia di cartelle. Puoi creare una nuova Valigetta di livello superiore nella cartella radice o inserirla in una Valigetta esistente.

  2. Clicca su Crea Valigetta. La nuova Valigetta viene visualizzata nel riquadro di navigazione.

Caricamento di file

Il caricamento di un file comporta il trasferimento di file dal tuo PC all'account email. In questo modo il file è disponibile ogni volta che entri nel tuo account. La quantità di file che è possibile caricare nella Valigetta dipende dallo spazio disponibile nell'account.

Per caricare i file

  1. Apri la Valigetta e nella barra degli strumenti clicca su Carica file.

  2. Per caricare un file, clicca su Sfoglia... per individuare il file.

  3. Seleziona il file e clicca su Apri.

  4. Carica altri allegati nello stesso modo.

  5. Dopo aver aggiunto tutti gli allegati, clicca su Fine.

Nota: È possibile che nel tuo account sia stata impostata una dimensione massima dei file da caricare. Se un file è troppo grande, viene visualizzato un avviso. I file caricati influiscono sulla quota di spazio disponibile dell'account.
