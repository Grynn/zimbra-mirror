Come allegare file al messaggio

I messaggi email possono includere allegati. Puoi allegare documenti, fogli di calcolo, immagini, presentazioni e altri tipi di file.

Per allegare un file a un messaggio:

  1. Scrivi il messaggio come descritto in Composizione di un nuovo messaggio email.

  2. Clicca su Aggiungi allegati. Viene visualizzato un nuovo pannello che ti permette di caricare fino a 10 allegati.

  3. Per caricare un file, clicca su Sfoglia... per individuare il file.

  4. Seleziona il file e clicca su Apri.

  5. Carica altri allegati nello stesso modo.

  6. Per mostrare le immagini caricate nel corpo del messaggio, seleziona la casella di controllo Mostra immagini nel corpo del messaggio.  

  7. Dopo aver aggiunto tutti gli allegati, clicca su Allega. Gli allegati vengono elencati al di sotto della riga dell'oggetto.

  8. Quando il messaggio è pronto per l'invio, clicca su Invia per inviarlo insieme agli allegati.

  • Nota – I destinatari del tuo messaggio email devono avere il software appropriato per aprire e leggere il file. I file creati in software di uso comune come Microsoft Office, potranno essere aperti anche da altri utenti sui sistemi desktop più diffusi. I formati di file comuni, come i file di testo o HTML e i file di immagini .GIF o .JPG, possono essere aperti con svariati programmi. Dipende comunque sempre dal tipo di sistema usato dall'utente.

Per eliminare un allegato, deseleziona la casella accanto all'allegato prima di inviare il messaggio email.
