スペルチェックを使用する

メールを送信する前にスペルをチェックできます。作成ツールバーの[スペルチェック]をクリックします。

送信前にすべてのメッセージで自動的にスペルチェックをするには、[プリファレンス]の[作成]ページで[メッセージを送信する前に必ずスペルチェックを行う]を有効にします。

 スペルチェック機能で認識できない単語がハイライトされます。

  1. [スペルチェック]をクリックします。正しいスペルでない単語がハイライトされます。

  2. ハイライトされている単語をクリックします。ポップアップが開き、修正候補が表示されます。

  3. 正しい単語を選択します。その単語が別の色でハイライトされます。

  4. 変更を受け入れてスペルチェックを終了するには、[編集を再開]をクリックします。

 

備考：スペルチェックを終了する前に、修正した単語を元のスペルに戻すこともできます。 ハイライトされている単語をクリックし、ポップアップの先頭に表示されている元のスペルを選択します。

スペルチェック辞書に語句を追加する

スペルチェック辞書に単語を追加できます。

  1. [スペルチェック]をクリックします。正しいスペルでない単語がハイライトされます。

  2. ハイライトされている単語をクリックします。ポップアップが開き、修正候補が表示されます。

  3. スペルの正しい単語がハイライト表示された場合、その単語をクリックして[追加]をクリックします。その単語が辞書に追加されます。

一度追加した単語は、辞書から削除できません。
