新しいカレンダーを作成する

Zimbraにはカレンダーが1つ用意されていますが、異なる種類のアクティビティを管理するために別のカレンダーを作成することもできます。 たとえば、仕事用、個人用、会社用のアクティビティのカレンダーを作成したり、インターネット上で使用できるカレンダーを購読したり、共有されているカレンダーにリンクしたりできます。

予定ビューを使用すれば、複数のカレンダーを並べて表示できます。

カレンダーを作成するには

  1. [カレンダー]タブを選択します。

  2. カレンダー概要ペインの[編集]をクリックします。

  3. ツールバーの[新しいカレンダー]をクリックします。

  4. カレンダーの名前を入力し、カレンダーアクティビティを表示するための色を選択します。カレンダーアクティビティが他のユーザーの画面に表示されないようにするには、[予定なし/予定ありの時間を報告するとき、このカレンダーを除外]チェックボックスをオンにします。 

  5. デフォルトでは、[UIでチェック済み]が有効になっています。カレンダービューを開くと、このカレンダーのアクティビティが表示されます。 このカレンダーを別のカレンダーと一緒に表示したくない場合は、このチェックボックスをオフにします。その場合、カレンダーのアクティビティを見るには、そのカレンダーを選択して開く必要があります。

  6. [カレンダーを作成]をクリックします。

別のカレンダーを購読するには

  1. [カレンダー]タブを選択します。

  2. カレンダー概要ペインの[編集]をクリックします。デフォルトのカレンダー情報が表示されます。  

  3. ツールバーの[カレンダーを購読]をクリックします。 

  4. カレンダーの名前を入力し、そのリモートカレンダーの有効なiCalファイルのURLを入力して、カレンダーアクティビティを表示するための色を選択します。

  5. [カレンダーを作成]をクリックします。
