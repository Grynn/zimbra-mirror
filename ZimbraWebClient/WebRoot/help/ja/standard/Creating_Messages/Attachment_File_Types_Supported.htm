サポートされている添付ファイルの種類

ファイルシステムに保存されているファイルのうち、[参照]ボタンを使用して見つけることができるファイルであれば、どのファイルでも添付できます。

また、お使いのワークステーションに正しいアプリケーションがインストールされており、該当するファイル拡張子がシステム管理者によってブロックされていない限り、どの添付ファイルでもZimbraウェブクライアントから開くことができます。 たとえば、Microsoft Wordを使用して作成された文書を開くには、お使いのコンピュータでMicrosoft Wordを実行できなければなりません。

ほとんどのコンピュータでは、ファイルの拡張子を見ることによってファイルの種類を区別できます。通常、拡張子はピリオドとその後の3文字で構成されています。たとえば、expenses.xlsやNew-Items.docなどのファイル名を見れば、その種類がわかります。ファイルの種類がお使いのコンピュータにインストールされているソフトウェアでサポートされているものであれば、通常そのファイルをダブルクリックするだけで、そのファイルを読み取るのに適したアプリケーションが自動的に起動します。

多くの種類のファイルは、さまざまなプラグインを使用してブラウザから直接開くことができます。通常、ページ内のファイルリンクをクリックすると、ブラウザによって正しいプラグインがあるかどうかが判断され、プラグインがない場合はそれをダウンロードしてインストールするよう指示するメッセージが表示されます。ただし、システム管理ポリシーによって、エンドユーザーが特定の種類のソフトウェアをダウンロードしてインストールすることが許可されていない場合もあります。

  • 注意：
    Windowsコンピュータでは、ファイル名に示されているファイルの種類が実際の形式に一致すると仮定されます。したがって、ファイルの内容が正しくフォーマットされていても、そのファイル名に間違ったファイル拡張子が付いていれば、コンピュータはそのファイルを認識できず、正しいアプリケーションが開きません。

    増え続けるコンピュータウイルスは、添付ファイルを通じて広がるように設計されています。そのため、一部の環境では、システム管理者によって特定の種類のファイル（特に.EXEまたは.ZIPの拡張子が付いたファイル）が添付された受信メールをブロックするように設定されている場合があります。特定の種類の添付ファイルをブロックするように設定されているメールシステムのユーザーにメールメッセージを送信しても、そのメールがブロックされたことを知らせる通知が送られてこない場合があります。

以下に、広く使用されているファイル拡張子について説明します。

  • .TXT－テキストファイル。テキストファイルは、テキスト情報のみを含んでいるファイルです。テキストファイルは、メモ帳、WordPad、Word、コマンドラインエディタ、ブラウザ、スプレッドシートプログラムなど、さまざまなテキストエディタを使って開き、読み取ることができます。

  • .HTM－HTMLファイル。通常、HTMLファイルはウェブブラウザを使用して開く特殊なテキストファイルです。

  • .ZIP－ZIP圧縮機能を使用して圧縮されたファイル。WinZip、またはZIPファイルを読み取って展開できる他のプログラムが必要です。

  • .EXE－実行可能なファイル、つまりプログラムファイル。通常、この種類のファイルをダブルクリックすると、アプリケーションが起動します。

  • .DOC－Microsoft Word文書ファイル。

  • .JPG－JPEG形式のグラフィックイメージ。JPEGファイルはブラウザで開くか、またはペイントやAdobe Photoshopなど、さまざまなグラフィックプログラムを使用して開くことができます。

  • .GIF－GIF形式のグラフィックイメージ。GIFファイルは、ブラウザまたはグラフィックプログラムで開くことができます。

  • .MPG－MPEG形式のムービークリップまたはアニメーション。通常、ブラウザプラグインまたはWindows Media Playerなどのプレーヤーで再生できます。

  • .WMV－WMV形式のムービークリップ。これはWindows専用のファイル形式です。

  • .PDF－Acrobat Reader形式の文書。通常、この形式のファイルは、ブラウザからプラグインを使用して開くことができます。
