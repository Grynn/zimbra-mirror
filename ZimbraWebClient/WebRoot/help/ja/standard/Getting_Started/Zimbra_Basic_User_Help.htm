@CollaborationProductShortName@標準ウェブクライアントの概要

標準ウェブクライアントでは、以下の機能を備えたメールアカウントを簡単に使用できます。

備考：お持ちのアカウントによっては、上記機能の一部が含まれていない場合もあります。使用可能な機能については、システム管理者に問い合わせてください。

「メールボックス内を移動する」を参照してください。

メール

    • 新しいメールメッセージを作成して送信する

    • メールメッセージを読んで返信する

    • スレッド機能を使ってメール交換を管理する、またはオプションとして従来のメッセージビューに切り替える

    • メッセージに添付ファイルを含める

    • メールメッセージを1人または複数の受信者に転送する

    • 独自のフォルダを作成してメールを整理する

    • メールの整理に使用するタグを作成する

    • フィルターを作成して受信メールを指定のフォルダに振り分ける

アドレスブック

    • 複数のアドレスブックを作成して管理する

    • 連絡先リストをインポート/エクスポートする

カレンダー

    • 複数のカレンダーを作成して管理する

    • 予約、会議、およびイベントを作成する

    • 他のカレンダープログラムからインポート、または他のカレンダープログラムにエクスポートする

タスク

    • 複数のタスクリストを作成する

    • 実行するべきタスクを作成する

    • タスクに添付ファイルを追加する

    • タスクを管理する、優先度を設定する、進行状況を追跡する

ブリーフケース

    • 任意の種類のファイルをブリーフケースのフォルダにアップロードして、アカウントにログオンするたびにそのファイルにアクセスできるようにする

    • アップロードしたファイルを整理するためのフォルダを作成する

    • 他のブリーフケースにリンクする

プリファレンス

    • メールボックス機能を管理する

    • [差出人]欄に表示されるメールアドレス、メールメッセージに署名を自動的に含めるかどうかなど、アカウント設定を指定する

    • メールの署名を設定する

    • メールフィルターを作成する

    • メールボックス内をすばやく移動するためのショートカットキーの組み合わせを表示する

 

さらに、任意のビューからメールメッセージ、個人連絡先、カレンダー、タスクリストを検索できます。
