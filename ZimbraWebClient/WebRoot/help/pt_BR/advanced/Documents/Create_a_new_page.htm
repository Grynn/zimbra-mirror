Criar um novo documento

Você pode criar um documento no Porta-arquivos e criar o layout do conteúdo usando as ferramentas de texto com formatação. xxxxÉ possível verificar a ortografia na página e usar palavras-chave para criar facilmente links para outras páginas.xxxx

Cada página inclui um rodapé que mostra o local da página, a versão que está aberta, o autor e a data da última modificação.

  1. Clique na pasta do porta-arquivos no qual você deseja trabalhar. O painel Conteúdo exibe os arquivos que já estão no porta-arquivos

  2. Na lista Novo da barra de ferramentas, selecione Documento. Uma página em branco é exibida. No cabeçalho, digite o nome para o documento.

  3. xxxxEscreva o conteúdo da página.xxxx Você pode usar o editor de texto com formatação padrão para adicionar estilos, cores e tabelas à sua página.

  1. xxxxClique em xxxxxxxxSalvarxxxxxxxx para salvar o arquivo.xxxx Agora você pode adicionar detalhes à página ou clicar em Salvar e fechar para retornar para a página Conteúdo.

xxxxObservação:xxxx Se você criou o documento no porta-arquivos errado, ele pode ser movido para o documento correto; basta selecioná-lo e arrastá-lo para a pasta correta.
