Criando link para um notebook compartilhado

Além de aceitar notificação por e-mail de que você recebeu acesso a um notebook, você também pode criar links para notebooks compartilhados caso tenha recebido acesso, usando a caixa de diálogo Link para notebook compartilhado. Por exemplo, esse pode ser o caso quando sua organização tem notebooks públicos que você deseja visualizar.

Observação: Os privilégios de acesso que você possui são definidos pelo concedente.

  1. Em Documentos, clique com o botão direito do mouse no cabeçalho Notebooks e selecione Link para o Notebook compartilhado. Uma caixa de diálogo é aberta.

  1. Para adicionar o link do notebook compartilhado, você deve saber o endereço de e-mail do proprietário e o nome do notebook. Por exemplo, Jcruz@exemplo.com.br criou um notebook chamado Projetos com um segundo notebook em Projetos chamado Projeto 1. Jcruz então compartilha Projeto 1 com você. Para poder acessar Projeto 1, digite as informações conforme é mostrado a seguir:

E-mail: jcruz@exemplo.com.br

Caminho: projetos/projeto1

Nome: O nome é completado automaticamente com o nome do e-mail e o caminho. Você pode alterar, se preferir.

Cor: Selecione uma cor na lista (opcional)

  1. Clique em OK. O notebook será adicionado à sua lista Notebooks.
