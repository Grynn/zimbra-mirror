Compartilhar sua lista de tarefas

Você pode compartilhar suas listas de tarefas com outras pessoas e pode definir permissões para a lista compartilhada conforme descrito a seguir:

  • Usuários ou grupos internos.  Selecione o tipo de privilégio que deseja conceder:

    • Visualizador. O Beneficiado pode ver o conteúdo da lista de tarefas, mas não pode fazer alterações no conteúdo da lista.

    • Gerenciador. O Beneficiado tem permissão para criar, editar e excluir tarefas na lista de tarefas.

    • Admin. O beneficiado tem permissão para criar, editar e excluir tarefas da lista de tarefas, bem como compartilhar sua lista de tarefas com outras pessoas.

    • Nenhum é uma opção para desativar temporariamente o acesso a uma lista do concedente sem revogar os privilégios de compartilhamento. O Beneficiado ainda terá a lista de tarefas na sua visualização, mas não poderá visualizar ou gerenciar as tarefas.

  • Convidados externos. Você pode criar uma senha para acessar a lista de tarefas compartilhada. Os convidados devem digitar essa senha para visualizar a lista. Eles não poderão fazer alterações em nenhuma das tarefas.

  • Público. Qualquer pessoa que saiba a URL para a lista compartilhada pode visualizá-la. Eles não podem fazer nenhuma alteração.

Para compartilhar sua Lista de tarefas

Clique com o botão direito do mouse na lista a ser compartilhada e selecione Compartilhar lista de tarefas.

Na caixa de diálogo Propriedades do compartilhamento, selecione com quem compartilhar e configure as informações adequadas.

 

Compartilhando com usuários ou grupos internos1. Selecione os Usuários ou grupos internos com os quais compartilhar

2. No campo E-mail, digite os endereços de e-mail dos beneficiários. À medida que você digita um endereço, uma lista de nomes que corresponde às letras digitadas é exibida.  Você pode selecionar o nome na lista.

3. Selecione o papel, Visualizador ou Gerenciador

4. Na seção Mensagem, selecione o tipo de mensagem de e-mail a ser enviada para notificar o beneficiário.

5. Clique em OK.

Compartilhando com convidados externos

1. Na seção Compartilhar com, selecione Convidados externos. Quando você seleciona essa opção, é exibido um campo Senha.

2. Digite os endereços de e-mail para os convidados.

3. Digite a senha que os convidados devem usar para acessar essa lista de tarefas.

4. Selecione que tipo de mensagem de notificação deve ser enviada. Essa mensagem fornece a eles as informações necessárias para acessar a lista.

5. Clique em OK.

 

Compartilhando com o público

1. Na seção Compartilhar com, selecione Público. Quando você seleciona essa opção, a caixa de diálogo propriedades do compartilhamento exibe a URL de sua lista de tarefas.

2. Anote essa URL e notifique os usuários "públicos" sobre esse endereço. Quando eles clicarem na URL, poderão ler o conteúdo da lista de tarefas, mas não poderão alterar nada.

Editando ou revogando permissão

Você pode revogar permissões a qualquer momento.

  1. Clique com o botão direito do mouse em uma lista compartilhada de tarefas.

  2. Clique em Editar propriedades. Os usuários que compartilham a pasta são listados.

  3. Clique em Editar para alterar as permissões ou
    Clique em Revogar para remover as permissões.

  4. Selecione se uma mensagem descrevendo a alteração deve ser enviada.

  5. Clique em OK.
