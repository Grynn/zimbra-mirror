xxxxConfigurando suas preferências geraisxxxx

xxxxA aba Geral inclui as seguintes configurações que você pode alterar.xxxx

  • Você está conectado no cliente avançado para a Web, mas pode definir seu tipo de cliente padrão aqui.

    • Avançado oferece um conjunto completo de recursos de colaboração na web. xxxxEsse Cliente Web funciona melhor com navegadores mais novos e conexões de Internet mais rápidas.xxxx

    • Padrão é recomendado quando as conexões de Internet forem lentas, quando usar navegadores antigos ou para uma acessibilidade facilitada. 

  • Tema é a cor de plano de fundo da interface de sua caixa de e-mails. Na aba Geral, Temas apresenta uma lista de cores de fundo diferentes que podem ser selecionadas.

  • Você pode alterar o Idioma usado no Cliente Zimbra para Web para um que seja mais adequado às suas necessidades.

  • Fuso horário padrão determina a data e a hora de mensagens que você enviou ou recebeu e a hora a ser exibida em compromissos da Agenda. Ele pode ser diferente da configuração de fuso horário do seu computador. Se o fuso horário exibido nessa aba não for correto, você poderá alterá-lo aqui. 

  • xxxxSe a opção xxxxxxxxAlterar senhaxxxxxxxx for exibida, você poderá alterar sua senha.xxxx xxxxCaso tenha dificuldades para alterar sua senha, entre em contato com o administrador para conhecer as regras de senha.xxxx xxxxEntre em contato com o administrador caso não tenha essa opção e deseje alterar sua senha.xxxx

  • Em Buscas, você pode escolher incluir automaticamente as pastas Spam e Lixeira em qualquer busca que realizar. xxxxPor padrão, essas pastas não são pesquisadas.xxxx

  • Em Idioma da busca, selecione Exibir o idioma da busca avançada na barra de ferramentas de busca para mostrar os detalhes da string na caixa de texto da busca na página do cliente. 

  • Outro

    • Para ver mais de 50 linhas ao rolar pela página de Conteúdo, modifique Número de itens para buscar ao rolar a tela.

    • Marque Exibir caixas de verificação para selecionar rapidamente itens em listas (precisa de atualização) para exibir uma caixa de verificação para cada item no painel Conteúdo. Quando ativada, use a opção para selecionar um ou mais itens para realizar a mesma ação, como apagar, mover, marcar como lido/não lido.

    • O tamanho de fonte padrão usado quando uma mensagem é impressa é de 12 pontos. Você pode selecionar outro tamanho de fonte para impressão na lista Tamanho da fonte de impressão.
