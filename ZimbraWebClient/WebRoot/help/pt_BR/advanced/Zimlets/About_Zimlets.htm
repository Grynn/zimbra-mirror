Sobre Zimlets

Zimlets são pequenos programas criados para serem um mecanismo de integração de informações e conteúdo de terceiros aos recursos do Cliente Zimbra para Web. Os Zimlets possibilitam que você interaja com diferentes tipos de conteúdo em suas mensagens de e-mail.  

  • Clique na URL em uma mensagem para iniciar uma janela de navegador para ir para aquela URL. Passe o mouse sobre a URL para obter uma visualização do site.

  • Clique com o botão direito do mouse em um número de telefone para fazer uma chamada do softphone do computador, como Skype ou um telefone VOIP Cisco.

  • Clique com o botão direito do mouse na data de uma mensagem para ver sua agenda e programar uma reunião sem precisar sair da mensagem.

  • Clique com o botão direito em um nome, endereço ou número de telefone para atualizar sua lista de endereços.

Outros Zimlets podem estar disponíveis no painel de Visão geral.

Observação: Caso não veja Zimlets listados em seu painel de Visão geral, o administrador do sistema os desabilitou.
