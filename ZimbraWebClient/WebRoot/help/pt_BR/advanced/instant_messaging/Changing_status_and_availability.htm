Alterando o status e a disponibilidade

Para usar mensagens instantâneas, seu status deve ser Online ou Disponível para bate-papo para que seus amigos possam vê-lo conectado e iniciar uma conversação.

Para definir seu status no @CompanyName@ MI:

  1. Clique no ícone na barra de MI para abrir o menu de status. Você pode selecionar qualquer um dos status a seguir:

    • Offline

    • Online

    • Disponível para bate-papo

    • Não perturbar

    • Ausente

    • Ausência estendida

    • Nova mensagem de status

  1. Selecione o status desejado para exibir aos seus amigos de MI.

Opcionalmente, você pode clicar no status, exibido ao lado de seu nome no painel Visão geral à esquerda, e selecionar outro no menu.

Criando uma nova mensagem de status

O @CompanyName@ MI dá a opção de criar suas próprias mensagens personalizadas de status.

Para criar uma nova mensagem de status:

  1. Clique no ícone na barra de MI para abrir o menu de status.

  2. Selecione Nova mensagem de status.

  3. Insira a mensagem que você deseja definir como seu novo status na caixa de diálogo. Clique em OK. Um exemplo de mensagem é "Logo eu volto". Seu status personalizado aparece no menu de status.

Para alterar o seu status para as opções padrão disponíveis, clique no menu Online novamente e selecione o status desejado.

Observação: As mensagens de status criadas por você são armazenadas para uso futuro e não podem ser apagadas. O @ClientNameShort@ salva as cinco mensagens de status mais recentes que você cria para MI.
