Criar link para a agenda compartilhada

Se você receber uma notificação por e-mail dizendo que você recebeu acesso para compartilhar a agenda de outra pessoa, será necessário criar um link para a agenda compartilhada para que ela apareça na sua lista Agenda.  Você terá de saber o endereço de e-mail completo do proprietário e o nome exato da agenda a ser compartilhada.

Para criar um link para uma agenda compartilhada:

  1. Selecione a aba Agenda e clique em editar.

  2. Na barra de ferramentas, selecione Link para a agenda compartilhada.

  3. No painel Nova agenda, dê à agenda compartilhada o nome que você quer que apareça na lista.

  4. Para acessar a agenda, insira o endereço de e-mail e o nome da agenda do proprietário.

  5. Selecione a cor dessa agenda compartilhada.

  6. Clique em Criar agenda.
