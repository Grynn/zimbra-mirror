Criando reuniões recorrentes

Reuniões recorrentes são compromissos que se repetem em algum tipo de programação. Você pode programar reuniões recorrentes para que elas se repitam diariamente, semanalmente, mensalmente ou anualmente. Além disso, é possível personalizar programações de compromisso recorrente.

  1. Ao criar um novo compromisso, clique em Repetir na barra de ferramentas para abrir a página de configuração de compromissos recorrentes.

  2. Na seção Repetir, clique no botão de opção para selecionar a freqüência com que deseja que a reunião seja repetida – todo dia, toda semana, todo mês ou todo ano.

  3. Se a reunião recorrente tiver uma data final, na seção Término previsto, selecione para terminar as reuniões após um número específico de reuniões ou até uma data específica.

  4. A seção Avançado permite que se programem padrões mais específicos.

    • Diariamente. Personalize repetições diariamente para escolher se é para todos os dias ou alguma outra seqüência, como a cada dois dias ou quatro dias.

    • Semanalmente. Personalize as repetições semanalmente para selecionar se é um compromisso recorrente semanal ou algum outro padrão semanal, como a cada duas semanas.  Você pode selecionar que dia da semana deseja a reunião.

    • Mensalmente. Personalize as repetições mensalmente para selecionar se é um compromisso recorrente mensal ou algum outro padrão mensal, como a cada dois meses.  Uma agenda é exibida e você pode selecionar que dia(s) deseja programar a reunião.

    • Anualmente. Personalize as repetições anualmente para selecionar o mês e a data.

  1. Clique em Concluído para retornar à página de compromissos.
