Criar listas de tarefas

Para criar uma lista de tarefas

  1. Selecione a aba Tarefas

  2. No cabeçalho de Tarefa, clique em editar.

  3. Selecione Nova lista de tarefas.

  4. Em Criar nova lista de tarefas, digite o nome da lista de tarefas e selecione a cor para exibir a tarefa.

  5. Clique em OK. A lista é adicionada a Tarefas no painel Visão geral.

  6. Clique em Fechar.
