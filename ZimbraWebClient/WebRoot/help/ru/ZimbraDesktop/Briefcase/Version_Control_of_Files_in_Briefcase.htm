Управление версиями файлов в Портфеле

Управление версиями файлов дает возможность управлять изменениями файлов в Портфеле. Номер версии отображается в списке документов. При просмотре истории версий можно увидеть предыдущие версии файла и автора. Вы можете восстановить предыдущую версию файла или удалить те версии, которые больше не нужны.

Примечание. Каждая версия файла учитывается в лимите учетной записи.  

Загружаемые файлы можно извлечь, тогда с одним и тем же файлом в Портфеле сможет работать только один пользователь. Эти файлы загружаются на рабочий стол пользователя. После загрузки обновленного файла он возвращается, а версия меняется.

Извлечение файла

  1. Щелкните файл правой кнопкой мыши и выберите пункт Извлечь файл.

  2. В диалоговом окне «Загрузка файлов» выберите пункт Извлечь файл для внесения изменений и нажмите кнопку Загрузить.  

Файлы загружаются на ваш компьютер.
