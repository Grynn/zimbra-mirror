Создание повторяющихся встреч

Повторяющиеся встречи — это встречи, которые повторяются в соответствии с определенным расписанием. Можно запланировать ежедневное, еженедельное, ежемесячное или ежегодное повторение повторяющихся собраний. Кроме того, можно изменять расписания повторяющихся встреч.

  1. В любом виде, кроме вида «Настройки», на панели инструментов щелкните стрелку на значке и выберите пункт Встреча.

Для получения сведений о других способах начать встречу щелкните здесь.

  1. Если отображается поле От, значит для учетной записи создано несколько образов. Выберите образ, который следует использовать для создания этой встречи.

  2. Поле Тема необходимо заполнить обязательно. Введите описание. Встреча будет отображена в ежедневнике с этим описанием.

  3. Введите имена участников. После ввода имен в календарике в области обзора отображается предложенное время, в которое свободны все участники, а также обозначены зеленым свободные места. Имена участников можно ввести одним из следующих способов.

    • В поле Участники введите адреса электронной почты, разделяя их точкой с запятой (;). Во время ввода появляются имена из списков контактов, отвечающие вводимому тексту. В Планировщике отображается существующая информация о доступности.

    • Щелкните Участники, чтобы найти адреса для добавления.

    • Откройте Планировщик, чтобы просмотреть информацию о доступности при вводе имен участников. Если расписания участников известны, информация о доступности отображается на горизонтальных панелях рядом с именами.

Необязательные участники. Если некоторые из участников собрания не являются обязательными, но их необходимо пригласить, установите флажок Показать «Необязательно» и введите адреса электронной почты в поле, или в Планировщике щелкните значок возле имени и выберите пункт Необязательный участник.  После отправки приглашения их имена будут перечислены как необязательные.

  1. Нажмите кнопку Место, чтобы найти место, созданное как учетная запись ресурса и перечисленное в каталоге вашей компании, или с помощью поля «Место» опишите место проведения собрания, указав, например, адрес, номер здания и расположение офиса.  Нажмите кнопку Отобразить ресурсы, чтобы выбрать оборудование для резервирования.

  2. Нажмите кнопку Дисплей, чтобы выбрать отображение вашей доступности в это время в Планировщике. Выберите свободен, под вопросом, занят или нет на работе.

  3. Если у вас несколько ежедневников, в меню Ежедневник выберите, в каком ежедневнике настраивается событие.

  4. Введите время начала и завершения.

Примечание. Если показан часовой пояс, то он отображает текущий часовой пояс пользователя. Обычно нет необходимости изменять данный параметр. При планировании собраний с участниками из разных часовых поясов отправляемое приглашение отражает время встречи в часовом поясе участников. Например, при создании собрания с участниками из Минска и Москвы, в приглашении будет показано минское время для участников из Минска и московское время (на один час больше) для участников из Москвы.

  1. Щелкните Повтор, чтобы выбрать шаблон повторения: ежедневно, еженедельно, ежемесячно, ежегодно или специальный. Собрание будет запланировано по повторяющемуся шаблону с указанных даты и времени начала. Специальный повтор дает возможность создать расписание более индивидуальных шаблонов.

    • Каждый день. Можно настроить ежедневные повторения, выбрав повторение каждый день или в какой-либо другой последовательности, например каждые два или четыре дня.

    • Каждую неделю. Можно настроить еженедельные повторения, выбрав повторение встречи каждую неделю или по какому-либо другому недельному шаблону, например каждые две недели.  Можно выбирать день недели, в который должно состояться собрание.

    • Каждый месяц. Можно настроить ежемесячные повторения, выбрав повторение встречи каждый месяц или по какому-либо другому месячному шаблону, например каждые два месяца.  Отображается календарь, и можно выбрать, на какой день или дни необходимо запланировать собрание.

    • Каждый год. Можно настроить ежегодные повторения, выбрав месяц и дату.

Щелкните Настроить, чтобы выбрать шаблон повторения.

  1. Чтобы изменить время отправки напоминания, щелкните поле Напоминание. Если необходимо отправить напоминание на адрес электронной почты или в виде текста SMS, установите соответствующий флажок.

  2. В поле Текст введите дополнительные сведения, которые необходимо включить в сообщение электронной почты. Чтобы вложить файлы, щелкните Вложить файлы на панели инструментов.

  3. На панели инструментов в разделе Параметры можно отключить запросы на ответы, отправку уведомления и пометить встречу как личную.

  4. Нажмите кнопку Отправить. Всем участникам будет отправлено приглашение по электронной почте.

Если приглашение следует отправить позже, нажмите кнопку Сохранить. Сохраняется локальная копия встречи, а приглашения участникам не отправляются.
