Просмотр сведений о занятости

Примечание. Только для учетных записей Zimbra.

Если во время планирования собраний есть подключение к Интернету, можно просматривать расписания участников, мест и ресурсов.  Чтобы просмотреть расписание занятости, откройте страницу создания встречи и в разделе Планировщик щелкните ссылку Показать.

После ввода участника его доступность отобразится в виде горизонтальной полосы возле имени, если есть данные о его состоянии занятости. Рабочие часы участника отображаются белым цветом, нерабочие часы — светло-серым.
