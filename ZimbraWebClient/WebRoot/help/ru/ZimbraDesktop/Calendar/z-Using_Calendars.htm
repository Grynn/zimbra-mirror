Использование ежедневников в Zimbra Desktop

В клиенте Zimbra Desktop можно вести ежедневник для каждой учётной записи.

Ежедневники учетной записи Zimbra синхронизированы с клиентом Zimbra Desktop. При добавлении нового события, изменении или удалении существующего из клиента Zimbra Desktop изменения синхронизируются с вашим ежедневником в учетной записи Zimbra.

В клиенте Zimbra Desktop можно вести ежедневник для всех других ваших учётных записей, но события ежедневника не синхронизируются с ежедневниками этих учётных записей.

При использовании других программ-ежедневников их можно сохранить в файле iCalendar (.ics) и импортировать в ваш ежедневник. Изменения, внесенные в ежедневник клиента Zimbra Desktop, не синхронизируются с ежедневником. После загрузки ежедневника .ical, ежедневник клиента Zimbra Desktop будет перезаписан, а все добавленные события утрачены.
