Изменение данных контакта

Поиск и открытие формы контакта

  1. Откройте вкладку «Адресная книга». В поле поиска отображается в:контактах.

  2. В поле «Поиск» введите критерии поиска, например имя, фамилию или имя группы.

Для поиска по адресу электронной почты необходимо ввести полный адрес. Имя или фамилия должны быть введены полностью. Например, можно ввести Петров или petrov@acme.com.

  1. В раскрывающемся списке рядом с полем поиска выберите Контакты, чтобы выполнить поиск по своим адресным книгам,

  2. В форме Изменить контакт дважды щелкните имя, чтобы открыть и изменить контакт.

  3. Внесите изменения и нажмите кнопку Сохранить, чтобы их зафиксировать.

Открытие формы контакта для внесения изменений

  1. В адресной книге щелкните правой кнопкой мыши контакт, который необходимо изменить, и выберите пункт Изменить контакт.

  2. Внесите изменения и нажмите кнопку Сохранить, чтобы их зафиксировать.

Перемещение контакта в другую адресную книгу

Контакт можно переместить одним из нижеперечисленных способов.

Из формы контакта

  1. Выберите контакт и нажмите кнопку Изменить, чтобы открыть форму контакта.

  2. В раскрывающемся меню Адресная книга выберите адресную книгу, в которую нужно переместить контакт.

  3. Нажмите кнопку Сохранить.

Из имени контакта

  1. Щелкните правой кнопкой мыши контакт, который необходимо переместить.

  2. Выберите пункт Переместить. В диалоговом окне «Переместить контакт» выберите адресную книгу.

  3. Нажмите кнопку ОК.
