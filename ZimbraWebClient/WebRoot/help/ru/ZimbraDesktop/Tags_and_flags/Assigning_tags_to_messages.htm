Назначение тегов сообщениям и другим элементам

Назначение тегов подобно установке флагов, за исключением того, что можно создать и применить несколько тегов для одного элемента для его систематизации одновременно по нескольким категориям. После создания тега, его можно применить к разговорам, сообщениям или контактам.

Примечание. Теги создаются в пределах учетной записи, с которой вы работаете. К ним не имеют доступа другие настроенные вами учетные записи.

Пометка тегом элемента с помощью перетаскивания

Перетащите тег из области обзора слева на элемент в области содержимого.

Применение тега к элементу с помощью меню

  1. Щелкните правой кнопкой мыши элемент в области обзора.

  2. Выберите пункт Пометить сообщение (или Пометить разговор или Пометить контакт).

  3. Выберите имя тега из открывшегося списка. В списке отображаются только те теги, которые не применены к выбранному элементу.

Создание нового тега и применение его к элементу

  1. Щелкните правой кнопкой мыши элемент из списка и выберите вариант меню «Тег».

  2. Выберите пункт Новый тег.

  3. Введите имя тега и нажмите кнопку OK.

Удаление тега из элемента

  1. Щелкните правой кнопкой мыши элемент из списка и выберите вариант меню «Тег».

  2. Выберите пункт Удалить тег. Если элемент содержит несколько тегов, можно удалить один или все теги.

Одновременное применение и удаление тега для нескольких элементов

  1. Удерживая нажатой клавишу Ctrl или Shift, выберите все элементы, которые необходимо пометить тегом.

  2. Щелкните правой кнопкой мыши любой из выбранных элементов и выберите вариант меню «Тег».

  3. Выберите тег для удаления или добавления.

Быстрый поиск помеченных тегами сообщений

Щелкните тег в области обзора.

Или можно выполнить поиск тега. В поле Поиск введите запрос в виде тега: <"имя тега"> и нажмите кнопку Поиск. Результатом запроса tag:Фиолетовый будут все элементы, к которым был применен тег с именем «Фиолетовый».
