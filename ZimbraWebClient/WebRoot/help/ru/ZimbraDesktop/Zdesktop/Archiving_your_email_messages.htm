Архивирование сообщений электронной почты

В ваших почтовых ящиках может быть установлен лимит размера, и чтобы избежать его превышения, необходимо удалять сообщения, когда почтовый ящик начинает заполняться. Вместо того чтобы удалять сообщения, их можно сохранять на локальном компьютере с помощью Zimbra Desktop. Для этого переместите сообщения и вложения в папку в Zimbra Desktop, которая не синхронизирована с вашей учетной записью.

Примечание. При архивировании сообщения сохраняются на вашем компьютере. В случае сбоя компьютера архивированные сообщения могут быть утрачены. Для защиты архивных данных используйте средство резервного копирования Zimbra. См. раздел Резервное копирование содержимого учетной записи.

Как архивировать сообщения для учетных записей Zimbra

Для Zimbra Web Client в Zimbra Desktop создается папка Локальные папки. Сообщения, помещенные в эту папку, сохраняются на вашем компьютере и удаляются с сервера.

 

Для архивирования сообщений Zimbra в папке «Локальные папки» следует создать вложенные папки и переместить в них сообщения. Также в папку «Локальные папки» можно перетащить личную папку сообщений целиком. При следующей синхронизации сообщения, перемещенные в папку «Локальные папки», будут удалены с сервера. При непосредственном входе в учетную запись Zimbra сообщения в папке «Локальные папки» недоступны.

Примечание. Файлы, удаляемые из папки «Локальные папки», не попадают в «Корзину», а сразу уничтожаются.

Как архивировать сообщения для других учетных записей почты

Для других настраиваемых учетных записей в Zimbra Desktop можно создать папки, которые не синхронизируются с сервером. Если сообщение перемещено в такую несинхронизированную папку, то при следующей синхронизации Zimbra Desktop с вашим почтовым ящиком сообщения удаляются с сервера. Сообщения в этих папках недоступны при непосредственном входе в учетную запись.

  1. Чтобы создать новую архивную папку, выберите в Zimbra Desktop учетную запись, щелкните правой кнопкой мыши в меню «Папки» и выберите пункт Новая папка.

  2. Присвойте папке имя и выберите расположение для нее.

  3. Нажмите кнопку ОК.

  4. Выберите созданную папку, щелкните ее правой кнопкой мыши и выберите пункт Выключить синхронизацию.  

 

Также можно выключить синхронизацию для любой личной папки.

При выключении синхронизации зеленый кружок около папки исчезает.

 

Сообщения, перемещенные в папку, не синхронизированную с учетной записью, удаляются с сервера.

Примечание. Чтобы вновь синхронизировать папки, включите синхронизацию. При следующей синхронизации Zimbra Desktop с вашей учетной записью содержимое папки будет синхронизировано.
