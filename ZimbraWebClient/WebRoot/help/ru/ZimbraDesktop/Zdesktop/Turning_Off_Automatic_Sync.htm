xxxxОтключение автоматической синхронизацииxxxx

xxxxПримечание.xxxx  Отключение синхронизации недоступно для папок Zimbra.

При настройке загрузки содержимого учетной записи в Zimbra Desktop можно задать синхронизацию всех папок учетной записи или только папки «Входящие». Если выбрана синхронизация всех папок, можно позже отключить синхронизацию личных папок.  

  1. xxxxЩелкните папку правой кнопкой мыши.xxxx

  2. В открывшемся меню выберите пункт Отключить синхронизацию.

Если отключить синхронизацию, а затем переместить сообщения из папки «Входящие» в папку, не синхронизированную с учетной записью, перемещенные сообщения будут удалены с сервера. Перемещение сообщений между несинхронизированными папками не влияет на папки учетной записи.

xxxxЧтобы вновь синхронизировать эти папки, включите синхронизацию.xxxx При следующей синхронизации Zimbra Desktop с вашей учетной записью содержимое папки будет синхронизировано.
