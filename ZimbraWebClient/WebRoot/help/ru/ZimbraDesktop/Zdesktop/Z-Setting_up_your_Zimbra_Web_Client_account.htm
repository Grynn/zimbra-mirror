Настройка учетной записи веб-клиента Zimbra Web Client.

Прежде чем вы сможете использовать клиент Zimbra Desktop, необходимо ввести настройки вашей существующей учетной записи. При первом открытии Zimbra Desktop вам будет предложено настроить учетную запись. Дополнительные учетные записи можно настроить в любое время.

Прежде чем начать настройку учетной записи, убедитесь в наличии подключения к Интернету.

  1. Откройте Zimbra Desktop и в правом верхнем углу нажмите кнопку Установка.  

  2. Нажмите кнопку ДОБАВИТЬ НОВУЮ УЧЕТНУЮ ЗАПИСЬ.  В списке «Тип учетной записи» выберите Zimbra. Откроется страница «Установка учетной записи Zimbra». Введите следующие данные.

    • Имя учетной записи. Это имя учетной записи, отображаемое в Zimbra Desktop. Все учетные записи должны иметь разные имена. Например, учетной записи ZWC можно присвоить имя Zimbra.

    • Адрес электронной почты. Введите электронный адрес учетной записи ZWC.

    • Пароль. Введите пароль для входа в учетную запись ZWC.

    • Сервер входящей почты. Введите адрес сервера Zimbra. Обычно эта запись имеет следующий формат: mail.example.com. Это сервер, на котором находится ваша учетная запись.

    • Безопасность. Если всегда необходимо использовать зашифрованное подключение, установите флажок Во время доступа к серверу используется шифрование SSL. Если вы не знаете, нужно ли использовать эту настройку, свяжитесь с системным администратором.

    • В настройках синхронизации укажите для Zimbra Desktop частоту получения данных из вашей учетной записи. По умолчанию проверка сообщений и синхронизация выполняются при получении новой почты.

При выборе параметра вручную необходимо нажать кнопку Отправить/Получить на панели инструментов Zimbra Desktop, чтобы синхронизировать учетную запись с сервером Zimbra.

Выбор слишком малого периода времени, например одной минуты, может снизить производительность компьютера.

Этот параметр можно изменить в любое время, нажав кнопку «Установка» и изменив соответствующее значение.

  1. По окончании нажмите кнопку Подтвердить и сохранить.

Zimbra Desktop проверит данные вашей учетной записи. В случае успешной проверки откроется диалоговое окно Служба создана. Если проверка окажется неудачной, снова отобразится диалоговое окно «Установка учетной записи», в котором можно просмотреть и откорректировать данные.

  1. Чтобы открыть Desktop, нажмите кнопку Запустить Desktop. Во время синхронизации учетных записей можно начать работу в Zimbra Desktop.
