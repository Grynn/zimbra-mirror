Организация вида Zimbra Desktop

При настройке клиента Desktop учетные записи отображаются отдельно в области обзора. Если необходимо, чтобы все сообщения отображались в одной учетной записи, включите общую настройку Показать раздел "Все почтовые ящики" в других учетных записях. Все сообщения каждой учетной записи из папок «Входящие», «Отправленные» и «Черновики» будут объединены в этот раздел. Также отображается количество непрочитанных элементов для всех учетных записей. Список отдельных учетных записей находится внизу раздела «Все почтовые ящики». Чтобы просмотреть элементы отдельной учетной записи, щелкните имя этой учетной записи.

 

 

При просмотре сообщений в виде «Все почтовые ящики» значок в списке сообщений определяет учетную запись, к которой относится данное сообщение.

 

Учетная запись по умолчанию

После открытия Zimbra Desktop учетные записи отображаются в области обзора в том порядке, в котором они создавались. Первая учетная запись — это учетная запись по умолчанию клиента Desktop. Она открывается при запуске Zimbra Desktop. Учетную запись по умолчанию можно менять.

  1. В Zimbra Desktop нажмите кнопку Настройка. На странице Мои учетные записи откроется список ваших учетных записей.

  2. Для изменения учетной записи по умолчанию щелкните рядом с именем учетной записи. Эта учетная запись будет перемещена в верх списка.  

 

Кроме того, что учетная запись по умолчанию открывается при отображении Zimbra Desktop, она также используется в случае настройки Zimbra Desktop в качестве почтовой программы по умолчанию. Эту настройку можно установить на странице «Настройки > Общие» в разделе «По умолчанию системы».
