xxxxНастройка учетных записей POP/IMAP, включая Microsoft Exchange IMAPxxxx

Для настройки учетной записи необходимо иметь сведения о настройках учетной записи, включая данные серверов входящей и исходящей почты. Эти сведения можно получить у системного администратора или поставщика услуг Интернета.

Для синхронизации можно указать все сообщения в почтовом ящике или только сообщения в папке «Входящие».

В разделе Настройки синхронизации укажите график синхронизации с сервером. xxxxПо умолчанию опрос выполняется каждые 15 минут.xxxx При выборе параметра вручную необходимо нажать кнопку «Отправить/Получить» на панели инструментов Zimbra Desktop, чтобы синхронизировать учетную запись с сервером Zimbra. Выбор слишком малого периода времени, например одной минуты, может снизить производительность компьютера.

Этот параметр можно изменить в любое время, нажав кнопку Установка и изменив соответствующее значение для выбранной учетной записи.

Клиент Zimbra Desktop проверит данные вашей учетной записи. В случае успешной проверки откроется диалоговое окно Служба создана. Если проверка окажется неудачной, снова отобразится диалоговое окно «Установка», в котором можно просмотреть и откорректировать введенные данные.

  1. Нажмите кнопку Запустить Desktop, чтобы открыть Zimbra Desktop. Вы можете работать в Zimbra Desktop во время синхронизации вашей учетной записи.
