Назначение Zimbra Desktop почтовой программой по умолчанию

Если щелкнуть ссылку на электронную почту в браузере или другом приложении, откроется почтовая программа, используемая на компьютере по умолчанию.  

Можно назначить Zimbra Desktop основной почтовой программой. Если щелкать ссылки на электронную почту в других программах, будет открываться окно создания сообщения для учетной записи по умолчанию в Zimbra Desktop.

Как выполнить настройку.  

  1. Выберите Общие настройки.

  2. В разделе Другое области По умолчанию системы установите флажок Почтовое приложение Zimbra Desktop используется по умолчанию.
