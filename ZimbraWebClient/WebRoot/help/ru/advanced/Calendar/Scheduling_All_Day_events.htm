Планирование событий на весь день

Можно создать событие на весь день или на несколько дней, например конференцию. Событие отображается в верхней части ежедневника в день события.

Планирование встречи на весь день

  1. В любом виде, кроме вида «Настройки», щелкните стрелку на значке и выберите пункт Встреча.

  2. Поле От отображается, если создано несколько образов для учетной записи. Выберите образ, который следует использовать для создания этой встречи.

  3. Введите тему собрания. Поле Тема необходимо заполнить обязательно. Тема становится описанием в ежедневнике.

  4. Щелкните Весь день и выберите дату начала и окончания.

Примечание. Если показан часовой пояс, то он отображает текущий часовой пояс пользователя. Обычно нет необходимости изменять данный параметр. При планировании собраний с участниками из разных часовых поясов отправляемое приглашение отражает время встречи в часовом поясе участников. Например, при создании собрания с участниками из Минска и Москвы, в приглашении будет показано минское время для участников из Минска и московское время (на один час больше) для участников из Москвы.

  1. Введите имена участников. После ввода имен в календарике в области обзора отображается предложенное время, в которое свободны все участники, а также обозначены свободные места. Имена участников можно ввести одним из следующих способов.

    • В поле Участники введите адреса электронной почты, разделяя их точкой с запятой (;). Во время ввода появляются имена из списков контактов, отвечающие вводимому тексту. В Планировщике, который откроется, можно просмотреть существующую информацию о доступности.

    • Щелкните Участники, чтобы найти адреса для добавления.

    • Щелкните Планировщик, чтобы просмотреть информацию о доступности при вводе имен участников. Если расписания участников известны, информация о доступности отображается на горизонтальных панелях рядом с именами.

Необязательные участники. Если некоторые из участников собрания не являются обязательными, но их необходимо пригласить, установите флажок Показать «Необязательно» и введите адреса электронной почты в поле, или в Планировщике щелкните значок возле имени и выберите пункт Необязательный участник.  После отправки приглашения их имена будут перечислены как необязательные.

  1. Нажмите кнопку Место, чтобы найти место, созданное как учетная запись ресурса и перечисленное в каталоге вашей компании, или с помощью поля «Место» опишите место проведения собрания, указав, например, адрес, номер здания и расположение офиса.  Нажмите кнопку Отобразить ресурсы, чтобы выбрать оборудование для резервирования.

  2. Нажмите кнопку Дисплей, чтобы выбрать отображение вашей доступности в это время в Планировщике. Выберите свободен, под вопросом, занят или нет на работе.

  3. В области Напоминание выберите время напоминания о собрании. Если следует отправить уведомление электронной почтой или SMS-сообщением, установите соответствующий флажок.  Если эти функции не настроены на странице настроек уведомления, щелкните «Настройка».

  4. В поле Текст введите дополнительные сведения, которые необходимо включить в сообщение электронной почты. Чтобы вложить файлы, щелкните Вложить файлы на панели инструментов.

  5. На панели инструментов в разделе Параметры можно отключить запросы на ответы, отправку уведомления и пометить встречу как личную.

  6. Нажмите кнопку Отправить. Всем участникам будет отправлено приглашение по электронной почте.

Если приглашение следует отправить позже, нажмите кнопку Сохранить. Сохраняется локальная копия встречи, а приглашения участникам не отправляются.
