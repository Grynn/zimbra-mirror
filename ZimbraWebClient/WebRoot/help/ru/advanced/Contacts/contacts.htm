Адресная книга

Адресная книга — это место для хранения контактных данных. По умолчанию в «Адресной книге» создаются адресные книги Контакты и Отправленные по электронной почте контакты. Можно создавать дополнительные адресные книги и предоставлять доступ к ним другим пользователям.

Можно добавлять контакты в любую из адресных книг. Для создания контакта требуется только имя. Также можно создать подробную форму контакта, включающую полное имя, несколько адресов электронной почты, рабочий, домашний и другие адреса, номера телефонов и изображение. Также можно создавать списки групповых контактов.

Адресная книга Отправленные по электронной почте контакты заполняется при отправлении электронной почты на новый адрес, который не содержится в других ваших адресных книгах. Эту функцию можно отключить в меню «Настройки» на вкладке «Адресная книга». Снимите в настройках флажок Добавлять новые контакты к отправленным по электронной почте контактам.

Администратор электронной почты может установить ограничение на максимальное количество записей, которые могут содержаться во всех ваших адресных книгах. При достижении лимита исчезнет возможность добавлять контакты. Можно удалить неиспользуемые контакты, чтобы освободить место.

При составлении сообщения электронной почты функция автозаполнения показывает список имен из адресной книги, соответствующих вводимому тексту.

Предоставление доступа к спискам контактов

Можно предоставлять доступ к любой из адресных книг. Эта функция полезна, если адресные книги используются для определенных категорий контактов и другим пользователям необходимо иметь доступ к именам. В зависимости от роли, которая назначена для получившего доступ пользователя, общий список контактов может иметь права только на просмотр или права полного доступа для изменения, добавления и удаления имен.
