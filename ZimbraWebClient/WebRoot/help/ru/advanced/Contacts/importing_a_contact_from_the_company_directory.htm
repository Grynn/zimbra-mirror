Автоматическое добавление контактов

Контакты можно быстро добавлять в адресную книгу, для этого выберите адрес в сообщении и щелкните его правой кнопкой мыши, чтобы добавить в адресную книгу. Кроме того, если в адресной книге установлен флажок Включить автоматическое добавление контактов, при отправке сообщения на новый адрес этот адрес будет автоматически добавлен в список «Отправленные по электронной почте контакты».

Добавление контактов из заголовка сообщения

Адреса электронной почты можно быстро добавлять в адресные книги. Можно добавлять электронные адреса из полей сообщения  «От:», «Кому:», «Копия:» или «Скрытая копия:».  Это можно делать в любом виде сообщения или разговора.

При наведении курсора на адрес всплывающая подсказка покажет полный адрес электронной почты, связанный с этим именем. Если имя уже содержится в адресной книге, оно также отобразится. Если имя не содержится в какой-либо из адресных книг, отобразится только адрес электронной почты.

 

Если щелкнуть адрес правой кнопкой мыши, появится меню с пунктом «Добавить в контакты». Щелкните Добавить в контакты для внесения имени в список контактов.  В форме «Контакты» отображается имя.

Можно внести дополнительные контактные данные и выбрать, в какой адресной книге сохранить информацию о контакте.

Добавление контактов при отправке сообщения

Адресная книга «Отправленные по электронной почте контакты» автоматически создается в почтовом ящике. Если в адресной книге установлен флажок «Автоматически добавлять адреса», то при отправке сообщения адрес, указанный в нем автоматически сохраняется в списке «Отправленные по электронной почте контакты», если этот адрес уже не содержится в какой-либо из адресных книг.  

Адресная книга «Отправленные по электронной почте контакты» дает возможность вести отдельную адресную книгу с нерегулярно используемыми адресами.
