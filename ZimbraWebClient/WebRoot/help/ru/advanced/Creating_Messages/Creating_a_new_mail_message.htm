Информация о составлении почтовых сообщений

Сообщения электронной почты можно создавать и отправлять сразу же, или можно создавать черновики и возвращаться к ним позже для завершения и отправки.

Первое, что нужно сделать для создания нового сообщения, это нажать кнопку на панели инструментов, чтобы открыть пустую страницу для создания сообщения. Можно также щелкнуть правой кнопкой мыши имя в поле сообщения От и выбрать пункт Написать письмо, чтобы открыть пустую страницу для создания сообщения.

В зависимости от настроек почты либо на панели инструментов «Приложение» появится вкладка «Создание сообщения» и откроется страница создания сообщения, либо откроется отдельное окно создания сообщения. На панели инструментов могут быть открыты несколько вкладок «Создание сообщения», между которыми можно переключаться.

На странице создания сообщения введите адрес электронной почты пользователя или пользователей, которым должно быть отправлено сообщение, в поле Кому.

  • Можно щелкнуть в поле Кому для поиска адреса электронной почты пользователя среди своих контактов или в глобальном списке адресов, если он доступен.

  • Чтобы ввести адреса в поле «Скрытая копия», нажмите кнопку Показать СК рядом с полем Копия: в заголовке сообщения.

Если настроено несколько образов, на странице создания сообщения отображается поле От. Выберите образ, который следует использовать в качестве адреса отправителя.

Сообщение можно создавать в формате HTML или в формате обычного текста. Формат по умолчанию настраивается на странице Настройки > Общие. Для быстрого изменения формата только для текущего сообщения нажмите кнопку Параметры на панели инструментов создания сообщения и выберите «HTML» или «Обычный текст».

  • Формат HTML — позволяет использовать в сообщении различные виды шрифтов, создавать таблицы, добавлять цвет и вставлять гиперссылки.

  • Формат обычного текста — используется текст без стилей и форматирования. Этот тип сообщений может быть прочитан на любом компьютере.

При работе в учетных записях @CompanyName@ можно запросить подтверждение прочтения письма. Нажмите кнопку Параметры на панели инструментов создания сообщения и выберите Запросить подтверждение прочтения.

К сообщениям можно присоединять файлы, в том числе изображения, документы, электронные таблицы и аудиофайлы.
