Использование проверки правописания

  1. Чтобы проверить правописание в сообщении, нажмите кнопку Проверить правописание на панели инструментов.

Будут выделены слова, неизвестные программе проверки правописания.

  1. Щелкните выделенное слово. Во всплывающем окне отобразятся варианты исправления.

  2. Выберите правильное слово. Слово будет выделено другим цветом.

  3. Чтобы принять изменения и закрыть программу проверки правописания, нажмите кнопку Продолжить редактирование.

 

  • Перед закрытием программы проверки правописания можно вернуть исходное написание исправленного слова.  Щелкните выделенное слово и выберите его исходное написание вверху всплывающего окна.
