Автоматическое заполнение адреса

Функция автозаполнения предлагает варианты имен при вводе текста в поле Кому:, Копия: и Скрытая копия: нового сообщения. Отображается список возможных адресов из списка контактов пользователя, начало которых соответствует введенному тексту.

По мере ввода текста в поле Кому список соответствующих адресов становится меньше, так как уменьшается количество совпадений. Если совпадений нет, список исчезает.

Пример

Например, представим, что необходимо отправить сообщение электронной почты Дмитрию Богатыреву, у которого следующий адрес электронной почты: dbogatyrev. После ввода первой буквы «д» в поле адреса появится список, содержащий все возможные соответствия, с выделенными частями, которые соответствуют. В данном случае, может отобразиться следующее:

Дмитрий Попов <dpopov@kontora.ru>

Дмитрий Богатырев <dbogatyr@kontora.ru>

Дуня Дусечкина <дуня878@xyz.com>

Чем больше символов введено в поле Кому, тем меньше становится список соответствий.

Соответствие

Функция автозаполнения сопоставляет вводимый текст со следующими элементами контакта.

  •  Имя

  •  Фамилия

  • Адрес электронной почты

Сопоставление осуществляется с началом поля. Тексту «vova» будет соответствовать «vova ivanov», но не «vladimir ivanov».

Использование автозаполнения

Если текст соответствует одному или нескольким контактам, соответствия отображаются в списке под полем, в котором осуществляется ввод. По умолчанию выделяется первое соответствие.

Несколько кнопок имеют особые функции. Мышь также можно использовать для выбора одного из соответствий.

  • Нажатие клавиш с запятой, точкой с запятой, Enter и Tab инициирует заполнение. Введенный текст будет заменен одним из соответствий, выбранным в данный момент.

  • Чтобы скрыть список, нажмите клавишу ESC.

  • Выбрать другое соответствие из списка можно с помощью клавиш «Стрелка вверх» и «Стрелка вниз». Перемещение курсора над списком также позволяет изменить выбор.

  • Щелкнув мышью одно из соответствий, можно выбрать его для заполнения.

Заполнение нажатием одной клавиши (например, клавиши с точкой с запятой) можно использовать для быстрого ввода адресов. Если известны несколько знаков имени определенного контакта, который будет первым в списке соответствий, можно ввести эти знаки, а затем нажать клавишу с точкой с запятой, и появится адрес данного контакта. Например, если известно, что ввод букв «db» приведет к тому, что первым соответствием будет Дмитрий Богатырев, можно ввести «db;» и полный адрес «Дмитрий Богатырев <dbogatyr@kontora.ru>» появится в поле адреса.

Автозаполнение доступно в других приложениях (например, в ежедневнике для добавления участников встречи) и работает так же.
