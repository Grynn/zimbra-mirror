Использование кода HTML для форматирования страниц «Документы»

Если вы знакомы с кодом HTML, то можете воспользоваться тегами HTML для создания содержимого.  Можно использовать все элементы HTML.  Чтобы создать страницы в формате HTML, откройте на панели инструментов меню Формат и выберите пункт Код HTML.

Во время редактирования страницы можно переключаться между форматированным текстом и кодом HTML. В правом верхнем углу откройте меню Формат, чтобы выбрать редактор.
