Работа в портфеле

Функция «Портфель» позволяет сохранять используемые файлы в портфеле. Доступ к файлам можно получить с любого компьютера, войдя в почтовый ящик.  

В портфель можно загружать документы, электронные таблицы, презентации, изображения, pdf-файлы с компьютера или из сети, кроме того, в нем можно сохранять вложения почтовых сообщений.  В портфеле можно создать разные папки для систематизации файлов, а также сделать их общедоступными.

В портфеле хранятся копии исходных файлов.  Изменения, вносимые в файлы в портфеле, не сохраняются в исходных файлах.

Создание папок портфеля

  1. Откройте вкладку «Портфель».

  2. Щелкните ссылку Новый портфель на панели задач.

  3. Откроется диалоговое окно Создать новый портфель. Введите имя новой папки портфеля.

Это имя должно быть уникальным в иерархии папок вашего почтового ящика. Папки портфеля не должны иметь такие же имена, как у папок верхнего уровня в почте, ежедневнике или адресной книге Zimbra. Например, если у вас есть ежедневник с именем «Праздники», то давать имя «Праздники» портфелю верхнего уровня нельзя. Имя «Праздники» можно дать портфелю в другом портфеле.

  1. Выберите  иерархию папки. Можно создать новый портфель верхнего уровня или поместить его в уже существующий портфель.

  2. Нажмите кнопку ОК. Новый портфель появится в области обзора.

Загрузка файлов

В процессе загрузки файлы передаются с компьютера в почтовый ящик ZWC. Это позволяет получить доступ к файлам в любое время, для этого нужно только войти в почтовый ящик.

Загрузка файлов

  1. Откройте портфель и щелкните значок «Загрузить» на панели задач.

  2. В диалоговом окне «Загрузить» нажмите кнопку «Обзор», чтобы найти файл для загрузки. Чтобы добавить еще один файл, нажмите кнопку Добавить.

Загружать файлы с одинаковыми именами в один и тот же портфель нельзя. В диалоговом окне «Загрузить» можно указать, как сохранять файлы с одинаковыми именами при загрузке в портфель. Внесенные в этот параметр изменения сохраняются в диалоговом окне.

  • Чтобы получать уведомления о загрузке файлов с одинаковыми именами, выберите пункт Спрашивать меня. Если файл с именем, аналогичным имени загружаемого файла уже существует, откроется диалоговое окно Конфликт загрузки.  Можно сохранить уже существующий файл или заменить его загружаемым. Можно также просмотреть новый файл. Это файл по умолчанию.

  • Если нужно, чтобы загружаемый файл всегда заменял уже существующий файл с таким же именем, выберите пункт Заменить моим файлом. Загруженный файл будет сохранен вместо существующего. Уведомление об этом не поступает.

  • Если файлы в портфеле должны быть сохранены, выберите пункт Оставить существующий файл. Файл не загружается, уведомление об этом не поступает.

  1. Если нужно загрузить все файлы, нажмите кнопку ОК. Файл отображается в портфеле.  

  2. В почтовом ящике можно установить предельный размер для загружаемых файлов. Если файл слишком большой, появится предупреждение.
    Загружаемые файлы влияют на лимит почтового ящика.  

Добавление почтовых вложений в портфель

В портфель можно быстро добавлять почтовые вложения.

  1. Чтобы добавить вложение, щелкните ссылку Добавить в портфель, которая находится в заголовке сообщения рядом со ссылкой на вложение.

  2. В диалоговом окне Добавить в портфель выберите папку назначения или создайте новую папку.

  3. Нажмите кнопку ОК.

Просмотр файлов в портфеле

Можно выбрать режим просмотра файлов в папках портфеля. В меню Вид на панели задач портфеля выберите один из трех вариантов.

  • Вид обозревателя. Файлы отображаются в виде эскизов.

  • Подробный вид. Файлы отображаются по горизонтали с указанием размера файла, даты загрузки, владельца, лиц, имеющих право доступа, и папки, в которой он хранится.

  • Вид браузера «Столбцы». Файлы отображаются по вертикали. На этом виде можно щелкнуть файл и посмотреть информацию о нем, включая размер, дату создания, создателя, дату изменения и изменяющего.

Работа в портфеле

Открытие файлов в портфеле

Файлы в портфеле можно открыть с любого компьютера. Для открытия. чтения и обновления файла необходимо соответствующее приложение. Такие распространенные форматы файлов, как текстовый, HTML, изображения, например .gif или .jpg, можно открывать в различных программах.

  • Чтобы открыть файл, щелкните его правой кнопкой мыши и выберите пункт Открыть. Если тип файла неизвестен, откроется диалоговое окно с вопросом, что делать с файлом: сохранить на диске или открыть в каком-либо приложении.

Перемещение файлов

Файлы можно перемещать из одного портфеля в другой.

  1. Щелкните значок на панели инструментов.

  2. Выберите папку назначения или создайте новую папку в портфеле.

  3. Нажмите кнопку ОК.

Пометка файлов

Файлы можно помечать.

  1. Выделите файл на панели инструментов и щелкните значок .

  2. Выберите тег. Теги видны только на виде «Подробный».

  3. Нажмите кнопку ОК.

Отправка сообщения со ссылкой на файл

Можно отправить сообщение со ссылкой на файл в портфеле. Чтобы открыть файл, получатели должны иметь соответствующее приложение.

  1. Щелкните выбранный файл правой кнопкой мыши.

  2. Выберите пункт Отправить. Появится диалоговое окно подтверждения.

  3. Нажмите кнопку Да.
