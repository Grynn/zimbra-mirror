Помощник @CompanyName@

Помощник @CompanyName@ — это усовершенствованная пользовательская функция, обеспечивающая быстрый способ взаимодействия с веб-клиентом с помощью клавиатуры для выполнения следующих задач.

  • Создание новой встречи

  • Переход к конкретной дате в ежедневнике

  • Создание новых контактов

  • Отправка нового сообщения без смены приложения, используемого в данный момент

Помощник @CompanyName@ можно открыть в любом виде, за исключением вкладки «Настройки». Помощник @CompanyName@ открывается с помощью клавиши ` (обратная кавычка). Эта клавиша находится под клавишей Escape ( Esc).

В диалоговом окне помощника @CompanyName@ предлагается подробная справка. Ниже предлагаются основные действия по использованию помощника @CompanyName@.

  1. В веб-клиенте @ClientNameLong@ нажмите горячую клавишу (обратную кавычку или тильду). Откроется диалоговое окно помощника @CompanyName@.

  2. Введите одну из следующих команд

    • Чтобы создать встречу, введите встреча. Затем можно ввести тему встречи в кавычках, время, дату и день недели и любые заметки о встрече. Нажмите кнопку ОК.  Примечание. Если нужно воспользоваться дополнительными функциями для встречи, щелкните Подробнее, чтобы перейти на страницу Встреча в ежедневнике.

    • Чтобы перейти к конкретной дате в ежедневнике, введите ежедневник и дату или день недели. Нажмите кнопку ОК.  Примечание. После указания даты можно выбрать вид ежедневника: день, рабочая неделя, неделя или месяц. Ежедневник откроется на этом виде.

    • Чтобы создать новый контакт, введите имя, электронный адрес контакта и др. Нажмите кнопку OK.  Примечание. Чтобы открыть подробную форму контакта, щелкните «Подробнее». Здесь можно указать, в какую адресную книгу добавить данный контакт.

    • Чтобы отправить почту, не открывая окно для написания письма, введите почта. Введите тему в кавычках; адресата: имя@адрес и наберите тело письма. Нажмите кнопку ОК.
