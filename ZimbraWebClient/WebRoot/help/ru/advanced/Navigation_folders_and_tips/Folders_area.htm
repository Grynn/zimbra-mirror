Просмотр области обзора

В левой части интерфейса веб-клиента Zimbra находятся пользовательские папки, профили поиска, теги и модули Zimlet (для расширенного веб-клиента).

 

Папки

  • Входящие — куда по умолчанию помещается новая почта.

  • Отправленные — содержит копии отправленных сообщений.

  • Черновики — содержит еще не отправленные сообщения.

  • Спам — содержит сообщения, помеченные пользователем или системой как спам.

  • Корзина содержит элемент

  • Локальные папки, который был удален пользователем. Эта папка доступна только для учетных записей ZWC.

  • Папки, создаваемые пользователем. Если ваш почтовый ящик мигрировал из другой почтовой системы, например Microsoft® Outlook, то возможно вы увидите дополнительные папки, перекочевавшие из предыдущей системы.

Счетчик непрочитанных сообщений. Число в скобках рядом с каждой папкой указывает количество непрочитанных сообщений в папке. В папках Теги также отображается количество непрочитанных сообщений. Элемент с непрочитанными сообщениями отображается жирным шрифтом.

Чтобы увидеть количество сообщений в папке и ее общий размер, наведите курсор на папку.

Профили поиска

Эта область содержит профили поиска, созданные пользователем и сохраненные для будущего использования.

Теги

В этой области показаны теги, созданные пользователем. Щелкните тег для быстрого просмотра всех помеченных тегом сообщений. Дополнительную информацию см. в разделе Использование тегов для систематизации почтовых сообщений.

Zimlet

Модули Zimlet создаются для интеграции информации и содержимого третьей стороны в функции @ClientNameLong@. Модули Zimlet позволяют работать с различными типами содержимого в почтовых сообщениях. В зависимости от настроек модули Zimlet могут быть недоступными в области обзора.
