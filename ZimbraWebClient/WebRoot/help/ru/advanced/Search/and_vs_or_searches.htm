Поиск типа AND и OR

Для поиска с использованием нескольких критериев, можно найти элементы соответствующие одному или всем указанным критериям. Можно выполнить оба типа поиска с использованием функции расширенного поиска.

Для всех областей поиска применимы приведенные ниже правила.

  • В области обычного поиска можно создать поисковый запрос с критерием «AND». Если открыть две области обычного поиска, можно создать поисковый запрос с критерием «OR» между ними.

  • Поиск сообщений, соответствующих любым указанным критериям, называется поиском с условием OR (ИЛИ), потому что, если сообщение содержит X или Y, оно считается соответствующим условиям поиска. Для панелей расширенного поиска с несколькими установленными флажками выбор нескольких вариантов на одной панели создает поиск для этих элементов с условием OR.

  • Поиск сообщений, которые содержат X и Y, называется поиском с условием AND (И), потому что сообщение должно соответствовать всем указанным критериям, чтобы считаться соответствием. Для областей расширенного поиска открытие нескольких экземпляров одной и той же области и установка разных флажков на каждой из них делает необходимым определение критериев поиска как поиска с условием AND.

Примечание. В запросе появится только «OR». Если задан параметр показывать запрос поиска на панели поиска, текстовое поле поиска будет обновляться и отображать полученный запрос. В профиле поиска типа AND слово AND не отображается.
