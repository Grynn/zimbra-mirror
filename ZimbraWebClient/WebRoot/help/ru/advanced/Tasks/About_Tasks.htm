Использование списков задач

Функция «Задачи» позволяет создавать список задач и управлять их выполнением. Задачи можно добавлять в список задач по умолчанию, можно создавать другие списки задач для систематизации списков задач по более конкретным видам деятельности, например, по работе или личным проектам. Можно создать новый список «Задачи» и синхронизировать его с задачами из удаленного списка задач iCal.

Для простой задачи нужно лишь создать задачу в области содержимого. Если задачей нужно управлять, можно добавить дополнительные данные, определить дату начала и окончания, установить приоритет задачи — высокий, обычный, низкий — и проследить выполнение задачи.

На иллюстрациях ниже показано создание трех списков задач. Список задач, отображаемый на панели содержимого, выделяется синим цветом. В области содержимого отображаются предстоящие, сегодняшние и прошедшие задачи, а также статус, процент выполнения и дата, когда задача должна быть выполнена.

 

Отображение задач в области просмотра

Область просмотра используется для отображения содержимого задачи без ее открытия. Область просмотра может отображаться в нижней или правой части области содержимого. Чтобы выбрать способ отображения области просмотра, щелкните Вид на панели инструментов «Задачи».

Когда область просмотра отображается внизу окна, вид списка задач остается без изменений. Когда область просмотра находится справа, в виде списка задач отсутствует статус задачи, а процент выполнения отображается как индикатор хода выполнения.  

Сортировка задач

Список задач можно сортировать по теме, статусу, проценту выполнения и дате выполнения.

В виде «Задача» выберите список задач и на панели инструментов области просмотра щелкните заголовок для сортировки.

Использование Помощника@CompanyName@ для быстрого создания задачи

@CompanyName@ можно использовать для быстрого добавления задачи во время обработки сообщения электронной почты, собрания в календаре или просмотра контакта. После создания задачи вся доступная информация добавляется в область текста задачи.

Печать задач

Для печати можно выбрать все или несколько задач из списка.

  • Для печати определенного набора задач пометьте нужные задачи и в области Печать панели инструментов выберите пункт Печать выбранных задач.

  • Для печати всей папки списка задач выберите список в области обзора и в области Печать панели инструментов выберите пункт Печать папки задач.

Общедоступные списки задач

Списки задач можно сделать доступными для других сотрудников в офисе, для внешних гостей, или сделать их открытыми. Делая списки задач общедоступными для внутренних пользователей, вы можете выбрать для них тип доступа: статус управляющего или администратора с полным доступом для просмотра и изменения задачи, или статус наблюдателя только с правом чтения. Списки со статусом открытых и для внешних гостей предоставляют только право чтения.
