Создание списка задач

Можно создать другие списки задач и настроить их синхронизацию с удаленными службами, такими как список задач iCal.

Создание списка задач

  1. Щелкните вкладку Задачи, чтобы открыть приложение «Задачи».

  2. В области обзора щелкните правой кнопкой мыши заголовок «Задачи» и выберите пункт Новый список задач.

  3. В поле Создать новый список задач введите имя списка задач и выберите цвет для отображения задачи.

  4. Установите флажок Синхронизировать задачи с удаленного сервера, чтобы синхронизировать новый список со списком задач iCal. Введите URL-адрес iCal.

  5. Нажмите кнопку ОК. Список добавлен к задачам в области просмотра.
