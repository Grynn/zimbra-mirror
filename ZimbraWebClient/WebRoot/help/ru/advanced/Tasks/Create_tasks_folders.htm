Создание списка задач

  1. Чтобы открыть задачи, откройте вкладку Задачи

  2. Щелкните правой кнопкой мыши заголовок «Задачи» и выберите пункт Новый список задач.

  3. В поле Создать новый список задач введите имя списка задач и выберите цвет для отображения задачи.

  4. Нажмите кнопку ОК. Список добавлен к задачам в области обзора.
