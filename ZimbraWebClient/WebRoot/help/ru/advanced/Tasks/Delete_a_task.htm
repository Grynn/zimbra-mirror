Удаление задачи

Задачу можно удалить из списка задач следующими способами. Откройте список задач в области содержимого.

  • Выберите задачу для удаления и щелкните значок Корзины на панели инструментов.

  • Щелкните правой кнопкой мыши задачу и выберите пункт Удалить. Нажмите кнопку Да.

Задача удалена из списка.

Примечание. Если вы удалили задачу, можно сразу отменить это действие, щелкнув ссылку Отменить, которая отображается вверху страницы. Задача восстанавливается в исходном расположении.
