Перемещение задач

Задачу можно переместить в другой список одним из следующих способов.

  • Щелкните правой кнопкой мыши задачу в диалоговом окне Переместить задачу и выберите список задач для перемещения или создайте новый список. Нажмите кнопку ОК.

  • Выберите задачу для перемещения и, удерживая нажатой левую кнопку мыши перетащите задачу в списки в области обзора.

  • Откройте страницу задачи и в поле «Списки задач» выберите список задач.
