xxxxУправление личными данными учетной записиxxxx

xxxxПри отправке нового сообщения, ответе или пересылке сообщения, с электронным адресом всегда связывается одна запись личных данных.xxxx  Это личные данные по умолчанию, созданные как имя учетной записи и используемые до тех пор, пока не будут установлены другие личные данные.  Создание различных личных данных, называемых образами, позволяет использовать несколько адресов электронной почты в одном почтовом ящике. xxxxЭто может понадобится для управления различными ролями на работе и в личной жизни с помощью одной учетной записи.xxxx

Можно также добавлять адреса внешних учетных записей для получения почты с других почтовых ящиков POP или IMAP. Кроме того, можно настроить внешние учетные записи так, чтобы при ответе на сообщения в поле «От» появлялся соответствующий адрес.  

Если установлено несколько образов или добавлены внешние учетные записи, то при создании нового сообщения можно выбирать личные данные, которые будут использоваться в качестве адреса в поле От.

xxxxОбразы, внешние учетные записи и подписи можно настроить на вкладке xxxxxxxxНастройки->Учетные записиxxxxxxxx и на вкладке xxxxxxxxПодписиxxxxxxxx.xxxx
