Установка пользовательских настроек

Пользовательские настройки по умолчанию устанавливаются во время создания учетной записи. Эти настройки определяют работу функций @ClientNameLong@.

Настройки можно изменять на вкладке Настройки. Страницы настроек отображаются в области обзора. Если описанный здесь параметр не виден на виде, значит он не доступен в вашей учетной записи.

Настройки можно установить в следующих папках. Щелкните имя папки, чтобы просмотреть описание содержимого.

  • Общие

    На странице «Общие» можно выполнять следующие действия.

    • Настраивать версию ZWC, используемую по умолчанию (расширенная или стандартная), выбирать тему почтового ящика, язык и часовой пояс по умолчанию.

    • Определять настройки поиска с включением функции поиска в папках спама и корзины.

    • Отображение флажков в области содержимого.

    См. раздел Установка общих настроек

  • Почта

    На странице «Почта» можно выполнять следующие действия.

    • Настраивать интервал проверки новой почты. По умолчанию применяется значение 5 минут.

    • Выбирать действие при нажатии кнопки «Получить почту»: запустить поиск по умолчанию или обновить текущий вид.

    • Выбирать способ отображения сообщений: HTML или обычный текст.

    • Настраивать способ обработки получаемых сообщений почтовым ящиком, в том числе параметры просмотра сообщений, область чтения и параметры перемещения.

    • Настраивать уведомления о сообщениях.

    • Создавать автоответ, который будет использоваться в случае вашего отсутствия.

    • Включать подтверждение прочтения.

    • Если используется клиент POP3, можно указать параметры доступа по протоколу POP при загрузке сообщений в @ClientNameShort@.

    • Настраивать параметры спама для блокировки и разрешения сообщений с указанных адресов.

    См. раздел Установка настроек почты.

  • Написание письма

    На странице «Создание сообщения» можно выполнять следующие действия.

    • Выбирать текстовый редактор, используемый по умолчанию: HTML или обычный текстовый.

    • Укажите, нужно ли включать исходное сообщение в тело ответа.

    • Делать обязательной проверку правописания.

    • Указывать способ пометки сообщений, которые были пересланы или на которые был дан ответ.

    • По умолчанию включен параметр Сохранять копии сообщений в папке "Отправленные". Этот параметр можно изменить.

    См. раздел Установка настроек для окна создания сообщения.

  • Подписи

    Можно создавать подписи для сообщений. Если вы создаете несколько личных данных, можно создать разные подписи и назначить их определенным адресам.

  • Адресная книга

    На странице «Адресная книга» можно выполнять следующие действия.

    • Включать функцию автоматического добавления адресов в адресную книгу «Отправленные по электронной почте контакты» при отправке сообщений.

    • Включать функцию использования глобального списка адресов при просмотре контактов в средстве выбора контактов.

    • Включать глобальный список адресов при автозаполнении адресов.

    • Включать адреса из общих адресных книг при автозаполнении адресов.

    См. раздел Установка настроек адресной книги.

  • Учетные записи

    Страница «Учетные записи» используется для создания образов и управления ими. Образ — это личные данные почты, используемые в поле «От:» отправляемых сообщений. Можно настраивать собственные учетные записи и учетные записи POP.

  • Фильтры почты

    Папка «Фильтры почты» предназначена для определения правил фильтрации сообщений. Чтобы узнать как установить правила почтовых фильтров, см. раздел Как работают фильтры.

  • Ежедневник

    На странице «Ежедневник» можно выполнять следующие действия.

    • Определять вид ежедневников по умолчанию.

    • Устанавливать параметры, используемые при создании встречи.

    • Выбирать функцию отображения календарика.

    • Указывать адрес для пересылки приглашений из общего ежедневника.

    • Установка напоминания для собраний.

    • Устанавливать права доступа к сведениям доступности и для тех, кто может приглашать вас на собрания.

    См. раздел Установка настроек ежедневника.

  • Общий доступ

    На странице «Общий доступ» отображаются элементы, к которым вам предоставляется доступ, а также элементы, к которым предоставляете доступ вы. См. раздел Управление элементами общего доступа.

  • УведомлениеМожно указать электронный адрес или выбрать уведомление по SMS для мобильного устройства, чтобы получать сообщение с напоминанием о задаче или собрании в ежедневнике.

  • Мобильные устройстваЗдесь перечисляются мобильные устройства, настроенные для синхронизации с ZWC.  С помощью этой страницы можно также очистить устройство. См. раздел Управление мобильными устройствами.

  • Импорт и экспорт

    На странице «Импорт и экспорт» можно выполнить импорт или экспорт всех данных учетной записи, отдельных папок, ежедневников и списков контактов. Это полезно, если необходимо выполнить локальную архивацию учетной записи или отдельной папки.

  • Горячие клавиши

    На странице «Горячие клавиши» можно просмотреть список горячих клавиш и распечатать его.

  • Zimlet

    В папке «Zimlet» можно указывать, какие модули Zimlet необходимо использовать. Модули Zimlet представляют собой надстройки, расширяющие функциональные возможности клиента.  Модули Zimlet включаются системным администратором. Используемые модули Zimlet отображаются в области обзора.
