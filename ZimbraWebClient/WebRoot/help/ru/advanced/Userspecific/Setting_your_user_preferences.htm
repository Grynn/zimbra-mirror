Установка пользовательских настроек

Пользовательские настройки по умолчанию устанавливаются во время создания учетной записи. Эти параметры определяют работу почтового ящика, адресных книг и ежедневников. Можно также установить личные данные электронной почты и учетные записи POP, а также выбрать тему для почтового ящика.

Эти настройки можно изменить на вкладке Настройки. Если описанный здесь параметр не виден на виде, значит он не доступен в вашей учетной записи.

Настройки можно установить на следующих вкладках. Щелкните слово, чтобы открыть подробное описание вкладки.

  • Общие

    • Пользователь входит в стандартный веб-клиент. Чтобы сделать этот тип клиента стандартным при входе, выберите параметр Стандартный.

    • Расширенный клиент предлагает полный набор функций групповой работы в сети. Веб-клиент работает лучше всего с более новыми версиями браузеров и быстрым подключением к Интернету.

    • Стандартный клиент рекомендуется для медленных подключений к Интернету с использованием старых версий или для упрощенного доступа.

    • В разделе Тема выберите цвет фона для интерфейса почтового ящика.

    • Если отображается неверный часовой пояс, выберите для учетной записи часовой пояспо умолчанию. Он будет определять дату и время отправки и получения сообщений и время, отображаемое для встреч ежедневника.  Этот параметр может отличаться от настроек часового пояса на вашем компьютере.

    • Если отображается параметр Изменить пароль, можно изменить пароль. Если при смене пароля возникают проблемы, обратитесь к администратору, чтобы узнать правила в отношении пароля.

      • Если этот параметр отсутствует и нужно изменить пароль, обратитесь к системному администратору.

    • В меню Настройки поиска можно указать, чтобы папка «Спам» и Корзина автоматически включались в любой выполняемый поиск. По умолчанию поиск в этих папках не выполняется.

    • Для параметра язык поиска установите флажок Всегда показывать строку поиска для отображения подробных данных строки поиска в текстовом окне поиска.

    • Включите настройку Отображать флажки для быстрого выбора элементов в списках (требуется обновление)  для отображения флажка для каждого элемента в области содержимого. Флажок используется для выбора одного или нескольких элементов над которыми нужно выполнить одно и то же действие, например удалить, переместить, пометить как прочитанное или непрочитанное.

  • Почта

    • Выберите способ отображения элементов (сообщений или разговоров) на странице. По умолчанию на странице отображается 25 элементов, но можно выбрать 10, 50 или 100 элементов на страницу.

    • Выберите способ группирования почты по разговорам или сообщениям. При наличии функции разговора по умолчанию используется группирование по разговорам.

    • Интервал опроса — это интервал через который компьютер выполняет проверку новой почты.  По умолчанию опрос выполняется каждые 5 минут. Минимальный период времени определяется администратором Zimbra.

    • Укажите способ просмотра почты в виде HTML для сообщений, которые имеют формат HTML. Отключение этого параметра приводит к отображению почты в виде простого текста с отображением собственно тегов HTML вместо их применения для разметки текста.

    • Параметры Просмотра сообщений

      • Укажите, нужно ли показывать область чтения при просмотре почты. По умолчанию область чтения всегда отображается.

      • Если этот параметр включен, веб-клиент отображает фрагменты сообщений в списке сообщений. Если провести курсором по теме сообщения, отображаются первые несколько слов сообщения.

      • Если этот параметр включен, двойной щелчок кнопки мыши открывает сообщения в новом окне.

    • Включите параметр Изображения, чтобы загружать изображения автоматически в сообщения в формате HTML. Если этот параметр включен, щелкать изображение в сообщении для просмотра нет необходимости.

    • Принятый по умолчанию профиль поиска почты определяет поиск, выполняемый при входе пользователя. По умолчанию выполняется поиск в папке Входящие.  При входе в почтовый ящик отображается содержимое папки «Входящие». Для отображения в этом поле при первом открытии веб-клиента Zimbra, можно выбрать другую папку, тег или сохраненный профиль поиска. Синтаксис см. в разделе Описание языка запросов. Включите и укажите адрес для отправления уведомлений о получении новых сообщений.

    • Укажите электронный адрес для пересылки. Исходные сообщения могут удаляться из почтового ящика Zimbra.

    • Укажите, нужно ли отправлять автоответ на входящие сообщения.  Эта функция, также называемая сообщением об отсутствии, позволяет извещать отправителя, что вас нет на работе, и сообщать ему другие дополнительные сведения.

    • Настройте Сообщения от меня... для обработки отправляемых вами сообщений, которые должны включать ваш адрес, как одного из получателей сообщения, или адрес списка рассылки, участником которого вы являетесь.

     

  • Написание письма

    • Установите текстовый редактор по умолчанию: HTML или обычный текст.

    • Установите настройки шрифта по умолчанию, используемые с кодом HTML для создания в почте, документах и ежедневнике.  Можно определить стиль, размер и цвет шрифта, используемые по умолчанию. По умолчанию используется шрифт Arial размером 12 пунктов, черный.

    • Параметр Всегда писать письмо в новом окне открывает новое окно для создания сообщения, отдельно от почтового ящика Zimbra. При открытом окне, создание сообщения можно просматривать почтовый ящик и перемещаться в нем.

    • Создаваемые сообщения автоматически сохраняются как черновики каждые пять минут. Если черновики сохранять автоматически не нужно, эту функцию можно выключить.

    • Укажите, нужно ли включать исходное сообщение в тело ответа. Можно включать или не включать текст исходного сообщения, включать его как вложение, включать его в тело письма с выбранным префиксом, включать только самое последнее сообщение.  (Другими словами, в кавычки будет заключен только текст написанный тем, кто написал сообщение, но не предыдущий текст).

    • Укажите нужно ли ставить префикс > или | перед каждой цитируемой строкой предыдущих сообщений, которые пересылаются или на которые дается ответ.

    • По умолчанию включен параметр Сохранять копии сообщений в папке «Отправленные». Копии отправленных сообщений сохраняются в папке «Отправленные».

    • Укажите как включать текст исходного сообщения при пересылке сообщения: в теле письма, в теле письма с префиксом или как вложение.

  • Подписи

    Можно создавать подписи для сообщений. Подпись может включать имя и дополнительный текст длиной не более максимального количества знаков, допустимого для вашей учетной записи. Если вы создаете несколько личных данных, можно создать разные подписи и назначить их определенным адресам.

    • Введите подпись, которая должна появляться в сообщениях.

    • Укажите место подписи в сообщениях: внизу всех сообщений почты, в конце создаваемого или пересылаемого сообщения, или в конце сообщения, на которое дается ответ.

  • Адресная книга

    • Включите функцию для автоматического добавления адресов в адресную книгу во время отправки сообщений. Если этот параметр включен, адреса сохраняются в адресной книге «Отправленные по электронной почте контакты».

    • Включите параметр «Использовать глобальный список адресов при автозаполнении адресов».

    • Выберите вид контактов: в виде списка или визитных карточек.

    • Выберите количество контактов для отображения на странице. По умолчанию отображается 25 контактов.

    • Импорт контактов в список контактов. Поля в файле для импорта должны быть разделены запятыми.

    • Экспорт контактов. Поля в файле для экспорта должны быть разделены запятыми.

     

  • Учетные записи

    Вкладка учетных записей используется для создания и управления образами. Образ — это личные данные почты, используемые в поле «От:» отправляемых сообщений.

    Стиль по умолчанию определен в основных личных данных. При создании новых личных данных, можно использовать те же самые настройки, которые используются в личных данных по умолчанию, можно так же установить другие настройки.

    Кнопка Добавить внешнюю учетную запись позволяет определить внешние учетные записи POP/IMAP для получения почты в веб-клиенте с других учетных записей.  Можно установить, чтобы в ответах на эти сообщения использовался соответствующий адрес POP/IMAP.

  • Фильтры почты

    На вкладке «Фильтры почты» можно определить правила фильтрации сообщений. Чтобы узнать как установить правила почтовых фильтров, см. раздел Как работают фильтры.

  • Ежедневник

    • Выберите вид ежедневника Zimbra. По умолчанию отображается рабочая неделя.

    • Укажите день недели, который должен быть первым днем недели ежедневника.

    • При изменении часовых поясов, можно выбрать отображение списка «Показывать часовой пояс на виде встречи».

    • При создании многих встреч без участников установите флажок Использовать диалоговое окно «Быстрое добавление».

    • В ежедневнике отображается календарик.  Календарик можно сделать видимым на любом виде.

    • Установите, за сколько минут до встречи должно поступать напоминание о ней. По умолчанию за пять минут.

    • Импорт и экспорт других программ-ежедневников.

  • Мгновенные сообщения

    Можно установить способ интеграции мгновенных сообщений (МС) в учетную запись.

    • Вход. Для автоматического входа в сервисы обмена мгновенными сообщениями этот параметр должен быть включен.

    • Уведомление. Этот параметр определяет уведомление о полученных МС и ваш режим доступности для других.

    • Чат.  История чата автоматически сохраняется в папке «Чат». Сообщения в этой папке такие же, как и любые другие сообщения: их можно распечатывать, пересылать и удалять из папки. Можно так же установить время бездействия.

  • Быстрые клавиши

    Быстрые клавиши дают возможность пользоваться приложением без помощи мыши. Имеется большое количество быстрых клавиш. Можно так же создать специальные комбинации клавиш, назначив цифровые псевдонимы для папок, тегов и сохраненных профилей поиска.

    • Список быстрых клавиш показывает все существующие комбинации быстрых клавиш.

    • Вкладка Быстрые клавиши папки «Почта» используется для создания комбинации клавиш для быстрого перехода в папки почты и перемещения сообщений в эти папки. Необходимо выбрать папку и назначить ей номер. Например, если назначить папке «Входящие» номер 3, можно переместиться в нее, введя v3 из любой папки почтового ящика. Выбрав элемент и введя .3, можно переместить элемент в папку «Входящие».

    • Вкладка Быстрые клавиши сохраненных профилей поиска используется для создания быстрых клавиш для быстрого доступа к сохраненным профилям поиска. Необходимо выбрать профиль поиска и назначить ему номер. Например, назначив номер 3 избранному профилю поиска, можно быстро запустить этот профиль, введя s3 из любой папки почтового ящика.

    • Вкладка Быстрые клавиши тегов используется для создания быстрых клавиш для быстрого доступа ко всем элементам с определенным тегом или применения тега к сообщению или контакту. Необходимо выбрать тег и назначить ему номер. Например, тегу с именем К исполнению присваивается номер 3. Введя y3 из любой папки почтового ящика, можно просмотреть все элементы, помеченные тегом К исполнению. Выберите элемент и введите t3, чтобы пометить элемент тегом К исполнению.

    Можно назначить любой номер, но он не должен повторятся на вкладке быстрых клавиш.
