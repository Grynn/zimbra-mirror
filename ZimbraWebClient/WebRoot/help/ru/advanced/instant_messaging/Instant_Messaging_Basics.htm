Мгновенные сообщения. Основные сведения (бета-версия)

xxxxФункция мгновенных сообщений позволяет обмениваться сообщениями с друзьями в реальном времени.xxxx Введенное и отправленное сообщение сразу же отображается в почтовом ящике получателя.

Мгновенные сообщения в @ClientNameShort@ можно использовать для следующих целей.

  • Создание списка контактов, которые называются также приятелями, для общения в режиме реального времени.

  • Общение с приятелями в отдельных чатах МС.

  • Сохранение записей чатов.

  • Интеграция мгновенных сообщений с почтой.

  • Настройка параметров мгновенных сообщений.

  • Выполнение входа во внешние клиенты МС, например Yahoo! Messenger.
