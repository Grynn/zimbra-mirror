Настройка параметров мгновенных сообщений

На вкладке «Настройки» > «Чат» используйте следующие параметры. Не забудьте нажать кнопку «Сохранить», чтобы сохранить все изменения в настройках.

  • xxxxВходxxxxxxxx.xxxx Установите этот флажок, если необходимо выполнять автоматический вход в службу мгновенных сообщений при входе в @ClientNameLong@. Если этот флажок не установлен, вход выполняется вручную путем изменения статуса «Не в сети» на статус «В сети» в меню статуса.

  • Уведомления. Параметры отображения уведомлений МС.

    • Включить мгновенные уведомления. Отображение краткого всплывающего уведомления для других пользователей о вашем статусе.

    • Отображать уведомления о присутствии. Отображение краткого всплывающего уведомления о входе и выходе приятелей из сети.

    • Отображать уведомления об изменении статуса приятеля. Отображение краткого всплывающего уведомления при изменении приятелями своего статуса в службе мгновенных сообщений @CompanyName@.

  • При получении сообщений. Действия при получении МС.

    • Воспроизведение звука. Воспроизведение звуков при получении сообщения. Для воспроизведения звуков должен быть установлен подключаемый модуль QuickTime или Windows Media.

    • Мерцание заголовка браузера. Мерцание заголовка браузера при получении мгновенного сообщения, если открыта другая вкладка или другое окно браузера.

    • Показать всплывающее уведомление. Для этой функции должно быть установлено приложение Yahoo! BrowserPlus.

  • Чат. Настройка параметров чата.

    • Сохранить журнал чата. Сохранение записей чатов. Сохраненные чаты находятся в папке «Чаты» на вкладке «Почта».

    • Сообщать о состоянии бездействия. Отображение состояния бездействия для приятелей при отсутствии каких-либо ваших действий в @ClientNameShort@ на протяжении времени, указанного в параметре Таймаут простоя.

  • Таймаут простоя. Указание времени, которое должно пройти перед изменением вашего статуса. Статус изменяется согласно настройкам параметра Установить присутствие.

  • Установить присутствие. Выбор статуса, который отображается, когда функция мгновенных сообщений не используется. Статус изменяется по прошествии времени, указанного в параметре Таймаут простоя.
