Сохранение содержимого учетной записи

Пользователь может экспортировать все данные учетной записи, в том числе сообщения и вложения, контакты, ежедневник, задачи, папки Портфеля, или можно экспортировать отдельные элементы учетной записи. Данные можно сохранить на компьютере или в другом месте в качестве резервной копии. Данные учетной записи сохраняются в архивном файле формата tar-gzipped ( tgz), который можно импортировать обратно для восстановления учетной записи.

 Примечание. В случае экспорта отдельных папок контактов они сохраняются в файлах формата .csv, а отдельные папки ежедневников сохраняются в файлах формата .ics.

Данные только копируются, но не удаляются из учетной записи.

Экспорт файла можно выполнить, открыв Настройки > Импорт/экспорт.

Как экспортировать данные

  1. На вкладке «Настройки», щелкните папку Импорт/экспорт.

  2. Экспортируйте все данные учетной записи или укажите для экспорта папку определенного типа.

    • Чтобы экспортировать все данные учетной записи, выберите пункт Учетная запись. Примечание. Папки «Корзина» и «Спам» не экспортируются.

    • Чтобы экспортировать список контактов или определенный ежедневник, выберите соответствующий вариант. В случае экспорта ежедневника или списка контактов необходимо указать формат файла для экспорта. См. разделы Экспорт адресных книг и Экспорт ежедневника iCal.

    • Чтобы выбрать несколько определенных типов элементов для одновременного экспорта, установите флажок Дополнительные настройки. Окно приобретет расширенный вид. Установите флажки для элементов данных, которые требуется экспортировать. В экспорт будут включены все элементы выбранного типа изо всех папок, находящихся в выбранной папке. Если требуется экспортировать данные определенного типа, для указания типа данных можно воспользоваться фильтром поиска. См. разделы Описание языка запросов и Примеры поиска.  

  1. Нажмите кнопку Экспорт и выберите место для сохранения файла.

  2. Нажмите кнопку ОК.

Примечание. Экспортированные данные учетной записи можно просмотреть с помощью программы архивирования, например WinRAR.

Как импортировать сохраненные данные учетной записи

Учетную запись можно восстановить из экспортированного файла.  

  1. На вкладке «Настройки» откройте папку Импорт/экспорт.

  2. В разделе Импорт выберите файл, который необходимо импортировать. В случае импорта ежедневников или контактов укажите формат файла импорта. См. разделы Импорт контактов и Импорт ежедневника iCal. Если выбран тип «Учетная запись», в учетную запись может быть импортирован только ранее экспортированный файл формата tgz.  

В разделе Типы данных выберите для импорта нужные типы элементов из сохраненного файла формата tgz. После нажатия кнопки «Импорт» импортируются только элементы указанных типов.

  1. Нажмите кнопку Импорт.
