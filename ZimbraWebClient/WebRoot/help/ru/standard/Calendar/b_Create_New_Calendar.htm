Создание новых ежедневников

Для пользователя создается один ежедневник, также можно создавать дополнительные ежедневники для ведения учета функций различных типов.  Например, можно создать ежедневник для рабочих, личных мероприятий и мероприятий организации, подписаться на доступный в Интернете ежедневник, а также можно подключиться к общему ежедневнику, к которому вам предоставлен доступ.

Можно использовать вид Расписание для одновременного просмотра нескольких ежедневников.

Cоздание ежедневника

  1. Перейдите на вкладку «Ежедневник».

  2. В области «Обзор» ежедневника щелкните пункт Изменить.

  3. На панели инструментов нажмите кнопку Новый ежедневник.

  4. Введите имя ежедневника

  5. Выберите цвет для отображения мероприятий ежедневника. Так ежедневникам в почтовом ящике назначаются цвета, что дает возможность всегда отображать мероприятия ежедневника в выбранном цвете.

  6. Если просмотр другими лицами мероприятий ежедневника нежелателен, установите флажок Исключить этот ежедневник при составлении отчета о свободном/занятом времени.

  7. По умолчанию параметр Помечено в UI включен. Мероприятия данного ежедневника будут отображаться при открытии вида «Ежедневник».  Если данный ежедневник не следует отображать вместе с другим ежедневником, снимите флажок. В таком случае для просмотра мероприятий ежедневника необходимо выбрать ежедневник и открыть его.

  8. Щелкните пункт Создать ежедневник.

Подписка на другой ежедневник

  1. Перейдите на вкладку Ежедневник.

  2. В области «Обзор» ежедневника щелкните пункт Изменить. Отобразятся сведения ежедневника по умолчанию.  

  3. На панели инструментов нажмите кнопку Подписаться на ежедневник.  

  4. Введите название ежедневника, допустимый URL-адрес файла iCal удаленного ежедневника и выберите цвет для отображения мероприятий данного ежедневника.

  5. Если просмотр другими лицами мероприятий ежедневника нежелателен, установите флажок Исключить этот ежедневник при составлении отчета о свободном/занятом времени.

  6. По умолчанию параметр Помечено в UI включен. Мероприятия данного ежедневника будут отображаться при открытии вида «Ежедневник».  Если данный ежедневник не следует отображать вместе с другим ежедневником, снимите флажок. В таком случае для просмотра мероприятий ежедневника необходимо выбрать ежедневник и открыть его.

  7. Щелкните пункт Создать ежедневник.
