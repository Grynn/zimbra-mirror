Автоматическое заполнение адреса

Функция автозаполнения предлагает варианты имен при вводе текста в полях Кому, Копия и Скрытая копия нового сообщения. Отображается список возможных адресов из списка контактов пользователя, начало которых соответствует введенному тексту.

По мере ввода текста в поле Кому список соответствующих адресов становится меньше, так как уменьшается количество совпадений. Если совпадений нет, список исчезает.

Пример

Например, представим, что необходимо отправить сообщение электронной почты Дмитрию Богатыреву, у которого следующий адрес электронной почты: dbogatyrev@acme.com. После ввода первой буквы «д» в поле адреса появится список, содержащий все возможные соответствия, с выделенными частями, которые соответствуют. В данном случае, может отобразиться следующее:

Денис Попов <dpopov@acme.com>

Дмитрий Богатырев <dbogatyrev@acme.com>

Анна Добровольская <anna878@xyz.com>

Чем больше символов введено в поле Кому, тем меньше становится список соответствий.

Соответствие

Функция автозаполнения сопоставляет вводимый текст со следующими элементами контакта.

  •  Имя

  •  Фамилия

  • Адрес электронной почты

Сопоставление осуществляется с началом поля. Тексту «р;вова» будет соответствовать «р;вова иванов», но не «р;володя иванов».

Использование автозаполнения

Если текст соответствует одному или нескольким контактам, соответствия отображаются в списке под полем, в котором осуществляется ввод. По умолчанию выделяется первое соответствие.

Имена отображаются в «рейтинговом» порядке. Контакты, которым уже отправлялись сообщения, располагаются выше контактов, которым сообщения еще не отправлялись или отправлялись редко. Поэтому наиболее часто вызываемый контакт отображается вверху списка. Если вас не устраивает имя контакта, которое постоянно отображается вверху, можно нажать кнопку Забыть, и контакт будет понижен в рейтинге.

Несколько кнопок имеют особые функции. Мышь также можно использовать для выбора одного из соответствий.

  • Нажатие клавиш с запятой, точкой с запятой, Enter и Tab инициирует заполнение. Введенный текст будет заменен одним из соответствий, выбранным в данный момент.

  • Если нажать клавишу ESC, список исчезнет.

  • Щелкнув мышью одно из соответствий, можно выбрать его для заполнения.

Заполнение нажатием одной клавиши (например, клавиши с точкой с запятой) можно использовать для быстрого ввода адресов. Если известны несколько знаков имени определенного контакта, который будет первым в списке соответствий, можно ввести эти знаки, а затем нажать клавишу с точкой с запятой, и появится адрес данного контакта. Например, если известно, что ввод букв «р; db» приведет к тому, что первым соответствием будет Дмитрий Богатырев, можно ввести «р; db;» и полный адрес «р;Дмитрий Богатырев <dbogatyrev@acme.com>» появится в поле адреса.

Примечание. Настроить работу автозаполнения можно на странице Настройки, Адресная книга.
