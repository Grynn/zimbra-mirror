xxxxСоздание и сохранение запроса поискаxxxx

В случае частого выполнения какого-либо запроса поиска можно создать запрос и сохранить его в папке «Поиск». Для повторного выполнения поиска нужно просто щелкнуть папку с сохраненным профилем поиска. Это действие запускает поиск, результаты которого отображаются в области содержимого.

xxxxКак создать и сохранить запрос поискаxxxx

  1. В заголовке «Область обзора > Поиск» нажмите кнопку Изменить. В результате в обновившейся области обзора будут отображены все папки в алфавитном порядке, а в области содержимого отобразится папка «Входящие».

  2. На панели инструментов выберите пункт Новый поиск. xxxxВ области содержимого появится новая форма поиска.xxxx

  3. В поле Имя укажите имя профиля поиска.

  4. В поле Родительская папка укажите, где необходимо сохранить этот профиль поиска. По умолчанию он сохраняется в корневой папке. Папка будет сохранена под заголовком «Поиск» в области обзора. Если профиль поиска необходимо сохранить в другой папке, выберите нужную папку в раскрывающемся списке. Папку поиска можно узнать по значку с изображением лупы.

  5. В поле Запрос поиска создайте запрос поиска. Сведения о грамматике языка поиска см. в разделе Описание языка запросов.

  6. В диалоговом окне Сохранить профиль поиска укажите имя папки, в которой будет сохранен профиль поиска, и каталог, в котором она будет создана. В случае сохранения в корневой папке профиль поиска будет отображаться в разделе «Поиск». В случае выбора другой папки профиль поиска сохраняется в выбранной папке.
