Использование поиска

Функция поиска Zimbra — мощный инструмент. Хотя вы можете научиться создавать сложные текстовые запросы, как описано в разделе Описание языка запросов, проще воспользоваться быстрой клавишей и ввести имя или слово без ключевых слов. Например, чтобы найти все сообщения, которые содержат слово «tim» где угодно в заголовке, в поле «Кому:», «От:», «Копия:» или «Скрытая копия:», в теле сообщения или файловом вложении, можно просто ввести tim в поле Найти и щелкнуть Поиск.

Область поиска выбирает пользователь. Поиск можно выполнять в сообщениях, личных контактах, личных ежедневниках и списках задач.

Можно ввести любое значение: слово, имя, фамилию, номер телефона или домен.   Для дополнительных вариантов можно использовать звездочку (*) как подстановочный знак.  

Использование * в качестве подстановочного знака в профиле поиска

Звездочку (*) можно использовать в качестве подстановочного знака при поиске содержимого со словами с похожим написанием.  

Используйте * (звездочку) ак подстановочный знак после префикса. Т.е. поиск ты* выдаст элементы со словом тыква, тысяча и т.д.

Для поиска контактов из адресных книг в запрос необходимо вводить целое слово. Например, для поиска контактов по номеру телефона, необходимо ввести полный номер телефона как он указан в записи контакта, выполнять поиск только по коду города нельзя.

Поиск контактов в личной адресной книге

  1. В поле «Найти» выберите из списка пункт Личные контакты. Поиск будет выполнен во всех адресных книгах.

  2. Введите имя или другие данные контакта в текстовом поле Найти.

  3. Нажмите кнопку Поиск, чтобы выполнить поиск.

Быстрый поиск электронных сообщений

  1. В поле «Найти» выберите из списка пункт Почтовые сообщения.

  2. Введите имя или другие данные о сообщении в текстовом поле Найти.

  3. Нажмите кнопку Поиск, чтобы выполнить поиск.

Создание сложного запроса поиска

Если вы часто выполняете один и тот же профиль поиска или хотите сохранить определенные подробные данные поиска, можно создать запрос поиска и сохранить его в папке «Поиск». Для повторного выполнения поиска достаточно просто щелкнуть этот профиль поиска.

Создание и сохранение запроса поиска

  1. На панели обзора поиска нажмите кнопку Изменить. Откроется диалоговое окно Создать новый профиль поиска.

  2. Выберите каталог для создания папки поиска.  Если сохранить папку в корневом каталоге, профиль поиска будет отображаться в разделе «Поиск». Если выбрать другую папку, профиль поиска будет сохранен в этой папке.

  3. В поле Имя поиска укажите имя профиля поиска.

  4. В поле Запрос поиска введите запрос с помощью языка запросов, как описано в разделе Описание языка запросов. Можно также воспользоваться операторами AND или OR в запросе.

    Использование And или OR при создании запроса поиска

    Для поиска с использованием нескольких критериев, можно найти элементы соответствующие одному или всем указанным критериям. :

    • Поиск сообщений, соответствующих любым указанным критериям называется поиском с условием OR (ИЛИ), потому что, если сообщение содержит X или Y, оно считается соответствующим условиям поиска.

    • Поиск сообщений, которые содержат X и Y называется поиском с условием AND (И), потому что сообщение должно соответствовать всем указанным критериям, чтобы считаться соответствием.

  5. Щелкните Создать профиль поиска, чтобы создать и выполнить профиль поиска и сохранить его в указанной папке.

Для выполнения поиска щелкните имя поиска в области обзора. Результаты поиска отображаются в области содержимого.
