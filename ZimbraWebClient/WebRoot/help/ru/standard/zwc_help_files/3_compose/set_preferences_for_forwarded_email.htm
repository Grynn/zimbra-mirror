Настройка параметров для пересылки сообщений электронной почты

При пересылке сообщения электронной почты можно выбирать способ обработки исходного текста.

  1. Перейдите на страницу Настройки>Почтав раздел Создание сообщений.

  2. Если необходимо поставить префикс перед исходным сообщением, выберите либо угловую скобку (>), либо вертикальную черту (|).

Примечание.   Если выбрана функция префикса перед ответными сообщениями, необходимо использовать один и тот же префикс как для ответов, так и для пересылаемых сообщений.
