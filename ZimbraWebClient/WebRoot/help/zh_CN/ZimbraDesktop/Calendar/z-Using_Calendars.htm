在 Zimbra Desktop 中使用行事历

您可以在 Zimbra Desktop 中为您的每一个帐户维护一个行事历。

Zimbra 帐户行事历与 Zimbra Desktop 保持同步。当您添加一个新事件，修改一个事件或从 Zimbra Desktop 中删除一个事件时，所作的更改会反向同步到您的 Zimbra 帐户中的行事历。

您可以在 Zimbra Desktop 中为您的其他每一个帐户维护一个行事历，但行事历事件不会反向同步到这些帐户中的行事历。

如果您在使用另一个行事历程序，您可以将行事历保存为 iCalendar (.ics) 文件，然后将该文件导入到您的行事历中。如果您对 Zimbra Desktop 中的行事历进行了更改，则这些更改不会反向同步到那个程序中的行事历。如果重新载入 .ical 行事历，则您的 Zimbra Desktop 中的行事历即会被覆盖，并且您所添加的所有事件均将丢失。
