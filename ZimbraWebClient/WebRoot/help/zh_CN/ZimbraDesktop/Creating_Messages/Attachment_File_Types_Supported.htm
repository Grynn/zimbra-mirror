支持的附件类型

您可以直接从 Zimbra Web Client 打开电子邮件中的任何附件，条件是：所用工作站须有合适的应用程序，并且扩展名没有被系统管理员封锁。 例如，要打开用 Microsoft Word 创建的文档，就需要能够在您的计算机上运行 Microsoft Word。

对于大多数计算机，文件类型由其扩展名指定，而扩展名通常就是一个句点，后跟三字母代码。例如，expenses.xls 或 New-Items.doc 等文件。如果文件类型是您计算机上安装的软件所支持的类型，通常可以双击文件，您的计算机就会自动启动相应的应用程序读取该文件。

通过各种插件，许多文件类型可在浏览器上直接打开。通常情况下，当您在页面上单击一个文件链接时，浏览器会自动检测是否有适用的插件，必要时提示您下载该插件。请注意，系统管理方面的规定可能不允许终端用户下载和安装任何类型的软件。

注：
对于 Windows 计算机而言，计算机假定文件名中的文件类型符合实际格式。即便文件本身格式正确，但如果文件名中的文件类型扩展名不对，计算机则不能识别，亦不能使用正确的应用程序。

越来越多的计算机病毒都是通过附件传播的。因此在某些环境下，系统管理员已采取了一定的措施，阻止有特定类型附件的邮件进入系统，特别是文件名中包含 .EXE 或 .ZIP 扩展名的文件。如果将邮件发到这样的系统，即被设定为阻止某些特定类型的附件之系统，则您有可能会收到被阻止的通知，或根本没有通知。

为便于查阅，这里列出了一些广为使用的文件扩展名。

  • .TXT - 指文本文件，即只包含文本信息的文件。文本文件可用各种文本编辑器打开和阅读，其中包括 Notepad、WordPad、Word、命令行编辑器、浏览器、甚至电子表格程序。

  • .HTM - 指 HTML 文件，通常可用浏览器打开，这是一种特殊类型的文本文件。

  • .ZIP - 指用 ZIP 压缩程序压缩的文件。若需阅读和展开 ZIP 文件，需要 WinZip 或其他压缩程序。

  • .EXE - 指可执行文件或程序文件。双击这样的文件通常可启动相关的应用程序。

  • .DOC - 指 Microsoft Word 文档文件。

  • .JPG - 指 JPEG 格式的图像文件。您可在浏览器中打开 JPEG 文件，也可使用 Paint 或 Adobe Photoshop 一类的图像处理程序打开。

  • .GIF - 指 GIF 格式的图像文件。您可用浏览器或图像处理程序打开 GIF 文件。

  • .MPG - 指 MPEG 格式的电影片段或动画文件。通常可用浏览器插件或 Windows Media Player 等播放器播放。

  • .WMV - 指 WMV 格式（Windows 特有格式）的电影片段文件。

  • .PDF - 指 Acrobat Reader 格式的文档。通常可在浏览器中用插件打开 PDF 文件。
