关于 Zimbra Desktop

Zimbra Desktop 是一种离线的客户端，可用于在同一个桌面工具下管理多个电子邮件帐户。Zimbra 邮件、Yahoo! 邮件、Gmail、Outlook 以及任何其他使用 POP 或 IMAP 的工作或私人电子邮件帐户都可以在 Zimbra Desktop 中轻松设置。每个帐户分别保存。

设置好您的帐户后，您的电子邮件将被下载到您的计算机中，不管是否连接网络，您都可以访问这些电子邮件。参见离线工作。您所有帐户中的电子邮件都可以在 Zimbra Desktop 中阅读、回复、转发和删除。邮件的存储量取决于您的计算机的磁盘空间大小，不取决于预定义的帐户配额

当您的任何帐户收到新电子邮件时，如果这时您已连接网络，则该邮件将自动下载到您的 Zimbra Desktop。此功能并不需要您打开 Desktop。Windows Tray 或 Mac Dock 行中有一个信封图标，它会显示您所有帐户的未读邮件总数。单击此图标可打开此桌面应用程序。

Zimbra Desktop 的基本功能。

除了从一个位置访问您的不同电子邮件帐户外，Zimbra Desktop 还可用来管理您的联系人，并且具有效率手册、任务列表等应用程序以及用于存储附件的公文包工具。

参见 Zimbra Desktop 功能概况中的基本功能列表。

可设置并管理的账户类型

可设置下列电邮账户

  • Zimbra 帐户。如果 Zimbra Desktop 运行的是 Zimbra，您就可以使用它来访问您的电子邮件帐户。这包括电子邮件、文件夹、地址簿、效率手册、任务列表和公文包文档。使用 Zimbra Web Client 通过浏览器访问 Zimbra 帐户。您在 ZWC 中为 Zimbra 帐户设置的首选项在 Zimbra Desktop 中可用。备注：若需通过 Zimbra Desktop 使用所有Zimbra 功能，Zimbra 应为 ZCS 5.0 版本或更高版本。您的 Zimbra 系统管理员可告知您 Zimbra 的版本号。

  • Yahoo!Mail。您可以设置 Yahoo!Mail、Yahoo!Mail Plus、Yahoo!Small Business、Ymail 或 Rocketmail 帐户。您的电子邮件、电子邮件文件夹和地址簿在 Zimbra Desktop 中可用。

  • Gmail。您的 Gmail 帐户必须设置为允许 IMAP 访问。您可能需要登录到您的 Gmail 帐户以启用 IMAP。您的电子邮件和联系人在 Zimbra Desktop 中可用。

  • Microsoft Exchange IMAP 帐户。您的 Exchange IMAP 帐户电子邮件在 Zimbra Desktop 中可用。

对于上述帐户之外的帐户，如果它是 POP3 或 IMAP4 帐户，则同样可以在 Zimbra Desktop 进行设置。您需要关于您的帐户的完整设置信息，包括接收邮件服务器、外发邮件服务器的详细资料。系统管理员或网络服务提供商可以为您提供这些信息。
