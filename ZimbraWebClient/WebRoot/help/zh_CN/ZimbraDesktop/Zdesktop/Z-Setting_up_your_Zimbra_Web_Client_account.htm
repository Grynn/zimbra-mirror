设置您的 Zimbra Web Client 帐户

您须在您现有的帐户中首先输入设置值，然后方可使用 Zimbra Desktop。首次打开 Zimbra Desktop 时，会要求您建立一个帐户。您可随时建立其他帐户。

在开始建立帐户前，确认您已连接到互联网。

  1. 打开 Zimbra Desktop 并在右上角单击设置。  

  2. 单击添加新帐户。在“帐户类型”列表中选择 Zimbra。“Zimbra 帐户设置”页面随即显示。输入以下信息。

    • 帐户名称。这是作为账户名显示在 Zimbra Desktop 中的名称。每个账户必须有不同的名称。例如，对于 ZWC 帐户，可称之为 Zimbra。

    • 电子邮件地址。输入您的 ZWC 账户的电子邮件地址。

    • 密码。输入用于登录您的 ZWC 帐户的密码。

    • 接收邮件服务器。输入 Zimbra 服务器的地址。通常，其格式为 mail.example.com。这是您的账户所在的服务器。

    • 安全。如果您始终应该使用加密连接，勾选访问此服务器时使用 SSL 加密。如果您不能确定是否选择此选项，请与您的系统管理员联系。

    • 在同步设置中，选择 Zimbra Desktop 从您的帐户中获取数据的频率。默认设置是在新邮件到达时检查邮件并进行同步。

如果您选择手动同步，您必须单击 Zimbra Desktop 工具栏上的发送/接收，从而将您的帐户与 Zimbra 服务器同步。

如果您选择的间隔时间太短，如 1 分钟，您的计算机的性能可能要受到影响。

您可随时修改这个设置，方法是单击“设置”，然后编辑相关信息。

  1. 完成后，单击验证并保存。

Zimbra Desktop 随即开始验证您的帐户信息。如果您的帐户顺利通过验证，您将看到服务已创建对话框。如果您的帐户未能通过验证，须返回到“帐户设置”对话框查看您的输入并进行必要的修改。

  1. 若需打开 Desktop，单击启动 Desktop。在您的帐户进行同步期间，您可在 Zimbra Desktop 中工作。
