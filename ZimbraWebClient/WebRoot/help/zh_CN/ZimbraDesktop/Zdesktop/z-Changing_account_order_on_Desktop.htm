整理 Desktop 视图

当您设置 Desktop 时，您的帐户将会分别显示在“概况”窗格中。如果想要让所有邮件都显示在同一个帐户中，请启用常规首选项与我的其他帐户一起显示“所有邮箱”部分。此部分下将列出您的各个帐户的“收件箱”、“已发送邮件”和“草稿箱”中的所有邮件。所有帐户未读邮件的数量随即显示。您的个人帐户随即在“所有邮箱”下列出。您只可单击某个帐户来查看该帐户中的内容。

 

 

当您查看“所有邮箱”视图中的邮件时，邮件列表中的图标将标识该邮件来自哪个帐户。

 

默认帐户

当您打开 Zimbra Desktop时，您的帐户将以您设置的顺序显示在“概况”窗格中。第一个帐户是默认的 desktop 帐户。这是您启动 Zimbra Desktop 时打开的帐户。您可以更改将哪个帐户设置为默认帐户。

  1. 单击 Zimbra Desktop 上的设置。我的帐户页面随即显示帐户列表。

  2. 若需更改默认帐户，请单击帐户名称旁边的 。该帐户随即被移动到顶端。  

 

默认帐户不仅可以是显示 Zimbra Desktop 时打开的帐户，还可以是将 Zimbra Desktop 配置为默认的邮件应用程序时使用的帐户。您可在“系统默认值”部分的“首选项”>“常规”页面进行设置。
