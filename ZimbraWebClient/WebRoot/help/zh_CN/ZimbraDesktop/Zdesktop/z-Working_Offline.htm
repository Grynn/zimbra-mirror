离线工作

在 Zimbra Desktop 中，您可以随时访问从您的帐户下载的邮件，即使您并没有连接到互联网也是如此。在未连接互联网的情况下，在 Zimbra Desktop 中工作称为“离线工作”。

离线工作时，Zimbra Desktop 不能访问您的电子邮件帐户，但您仍能在 Zimbra Desktop 中工作，就像您已连接到互联网一样。您可查看您的邮件、创建新邮件、清除文件夹以及其他改动操作，在下一次连接到互联网时，所做的改动将同步到您的帐户。

如果您的帐户名旁边有一个红框，就表明您正在离线工作

 

备注：您可能已离线但该图标尚未变红，这是由于网络状态更新时 Zimbra Desktop 正在尝试同步到服务器。如果尝试失败，该图标才变红。

 

您可通过单击概述窗格上“帐户”旁边的小图标迫使所有的帐户变为离线状态。

再次单击图标将返回在线状态。  

离线工作时，可做些什么？

  • 可以阅读邮件，把邮件移动到文件夹，并为邮件打标记以便于后继处理。

  • 可以键入对邮件的回复并将之发送，可以转发邮件或创建新邮件。这样的邮件将保存在发件箱中，等到下次连接到互联网时即被自动发送。

  • 可以删除邮件。删除的邮件被放入“已删除邮件”文件夹中。对于 Zimbra 帐户，等到下次连接到互联网时，相应的邮件即被移动到您的帐户的“已删除邮件”文件夹中。如果您清空“已删除邮件”文件夹，其中的邮件即被删除。

  • 可以使用“搜索”和“高级搜索”功能查找您的 Zimbra Desktop 帐户中的邮件和附件。

  • 可以创建并保存搜索项。

  • 您可以更改 Zimbra Desktop 首选项。这些设置会影响您在 Zimbra Desktop 中工作的方式。更改 Zimbra Web Client Desktop 中的首选项时，Zimbra Web Client 帐户的首选项亦将相应改变。对其他邮件帐户的“首选项”所做的更改只对 Zimbra Desktop 的设置有影响。这些更改不会改变您的帐户设置。
