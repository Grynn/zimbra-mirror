}}}在 Zimbra Desktop 共享文件夹中工作{{{

备注：}}}仅适用于 Zimbra 帐户。{{{

}}}当您接受共享者的共享文件夹时，该文件夹即显示在您 Zimbra Desktop 中的“概况”窗格中。{{{}}}若您在 Zimbra Desktop 中在线工作，则可看到共享文件夹的内容。{{{}}}该信息并没有实际地保存在您的邮箱中或您的计算机中。{{{

}}}在授权者共享文件夹时，可从下面的访问角色中选择一个：{{{

  • }}}浏览者。{{{}}}此类共享者可阅读文件夹的内容，但不能更改其内容。{{{

  • }}}管理者。{{{}}}此类共享者有权查看和编辑文件夹的内容，创建新的子文件夹，以您的名义设置显示内容以及删除文件夹中的内容。{{{

  • }}}管理员。{{{}}}此类共享者拥有所有权限，可以查看和编辑共享文件的内容，创建新的子文件夹，以您的名义显示内容，删除文件夹中的内容，以及与其他人共享文件夹。{{{

}}}邮件文件夹管理者或管理员角色{{{

  • }}}管理者。{{{}}}此类共享者有权查看和编辑邮件文件夹的内容，创建新的子文件夹，以您的名义发送电子邮件以及删除邮件。{{{

  • }}}管理员。{{{}}}此类共享者拥有所有权限，可以查看和编辑邮件文件夹的内容，创建新的子文件夹，以您的名义发送邮件，删除邮件，以及将您的邮件文件夹与其他人共享。{{{

}}}您发送的邮件的副本被保存在授权者的邮箱中，如果您删除授权者邮箱中的某些内容或将其标记为垃圾，它将移动到授权者的“已删除邮件”或“垃圾邮件”文件夹中。{{{

如果您有权访问授权者的多个文件夹，则可将内容从授权者邮箱的一个文件夹移动到另一个文件夹。}}}您可将授权者文件夹中的邮件复制到您的邮件文件夹中，但授权者邮件文件夹中仍然保留一个副本。{{{

您可在共享邮件文件夹中创建文件夹，新创建的文件夹在授权者的邮箱中显示。

效率手册管理者或管理员角色

  • }}}管理者。{{{}}}此类共享者有权在此效率手册中创建会议、接受或拒绝邀请、编辑和删除您的效率手册中的活动。{{{

  • }}}管理员。{{{}}}此类共享者有权在此效率手册中创建会议、接受或拒绝邀请、编辑和删除您的效率手册中的活动、将您的效率手册与其他人共享以及撤销效率手册共享授权。{{{

  • }}}管理员。{{{}}}此类共享者有权创建任务、编辑任务、从任务列表中删除任务以及与其他人共享您的任务列表。{{{

  • }}}管理员。{{{}}}此类共享者有权查看、编辑、添加新信息到公文包中、删除文件夹以及将您的文件夹与其他人共享。{{{
