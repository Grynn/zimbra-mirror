使用拼写检查

在发出电子邮件之前，可以对电子邮件内容的拼写进行检查。单击写信工具栏上的拼写检查

如果希望在发送任何邮件之前自动检查其内容的拼写，可以在首选项>写信页面上启用发送邮件前强行进行拼写检查。

 拼写检查器不知道的词将会高亮显示。

  1. 单击拼写检查。拼写不正确的词将高亮显示。

  2. 单击高亮显示的词。此时会弹出一个有改错建议的列表。

  3. 从中选择正确的词。该词即被高亮显示为其他颜色。

  4. 若接受所做的改变并关闭拼写检查器，单击继续编辑。

 

备注：在关闭拼写检查器之前，您可把已经纠正的词还原为原来的拼写。单击高亮显示的词，并从弹出的列表顶部选择原来的拼写。

向拼写检查词典中添加词

可以向拼写检查词典中添加词。

  1. 单击拼写检查。拼写不正确的词将高亮显示。

  2. 单击高亮显示的词。此时会弹出一个有改错建议的列表。

  3. 改正该词并将其高亮显示后，单击该词，然后单击添加。该词即被添加到词典中。

将词添加到词典中后，就不能从词典中将其删除了。
