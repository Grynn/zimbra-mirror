自动完成地址功能

}}}当您直接在新邮件的收件人、抄送和密送框中键入地址时，自动完成功能会给出一些提示。{{{当您键入地址的开头部分时，一个地址列表会显示一些地址，即您的联系人列表中现有的地址，供您选择。

随着您继续在收件人：字段输入，匹配的地址列表就会越来越小，匹配的地址也会越来越少。如果没有匹配的地址，列表即消失。

示例

}}}例如，假定您要给 David Brinks 发送电子邮件，他的电子邮件地址正好是 dbrinks@acme.com。当您在地址字段中键入第一个字母“d”时，就会出现一个列表，其中显示了所有可能的匹配，并且匹配部分高亮显示。{{{在本例情况下，有可能显示：

}}}Dale Edwards < ded@acme.com>{{{

}}}David Brinks < dbrinks@acme.com>{{{

}}}Erica Dodd <erica878@xyz.com>{{{

当您在收件人：字段中输入其他字母时，相应的列表就会越来越小。

匹配

}}}自动完成功能可将您键入的文字与某联系人的以下部分进行匹配：{{{

  • }}}名{{{

  • }}}姓{{{

  • 电子邮件地址

匹配总是从字段的开头进行的。}}}文本“r;bob”的匹配项中将有“r;bob smith”，但不会有“r; billybob smith”。{{{

}}}使用自动完成功能{{{

当您输入的文字与一个或多个联系人匹配时，匹配项将显示在当前编辑字段下方的列表中。}}}在默认状态下，第一个匹配项将高亮显示。{{{

}}}姓名以“排名”顺序显示；您向其发送电子邮件的联系人位置高于没有向其发送或不常向其发送电子邮件的联系人。{{{}}}即联系最频繁的联系人列在最上面。{{{}}}如果不喜欢某联系人姓名始终列在最上面，可以单击忘记，这样会将该联系人重新排名到较低的位置。{{{

有几个键可用来触发特殊功能。鼠标也有可能用来选择匹配的项目。

  • 逗号、分号、Return/Enter 以及 Tab 键都具有完成功能。您所键入的文字将被当前选定的匹配项替换。

  • }}}按 ESC 键将隐藏列表。{{{

  • 在一个匹配项上单击鼠标，即可选中该项。

单个键（如分号）可启动自动完成功能，并可用于快速输入地址。}}}如果您知道用几个字符就能显示特定联系人的地址，就可键入那几个字符，然后键入分号，相应联系人的地址随即就会出现。{{{}}}例如，如果我知道“r; db”在 David Brinks 的第一次匹配中的结果，我就可以键入“r; db;”，完整地址“r;David Brinks < dbrinks@acme.com>”就会出现在地址框中。{{{
