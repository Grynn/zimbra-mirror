开始使用 @CollaborationProductShortName@ 标准版网络客户端

标准版网络客户端可通过以下功能为您提供一个易于使用的电子邮件帐户。

备注：所用帐户有可能不包括上述全部功能。若需了解您的帐户都启用了哪些功能，请与您的系统管理员联系。

参见浏览您的信箱。

电子邮件

    • 写信和发送邮件

    • 阅读和回复邮件

    • 通过会话功能跟踪来往邮件，也可转换到传统邮件视图

    • 在邮件中加入附件

    • 将邮件转发给一个或多个收件人

    • 创建您自己的文件夹用以整理邮件

    • 创建标记用以整理邮件

    • 创建过滤器以便将来信导入指定的文件夹

地址簿

    • 创建和管理多个地址簿

    • 导入和导出联系人列表

行事历

    • 创建和管理多个行事历

    • 创建约会、会议和事件

    • 导入/导出其他行事历程序。

任务

    • 创建多个“任务列表”

    • 创建待办事项

    • 将附件添加到任务

    • 管理任务、设置优先级和跟踪进度

公文包

    • 上传任何类型的文件到您的公文包中的文件夹，以便随时登录到您的帐户访问该文件

    • 创建文件夹，用以整理上传的文件

    • 链接到其他公文包

首选项

    • 管理邮箱的工作方式

    • 配置您的帐户设置，包括显示在“发件人”字段中的电子邮件地址以及是否自动在电子邮件中包含签名

    • 设定邮件签名

    • 创建邮件过滤器

    • 查看可用的快捷键组合，用以快速浏览邮箱

 

此外，您还可在任意视图内搜索邮件、个人联系人、行事历和任务列表等。
