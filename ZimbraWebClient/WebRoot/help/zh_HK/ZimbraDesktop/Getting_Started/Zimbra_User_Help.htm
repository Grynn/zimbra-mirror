關於 Zimbra Desktop

Zimbra Desktop 是一種離線的用戶端，可用於在同一個桌面工具下管理多個電郵帳戶。Zimbra mail, Yahoo! mail, Gmail, Outlook 以及任何其他工作或私人的使用 POP 或 IMAP 電郵帳戶都可以很方便的在 Zimbra Desktop 中建立。每個帳戶是分別維護。

在設定好你的帳戶後，你的郵件將被下載到你的電腦中，不管是否連接網絡，你都可以存取這些郵件。參見離線工作。你所有帳戶中的郵件都可以在 Zimbra Desktop 中閱讀、回覆、轉寄和刪除。郵件的存儲量取決於你的電腦的磁碟空間大小，不取決於預定義的帳戶配額

當你的任何帳戶收到郵件時，如果這時你已連接網絡，則該郵件將自動下載到你的 Zimbra Desktop。xxxx此功能並不需要你打開 Desktopxxxxxxxx。xxxxWindows Tray 或 Mac Dock 中有一信封圖示，它會顯示你所有帳戶的未讀郵件總數。xxxx按此圖示打開此桌面應用程式xxxxxxxx。xxxx

Zimbra Desktop 的主要功能

除了可以透過一個工具存取不同的電郵帳戶，Zimbra Desktop 同樣可用來管理你的聯絡人，並且具有記事簿、事項清單等應用程式以及用於存儲附件的公事包工具。
