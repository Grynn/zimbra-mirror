設定你的 Zimbra Web Client 帳戶

你須先為你現有的帳戶輸入設定值，然後方可使用 Zimbra Desktop。首次打開 Zimbra Desktop 時，系統會要求你建立一個帳戶。你可隨時建立另外的帳戶。

在開始建立帳戶前，確定你已連接到互聯網。

  1. 打開 Zimbra Desktop，並按一下右邊角的設定。  

  2. 按一下加入新帳戶。在帳戶類型清單中，選擇 Zimbra。「Zimbra 帳戶設定」頁面即刻顯示。輸入以下資訊。

    • 帳戶名稱。這是作為帳戶名顯示在 Zimbra Desktop 中的名稱。每個帳戶必須有不同的名稱。例如，對於 ZWC 帳戶，可稱之為 Zimbra。

    • 電郵地址。輸入你的 ZWC 帳戶的電郵地址。

    • 密碼。輸入用於登入你的 ZWC 帳戶的密碼。

    • 內送郵件伺服器。輸入 Zimbra 伺服器的地址。一般形式為 mail.example.com。也就是你的帳戶所在的伺服器。

    • 安全。如果你經常使用加密連接，請在存取該伺服器時剔選「使用 SSL 加密」。如果你不能確定是否選擇此選項，請與你的系統管理員聯繫。

    • 在同步設定中，選擇 Zimbra Desktop 從你的帳戶獲取資料的頻次。預設為收到新郵件時檢查並同步郵件。

如果你選擇手動同步，你必須按 Zimbra Desktop 工具列上的「收發」，從而將你的帳戶與 Zimbra 伺服器同步。

如果你選擇的間隔時間太短，如 1 分鐘，你的電腦的效能可能要受到影響。

你可隨時修改這個設定，方法是按「設定」，然後修改相關資料。

  1. 完成後，按認證與儲存。

Zimbra Desktop 即開始驗證你的帳戶資料。如果你的帳戶順利通過驗證，你將看到「服務已建立」對話方格。如果你的帳戶未能通過驗證，須返回到「帳戶設定」對話方格查看你的輸入並進行必要的修改。

  1. 要打開 Desktop，按啟動 Desktop。在你的帳戶進行同步期間，你可在 Zimbra Desktop 中工作。
