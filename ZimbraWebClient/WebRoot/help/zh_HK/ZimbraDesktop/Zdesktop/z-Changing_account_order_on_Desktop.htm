組織你的 Desktop 視窗

你設定 Desktop 時，你的帳戶會分別顯示在「概況」窗格中。如果你希望所有郵件均在一個帳戶中，則啟用一般偏好設定顯示「所有郵箱」部分以及我的其它帳戶。你的各個帳戶的收件箱、寄件箱和草稿箱內的所有郵件將在該部分合併顯示。所有帳戶的未讀項目數目會顯示出來。你的單個帳戶會在「所有郵箱」下方列出。你可以按一下一個帳戶，僅查看該帳戶中的項目。

 

 

在「所有郵箱」視窗中查看郵件時，郵件清單中的一個圖示會確定郵件來自哪個帳戶。

 

預設帳戶

你開啟 Zimbra Desktop 時，你的帳戶會按設定時的順序顯示在「概況」窗格中。第一個帳戶是預設桌面帳戶。這是你啟動 Zimbra Desktop 時開啟的帳戶。你可以變更預設帳戶。

  1. 按一下 Zimbra Desktop 上的設定。我的帳戶頁面會顯示你的帳戶清單。

  2. 要變更預設帳戶，按一下帳戶名稱旁的 。該帳戶則移動至頂端。  

 

除了是 Zimbra Desktop 顯示時開啟的帳戶外，預設帳戶亦為 Zimbra Desktop 被設定為預設郵件應用程式的情況下使用的帳戶。你可在系統預設部分中的偏好設定 > 一般頁面對此進行設定。
