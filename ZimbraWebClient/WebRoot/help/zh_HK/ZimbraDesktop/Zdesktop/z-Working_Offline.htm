離線工作

在 Zimbra Desktop 中，你可以隨時存取從你的帳戶下載的郵件，即使你並沒有連接到互聯網。在未連接互聯網的情況下，在 Zimbra Desktop 中工作稱為「離線工作」。

離線工作時，Zimbra Desktop 不能存取你的電郵帳戶，但你仍能在 Zimbra Desktop 中工作，就像你已連接到互聯網一樣。你可查看你的郵件、編寫新郵件、清除資料夾以及其他改動操作，在下一次連接到互聯網時，所做的改動將反向同步到你的帳戶。

如果你的帳戶名旁的正方形是紅的，你就知道自己正在離線工作

 

備註：你可能已離線但該圖示尚未變紅，這是由於網絡狀態更新時 Zimbra Desktop 正在嘗試同步到伺服器。如果嘗試失敗，該圖示才變紅。

 

按下「概況」窗格上「帳戶」旁邊的小圖示，你可強制所有帳戶離線。

再按此圖示，重新上線。  

離線工作時，可做些什麼？

  • 可以讀郵件，把郵件移至資料夾並標示郵件，以便於後繼處理。

  • 可以回覆郵件並傳送郵件，可以轉寄郵件或撰寫新郵件。這樣的郵件被儲存在寄件箱中，等到下次連接上網時即被自動傳送。

  • 可以刪除郵件。刪除的郵件被放入「垃圾桶」中。對於 Zimbra 帳戶，下次連接到互聯網時，相應的郵件即被移至你的帳戶的「垃圾桶」中。如果你清理「垃圾桶」，其中的郵件即被刪除。

  • 可以利用「搜尋」和「進階搜尋」功能在你的 Zimbra Desktop 帳戶中尋找郵件和附件。

  • 可以建立並儲存搜尋項。

  • 可以修改你的 Zimbra Desktop 偏好設定。這些設定會影響你在 Zimbra Desktop 中的工作方式。修改 Zimbra Web Client Desktop 中的偏好設定時，Zimbra Web Client 帳戶的偏好設定亦將相應改變。在「偏好設定」中對其他電郵帳戶所做的修改只對 Zimbra Desktop 的設定有影響。這樣的修改不會改變你的帳戶設定。
