在 Zimbra Desktop 的分享資料夾中工作

xxxx備註：xxxx僅 Zimbra 帳戶可使用。

當你接受對方的分享請求時，該資料夾即顯示在你的 Zimbra Desktop 中的「概況」窗格中。若你在 Zimbra Desktop 中在線上工作，你可看到分享資料夾的內容。該資訊並沒有實際地儲存在你的郵箱中或你的電腦中。

在授權人進行分享資料夾授權時，可從下面的存取角色中選一個：

  • xxxx瀏覽者xxxxxxxx。xxxx此類分享者可讀取資料夾的內容，但不能改變其內容。

  • xxxx管理者xxxxxxxx。xxxx此類分享者有權檢視和修改郵件夾的內容、建立新的子資料夾、以你的名義傳送郵件以及刪除郵件。

  • xxxx管理員xxxxxxxx。xxxx此類分享者擁有全權，可以檢視和修改分享資料夾的內容、建立新的子資料夾、以你的名義刪除分享資料夾中的郵件，以及將資料夾與其他人分享。
