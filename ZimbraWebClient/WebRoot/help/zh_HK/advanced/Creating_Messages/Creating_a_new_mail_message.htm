撰寫和傳送郵件

你可撰寫郵件，並在寫好後馬上傳送，亦可先撰寫一份草稿，然後再返回完成郵件並傳送。

編寫新郵件時，首先按下工具列上的 ，以開啟一個空白寫信頁面。也可右鍵點擊郵件的寄件人欄內的姓名，然後選擇新郵件，以開啟一個空白寫信頁面。

根據你的郵件偏好設定，應用程式工具列中顯示「寫信」索引標籤時，可開啟寫信頁面或開啟單獨的寫信視窗。你可開啟多個「寫信」索引標籤，並且你可在工具列中的不同索引標籤間移動。

在「寫信」頁面上，將收件人的電郵地址輸入到收件人：欄位。

  • 若需尋找某個人的電郵地址，可按下收件人，以便搜尋你的聯絡人清單或全網域地址清單（如有）。

  • 若需加入「密件」地址，按下郵件標題中「副本：」文字欄位旁邊的顯示密件。

如果你已設定不同身份，則寫信頁面中會顯示「寄件人」欄位。選擇用作「寄件人」地址的身份。

撰寫郵件既可以採用 HTML 格式，也可採用純文字格式。預設的格式由在「一般偏好設定」中所做的設置決定。要快速改變目前郵件的格式，按下寫信工具列上的選項並從「HTML」或「純文字」中選擇一個。

  • 在 HTML 格式下，你可用不同的字型格式化你的郵件，並且可以建立表格，加入顏色和插入超連結。

  • 純文字格式所生成的文字沒有式樣或格式。任何電腦都可以讀取這種類型的郵件。

當你在 @CompanyName@ 帳戶中進行操作時，你可請求回執，按一下寫信工具列中的「選項」，然後選擇要求提供讀信回條。

你可附加檔案（包括圖片、文件、試算表和音效檔）至你的郵件。
