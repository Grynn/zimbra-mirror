使用 rich text 編輯器

在公事包中撰寫郵件和文件，可用 rich-text 編輯器格式化頁面內容。

在用 rich-text 編輯器時，你可透過工具列格式化版面配置。 選擇字號、字型、文字樣式、字型顏色和背景顏色。這些格式化工具的用法與許多文字處理工具相同。另外，你可透過按下的方式將表格、附件和圖片加入頁面中。

加入段落樣式

工具列中的  圖示顯示可用的段落樣式。利用這些段落樣式，你可輕而易舉地在文件中格式化文字並保持一致的風格。這些樣式包括：用於大多數段落的普通選項目，表示章節標題的六級標題，可使文字變為斜體的字型，以及可將文字顯示為 Courier 字型時使用的預定的格式。這些樣式不能修改，但可使用各種編輯選項來改變字型、字號、顏色等等。

在頁面上加入表格

加入表格到頁面時，你須分三步進行。即加入表格，設置表格屬性和向表格加入內容。若需加入表格，按一下工具列上的表格圖示，選取插入表格，然後選擇表格的行數和列數。

在頁面上加入附件和圖片

你可上傳「文件」頁面所連結的文件檔案。按一下迴形針或圖片圖示，然後瀏覽，以尋找要上傳的檔案。如果要上傳的檔案在你的「公事包」中已經存在，你會得到提醒。

連結至網頁

你可連結至一個外部的網路 URL，並為該連結配一個描述性的標題，以免顯示 URL。按一下圖示，並輸入頁面或網路 URL 至「連結至」欄位。
