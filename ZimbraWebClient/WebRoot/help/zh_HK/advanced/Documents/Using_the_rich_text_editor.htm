使用 rich text 編輯器

編寫頁面可用 Rich-text 編輯器格式化頁面內容，或用  HTML 源碼透過 HTML 標記格式化頁面。可在兩種格式之間來回切換，以便利用 Rich-text 的編輯功能並插入表格和試算表。

在用 Rich-text 編輯器時，你可透過雙行工具列格式化頁面佈局。選擇字號、字型、文字樣式、字型顏色和背景顏色。這些格式化工具的用法與許多文字處理工具相同。

另外，你可透過按下的方式將表格、試算表、附件和圖片加入頁面中。

加入段落樣式

工具列上的    圖示顯示有可用的段落樣式。利用這些段落樣式，你可輕而易舉地 在記事本頁面中格式化文字並保持一致的風格。這些樣式包括：用於大多數段落的普通選項目，表示章節標題的六級標題，可使文字變為斜體的 字型，以及可將文字顯示為 Courier 字型時使用的預定的格式。這些樣式不能修改，但可使用各種編輯選項分別對某一樣式進行格式化處理，如改變字型、字號、顏色等等。

在頁面上加入表格

加入表格到頁面時，你須分三步進行。即加入表格，設置表格屬性和向表格加入內容。若需加入表格，按下工具列上的表格圖示，選取「插入表格」，然後選擇表格的行數和列數。

在頁面上加入試算表  

要加入試算表，按下工具列上的試算表圖示 。一個六列試算表即被加入頁面中。試算表可用於建立 資料庫，記錄活動、日程安排、事項的完成情況。試算表包括基本的試算表功能和一系列預定義的函數。

在頁面上加入附件和圖片

你可上傳「文件」頁面所連結的檔案。按下別針圖示 ，或圖片 圖示 ，然後尋找要上傳的檔案。如果要上傳的檔案在你的「文件」中或「公事包」中已經存在，你會得到提醒。

連結至記事本頁面或網頁

你可以連結至別的「文件」記事本頁面或一個外部的網絡 URL 並 為該連結配一個描述性的標題，這樣 URL 地址就不會顯示出來。
