用戶介面功能和快速鍵

Zimbra Web Client 介面包括許多方便的功能。

  • 下箭咀選單。有些指令按鈕的右面有一個小小的下箭咀圖示。按下該按鈕便可執行一項功能，點擊下箭咀選單即可顯示更多的選項。

  • 拖曳。 按下並在資料夾之間拖曳郵件或對話。

  • 右鍵選單。右鍵點擊資料夾、對話、郵件、郵件或對話中的「寄件人」列等項目或郵件標題中的項目時，一個選單隨即顯示，列出可用於該項目的各種功能。此例為用右鍵點擊寄件人時的情況。

  • 可移動邊框 對於分隔窗格，如「對話」視窗，你可用邊框增大或縮小上下窗格的相對大小。按下並拖曳分隔窗格的橫條。

 

  • 可點擊 項。Zimbra 遵循標準瀏覽器的慣例，當滑鼠經過某個連結時，即變為「手」狀。

  • 提示工具。 對於某些項，如指令按鈕、對話以及郵件標題中的電郵地址，當滑鼠經過上面的時候，就會顯示一個小方格，用以顯示相關的說明。

 

  • 翻頁。 如果清單太長而無法在一個視窗中顯示，可用清單視窗的左右箭咀前後翻頁。請注意，翻頁按鈕在不同的畫面上可能有不同的功效。數字表示郵件在頁面中的順序。你可用提示工具的顯示，瞭解翻頁功能在目前畫面是什麼功能。
