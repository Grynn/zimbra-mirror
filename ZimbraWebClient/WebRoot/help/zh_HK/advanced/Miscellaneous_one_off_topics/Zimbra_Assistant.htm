Zimbra 助手

Zimbra 助手是一種進階用戶功能，可用鍵盤與網絡用戶端快速交互，以完成下列事項：

  • 建立新預約

  • 前往你記事簿中的某一日期

  • 建立新聯絡人

  • 可在不離開目前正在使用的應用程式的情況下傳送新郵件

除了「偏好設定」索引標籤之外，你可從 Zimbra Web Client 視窗的任何地方開啟 Zimbra 助手。開啟 Zimbra 助手的快速鍵是 ` （後引號）或 shift~（波浪號）。此鍵位於 Escape （Esc）鍵下面。

  • Shift ~ （波浪號） 在 Mac OS X 下使用 Safar 或 Firefox 時不起作用。後引號（`）起作用。

Zimbra 助手對話方格提供詳細的幫助資訊。以下是使用 Zimbra 助手的一般步驟。

  1. 在 Zimbra Web Client 中，使用快速鍵（後引號或 shift 波浪號）。Zimbra 助手對話方格隨即顯示。

  2. 輸入下列其中一個指令

    • 若需建立預約，輸入預約。然後在引號內輸入主題、時間、日期和星期以及對預約的任何註釋。按下確定。 備註：如果需要更進階預約功能，可按下更多詳細資料，以前往記事簿的預約頁面。

    • 若需前往你記事簿中的某一具體日期，可輸入記事簿和日期或星期。按下確定。 備註：在日期之後，你可在相關的「記事簿」視窗中輸入日、工作週、週或月。「記事簿」即以所選視窗開啟。

    • 若需建立新聯絡人，輸入聯絡人並輸入聯絡人姓名、電郵地址等。最後按下確定。 備註：如果想開啟「聯絡人」的詳細資料格式，可按下「更多詳細資料」。在這個格式中，你可指定向哪個通訊錄加入聯絡人。

    • 若需在不開啟寫信視窗的情況下傳送郵件，可輸入郵件。在引號內輸入主題；輸入 to:name@adress ，然後輸入郵件正文。 按下確定。
