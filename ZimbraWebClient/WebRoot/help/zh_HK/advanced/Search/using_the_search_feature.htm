使用搜尋功能

搜尋功能用於尋找郵件、聯絡人的預約、「文件」頁面和檔案。 你可透過特定的字詞、按日期、時間、URL、大小、標記、郵件是否為已讀、郵件是否含附件、是否有特定檔案類型的附件等等進行搜尋。

Zimbra 提供了兩種搜尋工具：

  • 搜尋。這是一種快速搜尋方式，執行目前顯示在搜尋框中的任何搜尋條件。搜尋框左側的下拉箭咀用於選擇所要搜尋的條目的類型。你可選擇在郵件、你的聯絡人（其中包括公司聯絡人清單）、事項、預約、頁面和檔案中進行搜尋。這種方式的目標就是快速搜尋，詳情請見快速搜尋設定中的說明。

  • 進階搜尋可開啟一個新窗格，更適合執行較複雜的搜尋。 你可將進階搜尋條件儲存起來，以便以後再次使用。

搜尋郵件

如果你熟悉文字相符搜尋或文字處理功能，如 Microsoft Word 中的尋找/替換功能，那麼請注意，Zimbra 網絡用戶端的內容搜尋功能與執行字面上的相符字串還是有一定區別的。

Zimbra 搜尋的句法功能如下：

  • 你可以搜尋短語，但是短語中每個單字只與整詞相符。不允許有變體。例如，如果搜尋bananas，則其中帶有 banana 一詞的郵件是不相符的。你可按網域名稱進行搜尋，其中可包括英文句號(.)

  • 搜尋功能不區分大小寫，South、south 和 SOUTH 被視為是同一個詞。

  • 以下特殊字元不能在搜尋文字中使用。 ~ ' ! @ # $ % ^ & * () _- + ? / { }[ ] ; : "

  • 可在字首後使用星號 * 作為通配字元。如要搜尋 do*，返回的條目可以是 dog、door 等。

  • 搜尋內容意味著搜尋郵件的正文，以及可能有的任何（系統可讀的）附件。系統可讀附件，是一種可以轉換成 HTML 格式的檔案類型。其中包括 Microsoft Office 文件（Word、PowerPoint、或 Excel），以及文字檔，但是不包括圖片或音效檔。
