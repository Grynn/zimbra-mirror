分享你的「事項」清單

你可與他人分享你的事項清單，還可為分享清單定義以下各種權限：

  • 內部用戶或群組 你可從以下兩種權限類型中選擇要授予的類型：

    • 瀏覽者。「分享者」可以看見事項清單的內容，但是不能對事項清單的內容進行任何修改。

    • 管理者。「分享者」擁有建立事項、修改事項和從事項清單中刪除事項的權限。

    • 管理員。此類分享者有權建立事項、修改事項、從事項清單中刪除事項以及將你的事項清單與其他人分享。

    • 無 選項用於暫時禁止存取授權人的事項清單但不撤銷分享權限。「分享者」的事項清單視窗中仍然有該清單，但不能查看或管理。

  • 外來客。你須建立一個用於存取分享事項清單的密碼。來客必須輸入該密碼才能查看事項清單。來客不能對事項進行做任何改變。

  • 公開。任何知道此分享事項清單 URL 的人都可以查看其內容。但不能做任何改動。

若需與人分享你的「事項」清單

右鍵點擊要分享的事項清單，然後選擇「分享事項清單」。

在「分享屬性」對話方格中，輸入分享者人選並設定適當的資訊。

 

與「內部用戶」或「群組」分享1. 選擇分享的「內部用戶」或「群組」

2. 在「郵件」欄位中，輸入分享者的電郵地址。在輸入地址時，與你所輸入字母相符的姓名會自動列出。 你可在該清單中選擇姓名。

3. 選擇角色，「瀏覽人」或「管理者」

4. 在「郵件」區，選擇要傳送給分享者的通知郵件的類型。

5. 按下確定。

與外來客分享

1. 在分享用戶部分，選擇外來客。選擇此項後，屏幕會出現「密碼」欄位。

2. 輸入該外來客的電郵地址。

3. 輸入密碼，該外來客今後便可將用該密碼存取此事項清單。

4. 選擇要傳送給他們的通知郵件的類型。此郵件將提供他們存取此清單所需的資訊。

5. 按確定。

 

與公眾分享

1. 在分享用戶部分，選擇「公開」。選取後，分享屬性對話方格隨即顯示你的事項清單的 URL。

2. 請記下該 URL，並向「公眾」用戶公佈該電郵地址。他們按下此 URL 時，便可讀取此事項清單的內容，但是不能做任何修改。

修改或撤銷權限

你可在任何時候撤銷權限。

  1. 右鍵點擊一個分享事項清單。

  2. 按下編輯屬性。用戶分享的公事包隨即列出。

  3. 若需改變權限，可按下編輯，
    若需移除權限，可按下撤銷。

  4. 選擇是否傳送一條有關這一變更的訊息。

  5. 按確定。
