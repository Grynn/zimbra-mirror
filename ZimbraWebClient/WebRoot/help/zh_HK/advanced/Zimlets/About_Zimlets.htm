關於 Zimlet

Zimlet 是為了將第三方資訊和內容與 Zimbra Web Client 的功能進行整合而開發的小程式。Zimlet 可使你在郵件中使用不同的內容類型，並與之互動。  

  • 按下郵件中的 URL 後，你便可啟動一個瀏覽器視窗，前往該 URL。將滑鼠懸停在 URL 上，便能以預覽方式看網站內容。

  • 右鍵點擊一個電話號碼後，即可從所用電腦的softphone（如Skype 或 Cisco 的VOIP ）撥號。

  • 右鍵點擊郵件中的一個日期，便可看見你的記事簿，並可在不離開郵件的情況下安排會議。

  • 右鍵點擊一個姓名、地址或電話號碼，即可更新你的通訊錄。

窗格中可能還有其他可用的 Zimlet。
