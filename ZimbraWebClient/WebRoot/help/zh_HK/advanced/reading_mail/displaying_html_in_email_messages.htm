顯示 HTML 格式郵件

你可用兩種格式查看電子郵件。

  • 純文字。  純文字郵件即不含 HTML 格式的郵件。以純文字格式傳送的郵件，在其他任何郵件用戶端中都可以讀取。即便郵件中含 HTML 格式，你也可以用純文字模式讀取。在純文字視窗中，HTML 標記會在郵件的正文部分顯示。

  • HTML。HTML 郵件可具有文字格式、編號、項目符號、背景色以及連結，以便讀取。有些電子郵件用戶端可能不接受 HTML 格式的郵件。

若需始終以 HTML 格式查看郵件，請剔選偏好設定 - 郵件索引標籤上的以 HTML 格式查看郵件 (若可能)核取方格。 如果是以 HTML 格式傳送的，郵件將顯示為格式化的版面。
