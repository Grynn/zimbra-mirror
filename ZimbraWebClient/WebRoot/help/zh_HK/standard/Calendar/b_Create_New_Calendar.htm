建立新的記事簿

系統已為你建立了一個記事簿，但你可建立更多的記事簿，用以追蹤不同類型的活動。 比如，你可以為工作、個人、公司的各項活動建立分開的記事簿，也可訂用互聯網上的記事簿，還可透過連結使用分享的記事簿。

可使用日程安排視窗對照查看兩個或更多的記事簿。

若需建立記事簿

  1. 選擇「記事簿」索引標籤。

  2. 在「記事簿」窗格中，按一下編輯。

  3. 在工具列中，按下新建記事簿。

  4. 輸入記事簿的名稱，選擇「記事簿」活動的顯示顏色。如果不想讓其他人看見記事簿中的活動，請 剔選報告空閒/忙碌時間時排除此記事簿。

  5. 預設情況下，已在 UI 中剔選為啟用狀態。此記事簿中的活動將在開啟「記事簿」視窗時顯示。 如果不想隨同別的記事簿一起查看此記事簿，請取消剔選此核取方格。在這種情況下，要查看此記事簿的活動，須選擇該記事簿並將其打開。

  6. 按下建立記事簿。

若需訂閱其他記事簿

  1. 選擇記事簿索引標籤。

  2. 在「記事簿」窗格中，按下編輯。預設的記事簿資訊隨即顯示。  

  3. 在工具列中，按下訂閱記事簿。 

  4. 輸入該記事簿的名稱，輸入該遠端記事簿的有效的iCal 檔案 URL，選擇一種顯示該記事簿活動的顏色。

  5. 按下建立記事簿。
