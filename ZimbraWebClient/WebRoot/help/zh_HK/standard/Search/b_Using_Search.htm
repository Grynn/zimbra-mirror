使用搜尋功能

Zimbra 的搜尋功能是一功能強大的工具。儘管你可以學會使用查詢語言描述所介紹的建立複雜文字查詢條件，但也可以採用較快速的方式，如此，你僅需要輸入一個姓名或單字，而無需任何關鍵字。例如，尋找所有含 tim 的郵件，不論是在主題行、收件人行、寄件人行、副本行、密件：行、郵件正文或附件中，都可在 尋找字框輸入tim ，並按下搜尋即可。

你須指定搜尋位置。你可在郵件、個人通訊錄、個人記事簿和個人事項清單中進行搜尋。

你可輸入任何值，如單字、名、姓、電話號碼或網域名稱。  若需更多選擇，可使用 星號 * 作為通配字元。  

在「搜尋」中使用 * 作為通配字元

星號 * 可以在 搜尋中作為通配字元使用，用以尋找詞拼寫相似的內容。  

請在字首後使用星號 *。如要搜尋 do*，送回的條目可以是 dog、door 等。

要在通訊錄中搜尋聯絡人，查詢條件必須是一個完整的詞。例如，如果按照電話號碼搜尋聯絡人，你必須輸入完整的電話號碼，不能僅按照區碼進行搜尋。

若需在通訊錄中搜尋聯絡人：

  1. 在「尋找」方格中，從下拉選單中選取個人通訊錄。這將搜尋你所有的通訊錄。

  2. 在尋找方格中輸入姓名或其他聯絡人資料。

  3. 按下搜尋即可執行。

若需快速搜尋郵件：

  1. 在「尋找」方格中，從下拉選單中選擇郵件。

  2. 在尋找方格中輸入單字、姓名或其他相關的資料。

  3. 按下搜尋即可執行。

建立複雜的搜尋條件

如果你需頻繁地執行同一搜尋條件或要搜尋特定的詳細資訊，你可能需要建立一個搜尋條件並將其儲存在「搜尋」資料夾中。若想再次執行該項搜尋，只需按一下即可。

若需建立並儲存某個搜尋條件

  1. 在「搜尋」面板中，按下編輯。建立新搜尋窗格隨即開啟。

  2. 選擇一個位置，以建立搜尋資料夾。 如果儲存在根資料夾中，此搜尋將顯示在「搜尋」區。 如果選擇其他資料夾，此搜尋即被儲存到那個資料夾。

  3. 在搜尋名稱欄位中，輸入此搜尋的名稱。

  4. 在搜尋查詢欄位中，按照查詢語言說明中的說明輸入查詢條件。你也可在查詢條件中使用And 或 Or。

    建立搜尋條件時使用 And 或 OR

    對於有多種條件的搜尋項，你可以尋找與其中任意條件相符的項目，也可尋找符合全部條件的項目。 :

    • 與指定條件中任意條件相符的搜尋稱為 OR 搜尋，意思是如果郵件中包含 X 或Y，則認為該郵件是相符的。

    • 同時包含 X 與 Y 的搜尋稱為 AND搜尋，也就是說，郵件必須符合所有指定的條件才是相符的。

  5. 點擊建立搜尋條件後，便可建立和執行搜尋，並可將其儲存到指定的資料夾中。

執行搜尋時，按下窗格中的搜尋名稱。搜尋結果將在「內容」窗格中顯示。
