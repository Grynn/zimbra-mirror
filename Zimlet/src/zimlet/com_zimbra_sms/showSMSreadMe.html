SETUP
1. Go to http://www.upsidewireless.com and create an account
2. purchase sms credits (60$ for 500 sms). There are several packages, but as long as you have Enterprise-SMS setup(next step) you should be fine.
3. Ask their customer support to make your account Enterprise-SMS account.
4. Single-click on the Zimlet or select Right-click > Preferences
5. Enter upsidewireless account information.
6. Make sure to 'Enable' Zimlet.
6. Select any other options and press OK
7. After the browser is refreshed, please Right-click on the Zimlet > Preferences and see if all the preferences are intact.

ZIMLET PREFERENCES
- Username: This is upsidewireless.com account's username
- Password: This is upsidewireless.com account's password
- 'Show 'Send SMS' Button in mail compose' checkbox: Allows sending SMS via Mail Compose
- 'Also Send Email to all recepients(along with SMS)' - checkbox to send SMS as well as email.

HOW TO SEND SMS
Option 1. Send SMS to single user:
- Single click on the Zimlet
- Open Mail Compose, Directly enter SMS number in To-field and press 'Send SMS'.
- Open Mail Compose, Enter email-address in To-field and press 'Send SMS'. Zimlet looks up addressbook for mobile number and sends SMS
PS: If you are sending SMS to US mobile, please start with '1' Since this supports international SMS, complete mobile number is needed.

Option 2. Send SMS to multiple users(from mail compose):
- Enter all the email addresses in To-field and press 'Send SMS'. Zimlet looks up addressbook for mobile numbers of all these users and sends SMS to all of them.
PS: You can't mix mobilenumber and email-address in to field, everything should be email-address.

Option 3. SMS an email, contact or appointment information
- Drag-drop an email or contact or appointment onto Zimlet

Option 4. SMS using Zimbra Assistant
- Hit `(tick) key, then type SMS, then type (123)1231234 smsInfo
